# 11.4 Быстроразъемные соединения (Camlock)

## Введение
Быстроразъемные соединения — фитинги для быстрого подключения/отключения рукавов без инструмента.

## Типы Camlock
- **Тип A**: адаптер с наружной резьбой
- **Тип B**: муфта с внутренней резьбой
- **Тип C**: муфта с хвостовиком
- **Тип D**: адаптер с хвостовиком
- **Тип E**: адаптер с внутренней резьбой
- **Тип F**: муфта с наружной резьбой

## Материалы
- Алюминий (легкие)
- Нержавеющая сталь (пищевая, химия)
- Полипропилен (агрессивные среды)
- Латунь (вода, воздух)

## Размеры
1/2", 3/4", 1", 1.5", 2", 3", 4"

## Применение
- Автоцистерны
- Пожарная техника
- Насосные станции
- Химическая промышленность

## Производители
- **Dixon** (США)
- **PT Coupling** (США)
- **Российские**: НПП "Технокомплект"
