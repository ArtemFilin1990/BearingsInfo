# Изображения

Директория содержит изображения, используемые в документации и статьях.

## Структура

### Основные категории

#### Схемы и диаграммы
- Конструктивные схемы подшипников
- Диаграммы размерных рядов
- Схемы монтажа и установки

#### Фотографии
- Фотографии подшипников
- Фотографии дефектов и повреждений
- Производственные процессы

#### Логотипы
- Логотипы производителей
- Брендовые изображения

#### Графики и таблицы
- Графики нагрузок
- Таблицы размеров
- Диаграммы характеристик

## Формат файлов

Поддерживаемые форматы:
- JPG/JPEG - для фотографий
- PNG - для схем и логотипов с прозрачностью
- SVG - для векторных изображений (если есть)

## Использование

Изображения используются в:
- Технических статьях
- Каталогах производителей
- Обучающих материалах
- Справочной документации

## Ссылки на изображения

В документах Markdown используется синтаксис:
```markdown
![Описание изображения](путь/к/изображению.jpg)
```
