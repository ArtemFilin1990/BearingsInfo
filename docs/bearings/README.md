# Bearings - Подшипники

Основная документация по подшипникам качения.

## Структура

### Основные разделы

#### Analogues (Аналоги)
- `analogues/` - Таблицы соответствий подшипников разных стандартов
- Аналоги ГОСТ ↔ ISO
- Таблицы взаимозаменяемости

#### Brands (Бренды)
- `brands/` - Информация о производителях подшипников
- Системы обозначений производителей
- Характеристики брендов

#### Catalog (Каталог)
- `catalog/` - Каталожная информация
- Размерные ряды
- Технические характеристики

#### Classification (Классификация)
- `classification/` - Классификация подшипников
- Типы и виды
- Конструктивные особенности

#### Designations (Обозначения)
- `designations/` - Системы обозначений
- Маркировка ГОСТ
- Маркировка ISO
- Префиксы и суффиксы

#### FAQ (Часто задаваемые вопросы)
- `faq/` - Ответы на распространенные вопросы
- Практические советы
- Рекомендации

#### Guides (Руководства)
- `guides/` - Практические руководства
- Инструкции по монтажу
- Методики расчета
- Рекомендации по эксплуатации

#### Standards (Стандарты)
- `standards/` - Стандарты и нормативы
- ГОСТ
- ISO
- DIN и другие

#### Training (Обучение)
- `training/` - Обучающие материалы
- Основы работы с подшипниками
- Учебные пособия

## Содержание

Документация охватывает:
- Теоретические основы
- Практические аспекты применения
- Техническую информацию
- Стандарты и нормативы
- Рекомендации производителей

## Использование

Материалы предназначены для:
- Инженеров-конструкторов
- Специалистов по обслуживанию
- Проектировщиков
- Студентов технических специальностей
- Закупщиков и снабженцев
