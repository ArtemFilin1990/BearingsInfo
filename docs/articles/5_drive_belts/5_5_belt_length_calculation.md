## Формула длины ремня
```
L = 2×a + π/2×(D+d) + (D-d)²/(4×a)
```

Где:
- **L** — длина ремня (мм)
- **a** — межосевое расстояние (мм)
- **D** — диаметр большого шкива (мм)
- **d** — диаметр малого шкива (мм)

## Пример расчета
**Данные**:
- a = 1000 мм
- D = 200 мм
- d = 100 мм

**Расчет**:
L = 2×1000 + 3.14/2×(200+100) + (200-100)²/(4×1000)  
L = 2000 + 471 + 2.5 = 2473.5 мм

**Ближайший стандарт**: 2500 мм
