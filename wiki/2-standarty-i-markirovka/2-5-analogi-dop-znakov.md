# Аналоги дополнительных знаков условных обозначений подшипников ГОСТ – ISO

## Определение

Таблицы соответствия дополнительных знаков ГОСТ ↔ ISO — справочные таблицы для перекодирования дополнительных знаков в обозначении подшипников: классы точности, группы радиального зазора, конструктивные изменения, материалы сепараторов.

## Назначение таблиц

Таблицы используются для:
- Перевода дополнительных знаков ГОСТ в ISO и обратно
- Проверки совместимости требований к точности и зазорам
- Сопоставления материалов сепараторов
- Идентификации конструктивных изменений

---

## Таблица 1. Обозначение класса точности подшипника

### ГОСТ ↔ ISO

| Класс точности ГОСТ | Класс точности ISO | Применение |
|---------------------|--------------------|-----------| 
| 0 | P0 (норм.) | Общее машиностроение |
| 6 | P6 | Станки нормальной точности, электродвигатели |
| 5 | P5 | Станки повышенной точности, шпиндели |
| 4 | P4 | Прецизионные станки, измерительные приборы |
| 2 | P2 | Сверхпрецизионные шпиндели, координатно-измерительные машины |

**Примечания:**
- Класс 0 (ГОСТ) = P0 (ISO) — нормальный класс точности
- Чем меньше номер класса, тем выше точность
- При замене подшипника необходимо соблюдать класс точности или выше
- Подробнее: Класс точности подшипников

| Подшипник | ГОСТ | ISO | Описание |
|-----------|------|-----|----------|
| Роликовый радиальный сферический двухрядный с цилиндрическим (коническим) отверстием | 1 | - | Минимальный зазор |

| | нормальная | С0 (норм.) | Нормальный зазор |

- Для большинства применений используется нормальный зазор (С0)
- Увеличенные зазоры (С3, С4) применяются при высоких температурах
- Уменьшенные зазоры (С2) — для точного позиционирования

## Таблица 3. Обозначения конструктивных изменений роликовых подшипников

### ГОСТ ↔ ISO

| Конструкция подшипника | ГОСТ | ISO |
|------------------------|------|-----|
| Измененная внутренняя конструкция | А | А |
| Габаритные размеры изменены в соответствии со стандартами ИСО | - | X |
| Модифицированный контакт | М | - |
| Кольцевая проточка и отверстия для смазывания на наружном кольце | Н | W33 (S) |

**Примечания:**
- Буква «А» — обозначает изменение конструкции без изменения габаритов
- Буква «X» (ISO) — габариты изменены для соответствия ISO
- W33 (ISO) — кольцевая проточка и 3 отверстия для смазки на наружном кольце
- Подробнее: Дополнительные знаки в условном обозначении подшипника

## Таблица 4. Обозначение материала сепаратора подшипника

### ГОСТ ↔ ISO

| Материал сепаратора подшипника | ГОСТ | ISO |
|--------------------------------|------|-----|
| Черные металлы (сталь) | Г | F |
| Алюминиевый сплав | Д | L |
| Латунь | Л | М |
| Стеклонаполненный полиамид | Е | TV (ТN) |
| Текстолит | Е | T (ТР) |

**Примечания:**
- Материал сепаратора влияет на рабочие характеристики подшипника
- Стальные сепараторы (Г/F) — наиболее распространённые, для высоких нагрузок
- Латунные сепараторы (Л/M) — для высоких скоростей
- Полиамидные сепараторы (Е/TV) — для низкого шума и малого трения
- Подробнее: Сепараторы подшипников

## Примеры применения таблиц

### Пример 1: Расшифровка обозначения ГОСТ → ISO

**ГОСТ:** 6-305Е
- **6** — класс точности → **P6** (ISO)
- **305** — базовое обозначение → **6305** (ISO)
- **Е** — полиамидный сепаратор → **TV или TN** (ISO)

**Результат:** 6305/P6 TN (ISO)

### Пример 2: Расшифровка обозначения ISO → ГОСТ

**ISO:** 6205/C3
- **6205** — базовое обозначение → **205** (ГОСТ)
- **C3** — увеличенный зазор → **7** (ГОСТ)

**Результат:** 7-205 (ГОСТ)

### Пример 3: Комплексное обозначение

**ГОСТ:** 5-46209АС3
- **5** — класс точности → **P5** (ISO)
- **46209** — базовое обозначение → **7209AC** (ISO)
- **С3** — увеличенный зазор → **C3** (ISO)

**Результат:** 7209AC/P5/C3 (ISO)

## Ограничения и рекомендации

**При использовании таблиц учитывать:**
- Не все дополнительные знаки имеют прямые аналоги
- Некоторые обозначения специфичны только для ГОСТ или только для ISO
- При замене подшипника необходима проверка всех дополнительных требований
- Материалы сепараторов могут иметь различные обозначения у разных производителей

**Рекомендации:**
- Всегда сверяйте полное обозначение подшипника, а не только базовую часть
- При сомнении консультируйтесь с технической документацией производителя
- Для ответственных применений проверяйте взаимозаменяемость по каталогам
