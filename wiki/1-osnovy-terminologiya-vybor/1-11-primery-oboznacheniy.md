# Примеры условного обозначения подшипников

## Определение

Расшифровка условного обозначения подшипника — практическое применение системы кодирования ГОСТ 3189-89 для определения конструкции, размеров и технических характеристик подшипника по его маркировке. В данном разделе приведены детальные примеры расшифровки обозначений различных типов подшипников.

## Примеры с внутренним диаметром d ≥ 10 мм

### Пример 1: 205 (самый простой)

```
2 0 5
│ │ └─ d = 05×5 = 25 мм
│ └─── Серия диаметров 0 (ультралегкая)
└───── Тип 0 (шариковый радиальный однорядный)
        Серия ширин не указана → 0 (нормальная)
```

**Полная расшифровка:** Подшипник шариковый радиальный однорядный, серия диаметров 0, серия ширин 0 (нормальная), внутренний диаметр 25 мм, класс точности 0, сталь ШХ15, стальной сепаратор, нормальный зазор.

---

### Пример 2: 32205 (роликовый цилиндрический)

```
3 2 2 0 5
│ │ │ │ └─ d = 05×5 = 25 мм
│ │ │ └─── Серия диаметров 2 (средняя)
│ │ └───── Тип 2 (роликовый с короткими цилиндрическими роликами)
│ └─────── Конструктивная разновидность 3
└───────── Серия ширин не указана → 0 (нормальная)
```

**Полная расшифровка:** Подшипник роликовый радиальный с короткими цилиндрическими роликами, конструктивная разновидность 3, серия диаметров 2, внутренний диаметр 25 мм.

---

### Пример 3: 602/32 (специальный диаметр)

```
6 0 2 / 32
│ │ │   └─ d = 32 мм (специальное значение)
│ │ └───── Серия диаметров 2
│ └─────── Тип 0 (шариковый радиальный)
└───────── Конструктивная разновидность 6
```

**Полная расшифровка:** Подшипник шариковый радиальный однорядный с защитной шайбой, серия диаметров 2, внутренний диаметр 32 мм (обозначается через дробь).

---

### Пример 4: 7608 (радиально-упорный)

```
7 6 0 8
│ │ │ └─ d = 8 мм
│ │ └─── Серия диаметров 0
│ └───── Тип 6 (радиально-упорный шариковый)
└─────── Серия ширин 7 (особо узкая)
```

**Полная расшифровка:** Подшипник шариковый радиально-упорный, серия ширин 7 (особо узкая), серия диаметров 0, внутренний диаметр 8 мм.

---

### Пример 5: 2910 (роликовый)

```
2 9 1 0
│ │ │ └─ d = 10×5 = 50 мм (но здесь 10 — код, не умножается!)
│ │ └─── Серия диаметров 9
│ └───── Тип 2 (роликовый цилиндрический)
└─────── Конструктивная разновидность 0
```

**Исключение:** 10 = 48 мм, код 10 не всегда означает 50 мм!

---

### Пример 6: 20071/1175 (конический)

```
2 0 0 7 1 / 1175
│ │ │ │ │   └─── d = 1175 мм (крупногабаритный)
│ │ │ │ └─────── Серия диаметров 1
│ │ │ └───────── Тип 7 (роликовый конический)
│ │ └─────────── Конструктивная разновидность 00
└─┴───────────── Серия ширин 2 (особо широкая)
```

**Полная расшифровка:** Подшипник роликовый конический однорядный, серия ширин 2, внутренний диаметр 1175 мм.

---

**Полная расшифровка:** Подшипник шариковый радиальный однорядный, серия ширин 1, серия диаметров 9, внутренний диаметр 4 мм.

**Полная расшифровка:** Подшипник шариковый радиальный однорядный, внутренний диаметр 5 мм.

---

### Пример 9: 184009/1,5

1 8 4 0 0 9 / 1,5

│ │ │ │ │ └─────── Серия диаметров 9
│ │ │ │ └───────── Обязательный ноль
│ │ │ └─────────── Тип 0 (шариковый радиальный)

**Полная расшифровка:** Подшипник шариковый радиальный однорядный с упорным бортом на наружном кольце, серия ширин 1, внутренний диаметр 1,5 мм.

---

### Пример 10: 1068

```
1 0 6 8
│ │ │ └─ d = 8 мм
│ │ └─── Серия диаметров 6
│ └───── Обязательный ноль (для d<10)

## Игольчатые подшипники

### Пример 16: ИК25×30×10Д

```
ИК 25 × 30 × 10 Д
│  │   │   │   └─ Сепаратор из алюминиевого сплава
│  │   │   └───── B = 10 мм (ширина)
│  │   └───────── D = 30 мм (наружный диаметр)
│  └───────────── d = 25 мм (внутренний диаметр)
└──────────────── ИК = игольчатый с сепаратором
```

**Полная расшифровка:** Подшипник радиальный роликовый с игольчатыми роликами однорядный с сепаратором, d=25 мм, D=30 мм, B=10 мм, сепаратор из алюминиевого сплава.

---

```
3 КК 30 × 35 × 46 Е
│ │  │   │   │   └─ Сепаратор из текстолита
│ │  │   │   └───── B = 46 мм (ширина)

## Примечания

**Ключевые моменты расшифровки:**

1. **Начинайте справа** — внутренний диаметр (последние 2 цифры)
2. **Позиция 3** — серия диаметров
3. **Позиция 4** — тип подшипника (0-9)
4. **Позиция 7** — серия ширин (если есть)
5. **Слева от "-"** — зазор, точность, момент, категория
6. **Справа от основного** — материалы, смазка, конструкция

**Типичные обозначения:**
- Класс 0, нормальный зазор, категория С → **не указываются**
- Серия ширин 0 → **не указывается**
- Стальной сепаратор → **не указывается**
