# KNOWLEDGE_BASE.md - Информация

## О файле

Этот файл содержит **полную, структурированную и машиночитаемую базу знаний** репозитория BearingsInfo, построенную в режиме максимального контекста.

## Метаданные

- **Дата создания:** 2026-01-23
- **Версия:** 1.0
- **Размер файла:** ~6 MB
- **Всего обработано файлов:** 3,279
- **Извлечено терминов:** 11,702
- **Идентифицировано структур данных:** 384

## Структура

База знаний организована в следующие разделы:

### 1. Инвентаризация файлов
Полная таблица всех обработанных файлов с метаданными:
- Имя и путь файла
- Тип файла
- Размер
- Назначение
- Ключевые темы

### 2. Термины и глоссарий
Извлеченные термины и определения из всех Markdown файлов с полной трассировкой к источникам.

### 3-10. Структурные разделы
- Процессы и алгоритмы
- Правила и ограничения
- Структуры данных и форматы
- Роли и ответственности
- Инструкции и сценарии
- Ошибки и исключения
- Связи и зависимости
- Источники и трассировка

### Детальная декомпозиция
Подробное описание каждого файла с превью содержимого.

## Использование

### Для LLM и обучения
Файл подходит для:
- Обучения языковых моделей
- Дообучения (fine-tuning) на специфике подшипников
- Контекста для генерации ответов

### Для RAG / Vector DB
Структура позволяет:
- Разбиение на chunks с сохранением источников
- Векторное индексирование
- Семантический поиск

### Для корпоративной Wiki
Готовая структура для:
- Справочной системы
- Документации
- Обучения сотрудников

### Для автоматизации
Машиночитаемый формат для:
- Парсинга и обработки
- Экспорта в другие форматы
- Интеграции с системами

## Как обновить

Чтобы обновить базу знаний с актуальными данными:

```bash
python scripts/build_knowledge_base.py
```

База знаний будет перестроена с учетом всех изменений в репозитории.
