# Аналоги и взаимозаменяемость

Таблицы соответствия и взаимозаменяемости подшипников.

## Файлы

### gost_iso.csv
Соответствие ГОСТ ↔ ISO

### units.csv
Подшипниковые узлы и их аналоги

### housings.csv
Корпуса подшипниковых узлов

## Использование

Для поиска аналога:
1. Найдите исходное обозначение в первом столбце
2. Получите соответствующий аналог из второго столбца
