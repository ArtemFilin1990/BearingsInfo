# 1.3.4 Как выбрать подшипник

## Цель

Предоставить пошаговое руководство по выбору подшипника качения для конкретного применения с учётом условий эксплуатации, нагрузок, скорости вращения и других факторов.

## Ключевые термины

- **Радиальная нагрузка** — нагрузка, направленная перпендикулярно оси вращения
- **Осевая (упорная) нагрузка** — нагрузка, действующая вдоль оси вращения
- **Динамическая грузоподъёмность (Cr)** — нагрузка, при которой подшипник выдерживает 1 млн оборотов
- **Статическая грузоподъёмность (C0r)** — максимальная нагрузка в неподвижном состоянии
- **Ресурс L10** — расчётный ресурс в часах или оборотах (90% подшипников достигнут этого ресурса)
- **Предельная частота вращения** — максимально допустимая скорость при данных условиях

## Суть

### Основные этапы подбора подшипника

#### 1. Определение условий работы

**А. Тип и направление нагрузки**

- **Только радиальная** → шариковые радиальные (6xxx), роликовые цилиндрические (N, NU, NJ)
- **Только осевая** → упорные шариковые (8xxx), упорные роликовые (9xxx)
- **Комбинированная** → радиально-упорные шариковые (7xxx, 36xxx), конические роликовые (30xxx, 32xxx)
- **С перекосом вала** → сферические роликовые (3xxx), сферические шариковые (1xxx)

**Б. Скорость вращения**

- **Низкая (<500 об/мин)** → любой тип, приоритет на грузоподъёмность
- **Средняя (500-3000 об/мин)** → шариковые, роликовые цилиндрические
- **Высокая (>3000 об/мин)** → шариковые радиальные (особенно лёгкой серии), керамические гибридные
- **Очень высокая (>10000 об/мин)** → прецизионные шариковые P4/P5, гибридные, с масляной смазкой

**В. Условия эксплуатации**

| Условие | Рекомендация |
|---------|--------------|
| Пыльная среда | Закрытые подшипники (2RS, 2Z) |
| Влажная среда | Уплотнения 2RS, нержавеющие (SS) |
| Высокая температура (>100°C) | Подшипники HT с высокотемпературной смазкой, C3/C4 зазор |
| Вибрация и удары | Сферические роликовые (3xxx), усиленные (E) |
| Химически агрессивная | Нержавеющие, керамические, с защитными покрытиями |
| Пищевая промышленность | Нержавеющие SS с пищевой смазкой NSF H1 |

#### 2. Расчёт требуемой грузоподъёмности

**Формула эквивалентной динамической нагрузки:**

```
P = X × Fr + Y × Fa

где:
P — эквивалентная динамическая нагрузка (кН)
Fr — радиальная нагрузка (кН)
Fa — осевая нагрузка (кН)
X, Y — коэффициенты (берутся из каталога для типа подшипника)
```

**Формула расчёта ресурса:**

```
L10h = (C / P)^p × (10^6 / (60 × n)) часов

где:
C — динамическая грузоподъёмность подшипника (кН)
P — эквивалентная динамическая нагрузка (кН)
p — показатель степени (3 для шариковых, 10/3 для роликовых)
n — частота вращения (об/мин)
```

**Рекомендуемый запас:**
- Для нормальных условий: C ≥ 3P
- Для тяжёлых условий: C ≥ 5P
- Для ударных нагрузок: C ≥ 8P

#### 3. Выбор размера подшипника

**А. Определение диаметра вала (d)**
- Исходя из конструкции механизма

**Б. Выбор серии диаметров**

| Серия | D/d | Применение |
|-------|-----|------------|
| Особо лёгкая (0, 1) | Минимальное | Компактные узлы, высокие обороты |
| Лёгкая (2) | Малое | Электродвигатели, вентиляторы |
| Средняя (3) | Среднее | Редукторы, промышленное оборудование |
| Тяжёлая (4) | Большое | Тяжёлое оборудование, высокие нагрузки |

**В. Серия ширин**
- Узкая (0, 1) — для компактных конструкций
- Нормальная (2) — универсальное применение
- Широкая (3, 4) — повышенная грузоподъёмность

#### 4. Подбор дополнительных параметров

**А. Защита**
- Без защиты — для чистых условий с внешними уплотнениями
- Z, 2Z — металлические шайбы для защиты от пыли
- RS, 2RS — резиновые уплотнения для влажной и загрязнённой среды

**Б. Радиальный зазор**

| Зазор | Когда применять |
|-------|------------------|
| C2 | Прецизионные узлы, предварительный натяг |
| C0 (норм.) | Стандартные условия |
| C3 | Температура >80°C, тепловое расширение |
| C4, C5 | Высокие температуры >120°C |

**В. Класс точности**

| Класс | Применение |
|-------|------------|
| 0 (норм.) | Общепромышленное оборудование |
| 6 (P6) | Станки, редукторы |
| 5 (P5) | Высокоточные станки, шпиндели |
| 4 (P4) | Прецизионное оборудование |
| 2 (P2) | Измерительные приборы |

#### 5. Выбор смазки

| Смазка | Скорость | Температура | Применение |
|--------|----------|-------------|------------|
| Литиевая NLGI 2 | Средняя | -30 до +120°C | Универсальная |
| Литиевый комплекс | Высокая | -40 до +150°C | Тяжёлые условия |
| Полимочевина | Очень высокая | -40 до +180°C | Электродвигатели |
| Масло ISO VG 32-68 | Высокая/очень высокая | -20 до +100°C | Шпиндели |

### Типовые решения по отраслям

#### Электродвигатели общего назначения
- **Тип**: 6000, 6200 серии
- **Защита**: 2RS или 2Z
- **Зазор**: C3
- **Смазка**: Литиевая NLGI 2 или полимочевина

#### Редукторы промышленные
- **Тип**: NU, NJ (цилиндрические роликовые)
- **Серия**: Средняя (3) или тяжёлая (4)
- **Смазка**: Пластичная EP NLGI 2-3

#### Автомобильные ступицы
- **Тип**: 30xxx (конические роликовые), парная установка
- **Смазка**: Высокотемпературная EP
- **Защита**: Дополнительные сальники

#### Шпиндели станков ЧПУ
- **Тип**: 7xxx (радиально-упорные)
- **Точность**: P4, P5
- **Схема**: DB, DF, DT (дуплекс)
- **Смазка**: Масло-воздушная

## Примеры

### Пример 1: Электродвигатель 4 кВт, 1500 об/мин, вал Ø30 мм

**Условия:**
- Радиальная нагрузка: 1.2 кН
- Непрерывная работа: 8000 ч/год
- Температура: до 90°C
- Среда: чистое помещение

**Расчёт:**
1. Тип: шариковый радиальный (высокая скорость)
2. Размер: d=30 → **6206**
3. Проверка: Cr = 19.5 кН, P = 1.2 кН
   ```
   L10h = (19.5/1.2)³ × 10⁶ / (60×1500) ≈ 50 000 ч
   ```
4. Модификация: **6206-2Z C3** (пыльники + зазор для температуры)
5. Смазка: литиевая NLGI 2

### Пример 2: Редуктор, вал Ø40 мм, радиальная нагрузка 8 кН, 750 об/мин

**Условия:**
- Высокая радиальная нагрузка
- Вибрация
- Температура нормальная

**Расчёт:**
1. Тип: роликовый цилиндрический (чисто радиальная)
2. Размер: d=40 → **NU 308**
3. Проверка: Cr = 81.9 кН (запас ≈10 раз — хорошо)
4. Модификация: **NU 308 E** (усиленная)
5. Смазка: пластичная EP NLGI 3

### Пример 3: Ступица легкового авто, 150000 км ресурс

**Условия:**
- Комбинированная нагрузка
- Переменные температуры
- Влага, дорожная соль

**Расчёт:**
1. Тип: конический роликовый
2. Размер: **30205** (парная установка)
3. Схема: конусами навстречу (О-образная)
4. Смазка: высокотемпературная литиевая EP
5. Защита: дополнительные сальники
