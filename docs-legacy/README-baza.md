# Baza - Учебник-справочник по подшипникам

*Этот файл является заглушкой для документации из репозитория ArtemFilin1990/Baza*

## Оригинальное содержимое

Репозиторий Baza содержал:
- 122 статьи учебника-справочника
- CSV данные: 43,000+ записей
- Python-скрипты для обработки данных
- Модульная структура обучения

## Текущее размещение

Содержимое из репозитория Baza теперь находится в:
- `knowledge-base/` - учебные материалы и структура разделов
- `data/csv/` - структурированные CSV данные
- `data/schemas/` - JSON схемы
- `data/sources/` - исходные данные

См. [MIGRATION_GUIDE.md](../MIGRATION_GUIDE.md) для подробностей.
