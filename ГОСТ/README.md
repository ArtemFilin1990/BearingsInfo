# ГОСТ база

## Определение
Свод материалов по подшипникам, выпускаемым по системе ГОСТ 3189-89, включая обозначения, классификацию, материалы и эксплуатационные параметры.

## Назначение
- Оперативная расшифровка обозначений ГОСТ без обращения к ISO.
- Подбор подшипников для отечественных стандартов и проверка совместимости с аналогами.
- Обучение специалистов на основе учебного пособия «ПОДШИПНИКИ ГОСТ.docx».

## Конструкция / особенности
- Основное обозначение по ГОСТ состоит из семи знаков: тип, серия ширины, серия диаметров, код внутреннего диаметра.
- Для кодов 00–03 применяются специальные значения 10/12/15/17 мм; для 04–96 диаметр рассчитывается как код × 5 мм.
- Дополнительные таблицы включают материалы деталей, классы точности (0, 6, 5, 4, 2), группы зазоров (C2–C5) и перечень смазок (С1–С29).

## Примеры
- **6205** — тип 6 (радиальный шариковый), серия диаметров 2 (лёгкая), серия ширины 0, код 05 → d = 25 мм.
- **7805** — тип 7 (конический роликовый), серия диаметров 8 (особо лёгкая), код 05 → d = 25 мм, ширина по серии 0.
- **180205** — двухрядный самоустанавливающийся шариковый подшипник, серия диаметров 8, код 205.

## Примечания и ограничения
- При выборе аналогов обязательно учитывать классы точности и группы зазора: они могут не указываться в базовом обозначении.
- Для смазок используйте перечень С1–С29 из учебного пособия, без округлений и с проверкой температурных диапазонов.
- Обновления разделов ГОСТ фиксируются в `extracted/INDEX.md` с указанием исходного документа.
