# Литература

Раздел содержит ссылки и информацию о технической литературе по подшипникам.

## Содержание раздела

Материалы о:
- Технических справочниках
- Учебной литературе
- Каталогах производителей
- Онлайн ресурсах

## Основные материалы

- **Техническая литература о подшипниках. Книги, учебники, справочники, информационные материалы по подшипникам** - обзор доступной литературы

## Типы литературы

### Справочники
- Справочники по подшипникам качения
- Справочники по допускам и посадкам
- Таблицы размеров и характеристик
- Справочники по смазочным материалам

### Учебные пособия
- Учебники по деталям машин
- Пособия по расчету подшипниковых узлов
- Методические материалы

### Стандарты
- ГОСТ
- ISO
- DIN
- Отраслевые стандарты

### Каталоги
- Каталоги производителей подшипников
- Каталоги сопутствующих компонентов
- Онлайн каталоги

## Онлайн ресурсы

- Сайты производителей с технической документацией
- Базы данных стандартов
- Онлайн калькуляторы
- Обучающие материалы

## Рекомендуемая литература

Для углубленного изучения рекомендуется обратиться к:
- Классическим справочникам по подшипникам
- Стандартам ГОСТ и ISO
- Каталогам ведущих производителей (SKF, FAG, NSK)
