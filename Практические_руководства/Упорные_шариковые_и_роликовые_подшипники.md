# Упорные шариковые и роликовые подшипники

## Общие сведения

Упорные подшипники — тип подшипников качения, предназначенных для восприятия преимущественно осевых (упорных) нагрузок. Они могут быть шариковыми или роликовыми, одно- или двухсторонними.

## Типы упорных подшипников

### Упорные шариковые

**Конструкция:**
- Два кольца (или одно кольцо + шайба)
- Шарики в качестве тел качения
- Сепаратор для удержания шариков

**Особенности:**
- Малое трение
- Высокая частота вращения
- Меньшая грузоподъемность по сравнению с роликовыми
- Чувствительность к радиальным нагрузкам

### Упорные роликовые

**Конструкция:**
- Два кольца (или кольцо + шайба)
- Цилиндрические, конические или игольчатые ролики
- Сепаратор

**Особенности:**
- Высокая осевая грузоподъемность
- Меньшая частота вращения
- Не воспринимают радиальные нагрузки
- Требуют точного монтажа

## Обозначение по ГОСТ

### Упорные шариковые

- **8** — упорные шариковые одинарные
- **9** — упорные шариковые двойные

Примеры:
- **8101** — упорный шариковый, d=12 мм
- **9102** — упорный шариковый двойной, d=15 мм

### Упорные роликовые

- **18** — упорные роликовые цилиндрические
- **19** — упорные роликовые конические

Примеры:
- **18205** — упорный роликовый цилиндрический, d=25 мм
- **19308** — упорный роликовый конический, d=40 мм

## Упорные шариковые подшипники

### Конструктивные особенности

#### Односторонние (тип 8)

Состоят из:
- Тугое кольцо (вращающееся вместе с валом)
- Свободное кольцо (неподвижное)
- Шарики
- Сепаратор

#### Двухсторонние (тип 9)

Состоят из:
- Среднее кольцо (вращающееся)
- Два наружных кольца (неподвижные)
- Два комплекта шариков
- Два сепаратора

### Угол контакта

Стандартные упорные шариковые подшипники имеют угол контакта α = 90°, что означает:
- Восприятие только чисто осевых нагрузок
- Непригодны для радиальных нагрузок
- Требуют наличия радиальной опоры

### Характеристики

**Преимущества:**
✓ Высокая частота вращения  
✓ Низкое трение  
✓ Простая конструкция  
✓ Низкая стоимость  
✓ Малые габариты по высоте  

**Ограничения:**
✗ Только осевые нагрузки в одном направлении  
✗ Не работают без осевой нагрузки (центробежные силы выбрасывают шарики)  
✗ Ограниченная грузоподъемность  
✗ Требуют наличия радиальной опоры  

### Области применения

- Станки (упорные опоры шпинделей)
- Коробки передач
- Червячные редукторы
- Лебедки и домкраты
- Поворотные механизмы кранов
- Сцепления автомобилей
- Велосипедные рулевые колонки

## Упорные роликовые цилиндрические (тип 18)

### Конструктивные особенности

Состоят из:
- Вал-шайба (тугое кольцо)
- Корпус-шайба (свободное кольцо)
- Цилиндрические ролики
- Сепаратор (штампованный или массивный)

### Особенности конструкции

- Ролики короткие, с малым соотношением длины к диаметру
- Торцы роликов плоские
- Контакт роликов с кольцами — линейный
- Разъемная конструкция

### Характеристики

**Преимущества:**
✓ Очень высокая осевая грузоподъемность  
✓ Компактные габариты  
✓ Жесткость узла  
✓ Простота монтажа  

**Ограничения:**
✗ Низкая частота вращения  
✗ Только осевые нагрузки  
✗ Чувствительность к перекосам  
✗ Требуют качественной смазки  

### Области применения

- Прокатные станы
- Прессы
- Подъемные краны (опоры поворотных платформ)
- Турбины
- Гидротурбины
- Тяжелые редукторы

## Упорные роликовые конические (тип 19)

### Конструктивные особенности

Состоят из:
- Вал-шайба с конической дорожкой
- Корпус-шайба с конической дорожкой
- Конические ролики
- Сепаратор

### Угол контакта

Угол контакта обычно составляет 45-60°, что позволяет:
- Воспринимать комбинированные нагрузки (осевые + небольшие радиальные)
- Обеспечить более плавную работу
- Снизить краевые напряжения

### Характеристики

**Преимущества:**
✓ Сверхвысокая осевая грузоподъемность  
✓ Восприятие небольших радиальных нагрузок  
✓ Компактность  
✓ Высокая жесткость  

**Ограничения:**
✗ Низкая частота вращения  
✗ Сложность изготовления  
✗ Высокая стоимость  

### Области применения

- Прокатные станы (упорные опоры валков)
- Мощные редукторы
- Вертикальные насосы
- Гидротурбины
- Экскаваторы (поворотные платформы)

## Упорно-радиальные шариковые подшипники

### Особенности

Специальный тип подшипников с углом контакта α = 60°:
- Воспринимают осевые и радиальные нагрузки
- Применяются при преобладании осевых нагрузок
- Могут работать при высоких скоростях

### Обозначение

Обозначаются как шариковые радиально-упорные с особым углом контакта.

## Монтаж и эксплуатация

### Правила монтажа

**Для шариковых:**
1. Определить вращающееся кольцо (обычно внутреннее — вал)
2. Обеспечить осевое крепление обоих колец
3. Проверить соосность
4. Обязательно наличие осевой нагрузки

**Для роликовых:**
1. Тщательная очистка посадочных мест
2. Контроль перпендикулярности опорных поверхностей
3. Равномерное затягивание крепежа
4. Проверка момента вращения

### Регулировка

- Упорные подшипники обычно устанавливаются с небольшим зазором
- Для конических — возможна регулировка преднатяга
- Контроль зазора по осевому люфту

### Смазка

**Пластичная смазка:**
- Заполнение 30-50% свободного пространства
- Для шариковых: литиевые смазки средней вязкости
- Для роликовых: густые смазки с противозадирными присадками

**Жидкая смазка:**
- Масляная ванна — для тяжелых условий
- Циркуляционная — для мощных установок
- Масляный туман — для высоких скоростей (только шариковые)

## Схемы установки

### С радиальными подшипниками

```
Радиальный ←→ Упорный
подшипник      подшипник
    ║              ╤
    ║              │
    ║═════Вал═════╧═══
```

- Радиальный воспринимает радиальные нагрузки
- Упорный воспринимает осевые нагрузки
- Обязательная схема для упорных подшипников

### Двусторонние упорные

- Воспринимают осевые нагрузки в обоих направлениях
- Компактное решение
- Применение: реверсивные механизмы

## Дефекты и неисправности

### Основные причины отказов

1. **Работа без осевой нагрузки** (для шариковых) — выброс шариков
2. **Перекосы** — неравномерный износ, разрушение
3. **Недостаток смазки** — задиры, перегрев
4. **Загрязнение** — абразивный износ
5. **Перегрузка** — деформация, выкрашивание

### Признаки неисправности

- Осевой люфт (износ дорожек)
- Повышенный шум
- Нагрев
- Заклинивание
- Вибрация

## Специальные типы

### Упорные игольчатые

- Минимальная высота
- Очень высокая грузоподъемность при малых габаритах
- Применение: автоматические коробки передач, сцепления

### Упорные сферические роликовые

- Бочкообразные ролики
- Компенсация перекосов
- Сверхвысокая грузоподъемность
- Применение: прокатные станы, гидротурбины
