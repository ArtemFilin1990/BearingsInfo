# Аналоги подшипников ISO – ГОСТ

## Определение

Таблица соответствия подшипников по системе ISO → ГОСТ — обратная справочная таблица для поиска отечественного аналога импортного подшипника. Для каждого обозначения ISO указан эквивалент ГОСТ и тип конструкции.

## Назначение таблицы

Таблица используется для:
- Поиска отечественного аналога импортного подшипника
- Подбора замены при переходе на российские комплектующие
- Составления спецификаций на базе ISO
- Проверки совместимости оборудования

## Примечание

В номере и аналоге подшипника два правых знака 00 меняются на реальные значения цифр (обозначение диаметра подшипника).

**Пример:** ISO 6200 = ГОСТ 200
- Подшипник 6205 (ISO) = 205 (ГОСТ)
- Подшипник 6210 (ISO) = 210 (ГОСТ)

## Ограничения таблицы

**Таблица не охватывает:**
- Специальные конструкции подшипников
- Подшипники с нестандартными размерами
- Модифицированные исполнения

**При замене учитывать:**
- Дополнительные знаки обозначения (класс точности, зазоры)
- Материалы сепараторов (могут различаться)
- Смазку и уплотнения
- Условия эксплуатации
- Доступность на рынке
