# 1.3.5 Классификация подшипников

## Цель

Систематизировать типы подшипников качения по различным классификационным признакам для облегчения выбора и понимания конструктивных особенностей.

## Ключевые термины

- **Тела качения** — элементы, передающие нагрузку между кольцами (шарики, ролики, иглы)
- **Радиальная нагрузка** — нагрузка, направленная перпендикулярно оси вращения
- **Осевая (упорная) нагрузка** — нагрузка, направленная вдоль оси вращения
- **Самоустановка** — способность компенсировать перекосы вала относительно корпуса
- **Разъёмность** — возможность разборки подшипника на отдельные элементы

## Суть

### 1. Классификация по форме тел качения

#### 1.1. Шариковые подшипники

**Тела качения**: шарики (сферы)

**Преимущества:**
- Низкое трение → высокие скорости вращения
- Универсальность применения
- Низкий момент трения
- Доступная цена

**Недостатки:**
- Меньшая грузоподъёмность по сравнению с роликовыми
- Чувствительность к ударным нагрузкам

**Основные типы:**
- Радиальные однорядные (6xxx)
- Радиально-упорные (7xxx, 36xxx)
- Сферические двухрядные (1xxx)
- Упорные (8xxx)

#### 1.2. Роликовые подшипники

**Тела качения**: ролики (цилиндры, конусы, бочки)

**Преимущества:**
- Высокая грузоподъёмность
- Лучше переносят ударные нагрузки
- Большая контактная площадь

**Недостатки:**
- Выше трение → ниже предельная скорость
- Сложнее в производстве → дороже

**Подтипы:**

| Тип ролика | Форма | Подшипник | Применение |
|------------|-------|-----------|------------|
| Цилиндрический | Цилиндр | N, NU, NJ, NUP (2xxx) | Высокие радиальные нагрузки |
| Конический | Усечённый конус | 7xxx, 30xxx, 32xxx | Комбинированные нагрузки |
| Бочкообразный | Выпуклый цилиндр | 3xxx (сферические) | Перекосы, ударные нагрузки |
| Игольчатый | Тонкий длинный цилиндр | 4xxx, NK, RNA | Компактные узлы |
| Витые ролики | Спиральная навивка | 5xxx | Специальные применения |

### 2. Классификация по направлению воспринимаемой нагрузки

#### 2.1. Радиальные подшипники

**Назначение**: воспринимают нагрузку перпендикулярно оси вращения

**Типы:**
- Шариковые радиальные однорядные (6xxx) — универсальные
- Роликовые цилиндрические (N, NU, NJ) — высокие радиальные нагрузки
- Сферические роликовые (3xxx) — с компенсацией перекосов
- Игольчатые (4xxx) — компактные

**Применение**: электродвигатели, редукторы, насосы, вентиляторы

#### 2.2. Упорные подшипники

**Назначение**: воспринимают нагрузку вдоль оси вращения

**Типы:**
- Упорные шариковые (8xxx) — средние осевые нагрузки
- Упорные роликовые цилиндрические (9xxx) — большие осевые нагрузки
- Упорные сферические (293xx) — осевые + перекосы

**Применение**: вертикальные валы, червячные редукторы, домкраты

#### 2.3. Радиально-упорные подшипники

**Назначение**: воспринимают комбинированную нагрузку (радиальную + осевую)

**Типы:**
- Радиально-упорные шариковые (7xxx, 36xxx) — средние нагрузки, высокие скорости
- Конические роликовые (30xxx, 32xxx) — тяжёлые условия
- Сферические роликовые (могут воспринимать небольшую осевую)

**Применение**: автомобильные ступицы, шпиндели станков, насосы

### 3. Классификация по количеству рядов тел качения

#### 3.1. Однорядные

**Признак**: один ряд тел качения

**Примеры:**
- 6205 (шариковый радиальный)
- NU 308 (роликовый цилиндрический)
- 7205 (радиально-упорный)

**Преимущества**: компактность, низкое трение, высокие скорости

#### 3.2. Двухрядные

**Признак**: два ряда тел качения

**Примеры:**
- 3205 (сферический роликовый)
- 1205 (сферический шариковый)
- 4205 (радиально-упорный двухрядный)

**Преимущества**: повышенная грузоподъёмность, компактность по ширине

#### 3.3. Многорядные

**Признак**: три и более рядов (редко)

**Примеры**: специальные подшипники для тяжёлого оборудования

### 4. Классификация по способности самоустановки

#### 4.1. Несамоустанавливающиеся

**Признак**: не компенсируют перекосы

**Типы:**
- Радиальные шариковые (6xxx)
- Роликовые цилиндрические (2xxx)
- Конические роликовые (7xxx, 30xxx)
- Радиально-упорные (36xxx)

**Требование**: точная соосность вала и корпуса

#### 4.2. Самоустанавливающиеся

**Признак**: компенсируют перекосы вала (обычно до 2-4°)

**Типы:**
- Сферические роликовые (3xxx)
- Сферические шариковые (1xxx)
- Шарнирные (ШС, ШЛ)

**Преимущества**: работа при деформации корпуса, вибрации, неточной установке

### 5. Классификация по конструктивному исполнению

#### 5.1. Неразборные

**Признак**: кольца и тела качения образуют единый узел

**Типы**: большинство шариковых радиальных

#### 5.2. Разборные (сепарабельные)

**Признак**: кольца можно отделить друг от друга

**Типы:**
- Роликовые цилиндрические (N, NU, NJ)
- Конические роликовые (полностью разборные)
- Игольчатые

**Преимущества**: упрощённый монтаж, возможность регулировки зазора

### 6. Классификация по наличию защитных элементов

#### 6.1. Открытые

**Признак**: без уплотнений и защитных шайб

**Применение**: системы с внешней смазкой и уплотнениями

#### 6.2. Защищённые

**Типы защиты:**
- **Z, 2Z** — металлические защитные шайбы (пыль)
- **RS, 2RS** — резиновые уплотнения (влага, грязь)
- **2RZ** — низкомоментные уплотнения (высокие скорости)

**Преимущество**: не требуют обслуживания, заводская смазка на весь срок службы

### 7. Классификация по размерным сериям

#### По серии диаметров (D/d)

| Серия | Обозначение | Относительный размер D |
|-------|-------------|------------------------|
| Особо лёгкая | 0, 1 | Минимальный |
| Лёгкая | 2 | Малый |
| Средняя | 3 | Средний |
| Тяжёлая | 4 | Большой |

#### По серии ширин (B/d)

| Серия | Обозначение | Относительная ширина |
|-------|-------------|---------------------|
| Особо узкая | 0, 1 | Минимальная |
| Нормальная | 2 | Средняя |
| Широкая | 3, 4 | Большая |

### 8. Специальные категории

#### 8.1. Прецизионные

- Классы точности: P6, P5, P4, P2
- Применение: шпиндели станков, измерительное оборудование
- Особенности: минимальные допуски, низкая вибрация

#### 8.2. Высокотемпературные

- Суффикс: HT
- Температура: до +150...+200°C
- Специальная термообработка и смазка

#### 8.3. Нержавеющие

- Материал: сталь AISI 440C, 304, 316
- Применение: пищевая, химическая промышленность
- Устойчивость к коррозии

#### 8.4. Гибридные

- Кольца: сталь
- Тела качения: керамика (Si₃N₄)
- Преимущества: высокие скорости, низкая плотность, диэлектрические свойства

#### 8.5. Корпусные (housed units)

- UCF, UCP, UCFL и др.
- Интегрированы в корпус
- Упрощённый монтаж

## Примеры

### Пример 1: Выбор по типу нагрузки

**Задача**: вал редуктора, радиальная нагрузка 10 кН, осевая 0.5 кН

**Решение**: радиально-упорный или конический роликовый (например, 30310)

### Пример 2: Выбор при перекосах

**Задача**: вал конвейера с возможными деформациями рамы до 2°

**Решение**: сферический роликовый (например, 3310) — самоустанавливающийся

### Пример 3: Компактный узел

**Задача**: диаметр вала 20 мм, ограничение по диаметру корпуса 42 мм

**Решение**: игольчатый подшипник (например, NK 20/20) — минимальный D при заданном d

### Пример 4: Высокие скорости

**Задача**: шпиндель станка, 12000 об/мин

**Решение**: радиально-упорный шариковый класса P4 (например, 7208 BECBP P4)
