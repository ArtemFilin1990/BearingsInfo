# Префиксы ISO

## Определение
Префиксы в обозначениях ISO размещаются перед основным номером подшипника и указывают на конструктивные модификации, тип ряда или принадлежность к комплектующим (втулкам, гайкам, специальным исполнениям).

## Назначение
- Быстро выделять конструктивные особенности до анализа основного номера.
- Исключать ошибки при подборе втулок и специальных исполнений.
- Ускорять проверку совместимости посадок и монтажных комплектов.

## Конструкция / особенности
- Для цилиндрических роликовых подшипников префиксы задают конфигурацию буртов (N, NU, NJ, NUP, NH, NF).
- Специальные подшипники могут кодироваться префиксами с номером чертежа (Bxxx), а также маркировками F, G, K в соответствии с заводским обозначением.
- Монтажные втулки и гайки кодируются буквами H/AH/AHX/OH и идут отдельно от основного обозначения подшипника.

## Обозначения и расшифровка

### Цилиндрические роликовые
| Префикс | Конструкция | Особенности применения |
|---------|-------------|------------------------|
| N | Бурты на внутреннем кольце | Осевая фиксация внутреннего кольца |
| NU | Бурты на наружном кольце | Осевая фиксация наружного кольца |
| NJ | Бурты на внутреннем + один бортик на наружном кольце | Фиксация в одну сторону |
| NUP | NU + свободное кольцо | Двусторонняя осевая фиксация |
| NH | Втулка вместо внутреннего кольца | Для валов без бурта |
| NF | Два бурта на внутреннем кольце + два на наружном | Полная осевая фиксация |

### Специальные исполнения производителей
| Префикс | Назначение |
|---------|------------|
| B + номер чертежа | Специальный подшипник по заводскому чертежу (6–7 цифр) |
| F / G / K | Специальные исполнения по каталогу производителя (обозначения перед основным номером) |

### Втулки и гайки для монтажа
| Префикс | Деталь | Особенности |
|---------|--------|-------------|
| H | Закрепительная втулка (метрическая) | Стандартная метрическая коническая втулка |
| HA | Закрепительная втулка (дюймовая 1/16") | Для дюймовых валов с шагом 1/16" |
| HE | Закрепительная втулка (дюймовая 1/4") | Для дюймовых валов с шагом 1/4" |
| HS | Закрепительная втулка (дюймовая 1/8") | Для дюймовых валов с шагом 1/8" |
| OH | Закрепительная втулка с масляными каналами | Подпитка маслом при монтаже |
| AH | Стяжная втулка | Стандартное исполнение |
| AHX | Стяжная втулка (ISO) | Соответствует размерам ISO |
| AOH | Стяжная втулка с масляными каналами | Облегчение монтажа/демонтажа |
| AOHX | Стяжная втулка (ISO) с масляными каналами | Стандартизированное исполнение с маслоподачей |
| HM | Гайка с трапецеидальной резьбой ISO | Для фиксации втулок |
| HML | Гайка уменьшенного сечения | Для ограниченного пространства |
| HMV | Гидравлическая гайка | Для гидравлического монтажа |

## Примеры
- **NUP 2205** — цилиндрический роликовый с дополнительным свободным кольцом, обеспечивающим двустороннюю фиксацию.
- **B** + «номер чертежа» — обозначает специальный подшипник по заводской документации (пример: B-xxxx перед базовым размером).
- **H310 + 21310 CCK** — комплект втулки и подшипника с коническим отверстием, где H310 задаёт метрическую втулку, а базовое обозначение сохраняется для самого подшипника.
- **AHX** перед серией — стяжная втулка стандарта ISO, выбирается под конкретный диаметр и конус подшипника.
- **HMV** с номером — гидравлическая гайка для монтажа втулок и подшипников требуемого размера.

## Примечания и ограничения
- Префиксы для специальных исполнений (B, F, G, K) зависят от заводской номенклатуры и требуют сверки с каталогом производителя.
- Втулки и гайки указываются в заказе отдельно от подшипника, но должны соответствовать его конусу и серии.
- При подборе цилиндрических роликовых подшипников префикс задаёт допустимые осевые перемещения; несоответствие приведёт к неправильной фиксации узла.
