# 1.1.1 Система условных обозначений подшипников (ГОСТ)

## Цель

Объяснить принципы построения условного обозначения подшипников качения по ГОСТ, чтобы специалисты могли правильно читать маркировку и подбирать необходимые изделия.

## Ключевые термины

- **Основное обозначение** — числовой код, определяющий тип, серию и размер подшипника
- **Серия диаметров** — соотношение наружного и внутреннего диаметров подшипника
- **Серия ширин** — соотношение ширины подшипника к его диаметру
- **Внутренний диаметр (d)** — диаметр отверстия подшипника
- **Дополнительные знаки** — буквенно-цифровые коды, указывающие конструктивные особенности, класс точности, зазор и другие характеристики
- **Префикс** — буквенное обозначение перед основным номером (например, N, NU, NJ для роликовых)
- **Суффикс** — буквенно-цифровой код после основного номера (например, 2RS, C3, P5)

## Суть

### Структура обозначения по ГОСТ

Полное обозначение подшипника состоит из основного обозначения и дополнительных знаков:

```
[Префикс] XXXXXX [Суффикс]
          ││││││
          │││││└─ Две последние цифры: внутренний диаметр (×5 для d≥20мм)
          ││││└── Серия ширин (0, 1, 2, 3...)
          │││└─── Серия диаметров (0-4: особо лёгкая-тяжёлая)
          ││└──── Конструктивная разновидность
          │└───── Тип подшипника (0-9)
          └────── Серия подшипника (может отсутствовать)
```

### Расшифровка основного обозначения

#### Тип подшипника (первая или две первые цифры)

| Код | Тип подшипника | Тела качения |
|-----|----------------|--------------|
| 0 | Двухрядный радиально-упорный шариковый | Шарики |
| 1 | Шариковый сферический двухрядный | Шарики |
| 2 | Роликовый с короткими цилиндрическими роликами | Ролики |
| 3 | Роликовый сферический двухрядный | Ролики |
| 4 | Игольчатый или роликовый с длинными роликами | Иглы/ролики |
| 5 | Роликовый с витыми роликами | Ролики |
| 6 | Шариковый радиальный однорядный | Шарики |
| 7 | Роликовый конический | Ролики |
| 8 | Упорный шариковый | Шарики |
| 9 | Упорный роликовый | Ролики |

#### Серия диаметров (третья цифра справа или вторая слева)

| Код | Серия | Соотношение D/d |
|-----|-------|-----------------|
| 0, 1 | Особо лёгкая | Минимальное |
| 2 | Лёгкая | Малое |
| 3 | Средняя | Среднее |
| 4 | Тяжёлая | Большое |

#### Серия ширин (вторая цифра справа)

| Код | Серия ширин | Относительная ширина |
|-----|-------------|---------------------|
| 0, 1 | Особо узкая/узкая | Малая |
| 2 | Нормальная | Средняя |
| 3 | Широкая | Большая |
| 4 | Особо широкая | Очень большая |

#### Внутренний диаметр

- **Для d от 20 до 495 мм**: две последние цифры номера, умноженные на 5
  - Например: **05** = 5 × 5 = 25 мм
  - Например: **12** = 12 × 5 = 60 мм
- **Для d < 20 мм**: указывается фактический размер
  - **00** = 10 мм
  - **01** = 12 мм
  - **02** = 15 мм
  - **03** = 17 мм
- **Для d ≥ 500 мм**: указывается через дробь (например, 500/600)

### Дополнительные знаки (суффиксы)

#### Защита и уплотнения

| Суффикс | Значение |
|---------|----------|
| Z | Одна металлическая защитная шайба |
| 2Z, ZZ | Две металлические защитные шайбы |
| RS | Одно резиновое уплотнение (контактное) |
| 2RS | Два резиновых уплотнения |

#### Радиальный зазор

| Суффикс | Зазор |
|---------|-------|
| C2 | Меньше нормального |
| C0 | Нормальный (обычно не указывается) |
| C3 | Больше нормального |
| C4 | Большой |
| C5 | Особо большой |

#### Класс точности

| Суффикс | Класс точности по ГОСТ |
|---------|------------------------|
| (без обозначения) | Класс 0 (нормальный) |
| 6 или P6 | Класс 6 (повышенный) |
| 5 или P5 | Класс 5 (высокий) |
| 4 или P4 | Класс 4 (особо высокий) |
| 2 | Класс 2 (прецизионный) |

#### Материал и конструкция

| Суффикс | Значение |
|---------|----------|
| E | Усиленная конструкция |
| Е | Оптимизированная внутренняя геометрия |
| Н | Нержавеющая сталь |
| К | Конический хвостовик для установки на конический вал |
| N, NR | Канавка на наружном кольце (NR — с кольцом) |

## Примеры

### Пример 1: Обозначение 6205-2RS C3

- **6** — шариковый радиальный однорядный
- **2** — лёгкая серия диаметров
- **0** — нормальная серия ширин
- **05** — внутренний диаметр 25 мм (5 × 5)
- **2RS** — два резиновых уплотнения
- **C3** — увеличенный радиальный зазор

**Применение**: Электродвигатели, вентиляторы, насосы — стандартное исполнение с защитой от пыли и влаги.

### Пример 2: Обозначение NU 308 E

- **N** — префикс для роликового цилиндрического
- **U** — с двумя бортами на наружном кольце
- **3** — средняя серия диаметров
- **08** — внутренний диаметр 40 мм (8 × 5)
- **E** — усиленная конструкция (оптимизированная геометрия)

**Применение**: Редукторы, станки — выдерживает высокие радиальные нагрузки.

### Пример 3: Обозначение 7205 AC P4

- **7** — роликовый конический
- **2** — лёгкая серия диаметров
- **05** — внутренний диаметр 25 мм
- **AC** — угол контакта (для конических)
- **P4** — класс точности 4 (прецизионный)

**Применение**: Шпиндели металлорежущих станков, прецизионное оборудование.

### Пример 4: Обозначение 30205

- **3** — роликовый конический (по ГОСТ может обозначаться префиксом или первой цифрой)
- **02** — серия
- **05** — внутренний диаметр 25 мм

**Применение**: Автомобильные ступицы, коробки передач.

### Пример 5: Обозначение 180205 C3

- **1** — шариковый сферический двухрядный
- **8** — конструктивная разновидность
- **02** — серия
- **05** — внутренний диаметр 25 мм
- **C3** — увеличенный зазор

**Применение**: Узлы с перекосами, вибрационное оборудование.
