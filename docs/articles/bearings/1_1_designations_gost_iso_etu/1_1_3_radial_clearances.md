# 1.1.3 Радиальные зазоры в подшипниках качения

## Группы радиальных зазоров

Радиальный зазор — расстояние, на которое можно переместить одно кольцо подшипника относительно другого в радиальном направлении.

### Обозначения по ГОСТ

| Группа | Величина зазора | Применение |
|--------|----------------|------------|
| **C1** | Меньше C2 | Специальные применения |
| **C2** | Меньше нормального | Прецизионные узлы, малые температуры |
| **C0** | Нормальный (без обозначения) | Стандартные применения |
| **C3** | Больше нормального | Высокие температуры, натяг на валу |
| **C4** | Больше C3 | Очень высокие температуры |
| **C5** | Больше C4 | Экстремальные температуры |

## Выбор зазора

### Влияние температуры
При нагреве внутреннее кольцо расширяется сильнее → зазор уменьшается.

**Рекомендации**:
- Δt < 10°C → C0
- Δt = 10-30°C → C3
- Δt > 30°C → C4

### Влияние посадки
Плотная посадка на вал уменьшает зазор.

### Типичные применения

- **C2**: Прецизионные станки, подшипники качения
- **C0**: Электродвигатели, насосы, редукторы
- **C3**: Электродвигатели с повышенной температурой, вентиляторы
- **C4**: Сушильные барабаны, печи

## Измерение зазора

Замеряется в свободном состоянии (до установки) индикатором часового типа.
