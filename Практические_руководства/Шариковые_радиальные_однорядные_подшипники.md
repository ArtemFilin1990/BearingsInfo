# Шариковые радиальные однорядные подшипники

## Общие сведения

Шариковые радиальные однорядные подшипники — наиболее распространенный тип подшипников качения. Они состоят из наружного и внутреннего колец с дорожками качения, комплекта шариков и сепаратора, удерживающего шарики на равном расстоянии друг от друга.

## Конструктивные особенности

### Основные элементы

- **Внутреннее кольцо** — посажено на вал, имеет дорожку качения
- **Наружное кольцо** — устанавливается в корпус, имеет дорожку качения
- **Шарики** — тела качения, обеспечивающие минимальное трение
- **Сепаратор** — удерживает шарики на определенном расстоянии друг от друга

### Типы конструкций

1. **С канавкой для ввода шариков** — стандартная конструкция
2. **С заполнением большим числом шариков** — для восприятия больших нагрузок
3. **С защитными шайбами или уплотнениями** — серии Z, ZZ, RS, 2RS
4. **С увеличенным внутренним диаметром** — серии Е

## Обозначение по ГОСТ

Шариковые радиальные однорядные подшипники обозначаются цифрой **0** (нулем) в коде типа подшипника или отсутствием этой цифры в обозначении.

### Примеры обозначений

- **180** — серия диаметров 0, серия ширин 1, внутренний диаметр из таблицы
- **6000** — серия диаметров 0, серия ширин 6
- **607** — d=7 мм, серия 60
- **6205** — d=25 мм, серия 62
- **6308 2RS** — d=40 мм, серия 63, с двумя уплотнениями

## Размерные серии

### По ширине
- **Особо легкая** (серия 1)
- **Сверхлегкая** (серия 8, 9)
- **Легкая** (серия 0)
- **Средняя** (серия 2, 3)
- **Тяжелая** (серия 4)

### По диаметру
- **Особо легкая** (серия 7, 8, 9)
- **Легкая** (серия 0, 1)
- **Средняя** (серия 2, 3)
- **Тяжелая** (серия 4)

## Основные серии

| Серия | Описание | Применение |
|-------|----------|------------|
| 60 | Сверхлегкая серия | Приборы, малые нагрузки |
| 62 | Легкая серия | Общего назначения |
| 63 | Средняя серия | Повышенные нагрузки |
| 64 | Тяжелая серия | Большие нагрузки |
| 160 | Особо легкая узкая | Специальное применение |

## Характеристики и возможности

### Преимущества

✓ Низкое трение и нагрев  
✓ Высокая частота вращения  
✓ Малый вес и компактность  
✓ Низкий уровень шума  
✓ Простота монтажа и обслуживания  
✓ Широкий диапазон типоразмеров  

### Ограничения

✗ Меньшая грузоподъемность по сравнению с роликовыми  
✗ Чувствительность к перекосам  
✗ Ограниченная ударная прочность  

## Воспринимаемые нагрузки

- **Радиальные нагрузки** — основные (100%)
- **Осевые нагрузки** — небольшие (до 50% от радиальной грузоподъемности)
- **Комбинированные нагрузки** — радиальная + небольшая осевая

## Модификации и исполнения

### По уплотнениям
- **Открытый тип** — без защиты
- **Z, ZZ** — с защитными шайбами (с одной или двух сторон)
- **RS, 2RS** — с резиновыми уплотнениями (с одной или двух сторон)
- **N, NR** — с канавкой и стопорным кольцом на наружном кольце

### По материалу
- **Стандартные** — подшипниковая сталь ШХ15
- **Нержавеющие** — коррозионностойкая сталь
- **Гибридные** — керамические шарики + стальные кольца
- **Высокотемпературные** — специальные стали

### По точности
- **ГОСТ**: П0 (нормальная), П6, П5, П4, П2
- **ISO/ABEC**: P0, P6, P5, P4, P2
- **ABEC**: ABEC 1, 3, 5, 7, 9

## Рекомендации по выбору

### Критерии подбора

1. **Нагрузка** — выбор серии по грузоподъемности
2. **Частота вращения** — проверка предельных оборотов
3. **Условия эксплуатации** — выбор уплотнений и смазки
4. **Точность** — класс точности в зависимости от требований
5. **Долговечность** — расчет ресурса

### Рекомендуемые посадки

**Для внутреннего кольца (на вал):**
- Легкие нагрузки: k6, js6
- Нормальные нагрузки: m6, n6
- Тяжелые нагрузки: p6, r6

**Для наружного кольца (в корпус):**
- Нормальные условия: H7, G7
- С осевым смещением: F7
- Неразъемный корпус: K7, M7

## Монтаж и эксплуатация

### Правила монтажа

1. Не допускать ударов по кольцам и телам качения
2. Запрессовывать через то кольцо, которое имеет натяг
3. Использовать оправки и съемники
4. Проверять правильность посадки
5. Не допускать перекосов при установке

### Смазка

- **Пластичная смазка** — для большинства применений (заполнение 30-50% свободного объема)
- **Жидкая смазка** — для высоких скоростей
- **Необслуживаемые** — подшипники с уплотнениями и заводской смазкой

### Хранение

- В заводской упаковке
- В сухом помещении
- Горизонтальное положение
- Защита от загрязнений
- Переконсервация при длительном хранении (более 2 лет)

## Дефекты и неисправности

### Основные причины отказов

1. **Усталостное выкрашивание** — естественный износ
2. **Задиры** — недостаток смазки, перегрузка
3. **Коррозия** — попадание влаги
4. **Износ** — абразивные частицы в смазке
5. **Электроэрозия** — прохождение тока через подшипник

### Признаки неисправности

- Повышенный шум и вибрация
- Нагрев подшипникового узла
- Увеличение люфта
- Заклинивание

## Аналоги

### ГОСТ ↔ ISO

Подшипники ГОСТ имеют прямые аналоги ISO/DIN с такими же основными размерами:
- ГОСТ 180 → ISO/DIN 618/0
- ГОСТ 6000 → ISO/DIN 6000
- ГОСТ 6205 → ISO/DIN 6205
- ГОСТ 6308 → ISO/DIN 6308

Дополнительные обозначения могут отличаться.
