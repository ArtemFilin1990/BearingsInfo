# Как выбрать подшипник по типу нагрузки

![Сложность](https://img.shields.io/badge/сложность-⭐⭐⭐⭐☆-red.svg)
![Время](https://img.shields.io/badge/время-20_минут-blue.svg)

> Радиальная нагрузка, осевая нагрузка, комбинированная нагрузка, блок-схема выбора типа, расчет эквивалентной нагрузки

## 🎯 Цель

Научиться **правильно выбирать тип подшипника** в зависимости от направления и характера действующих нагрузок.

## 🛠️ Необходимые инструменты

- Схема узла (чертеж или эскиз)
- Данные о нагрузках (Fr, Fa)
- Калькулятор
- Каталог подшипников

---

## 📋 Типы нагрузок

### 1. Радиальная нагрузка (Fr)

**Направление:** Перпендикулярно оси вала

```
      ↓ Fr (радиальная)
      │
  ════╪════  ← Вал
      │
```

**Примеры:**
- Вес шкива на валу
- Натяжение ремня
- Вес ротора
- Поперечная сила

---

### 2. Осевая нагрузка (Fa)

**Направление:** Вдоль оси вала

```
  ←──── Fa (осевая)
  
  ══════════  ← Вал
```

**Примеры:**
- Усилие подачи в станке
- Распор шестерен
- Винтовая передача
- Осевая сила от косозубых передач

---

### 3. Комбинированная нагрузка (Fr + Fa)

**Направление:** Одновременно радиальная и осевая

```
      ↓ Fr
      │
  ←───╪──── Fa
      │
  ════╬════  ← Вал
```

**Примеры:**
- Конические шестерни
- Винтовые конвейеры
- Редукторы с косозубыми колесами
- Буровые установки

---

## 🔍 Блок-схема выбора типа подшипника

```
└──────────┬──────────────────┘
           │
      ┌────┴────┐
      │         │
   Только Fr  Только Fa  Fr + Fa (комбинированная)
      │         │         │
      │         │         │
      ▼         ▼         ▼
┌─────────┐ ┌──────┐  ┌───────────┐
│Радиальная│Осевая │ │Комбинир.  │

     ▼          ▼            ▼
  ┌──────┐  ┌──────┐   ┌──────────┐
  │Шариковый│ │Упорный│  │Fa/Fr < 1 │
  │радиальный│ │шариковый│ │или > 1?  │
  │         │ │роликовый│ └─────┬────┘
  │Роликовый│ │         │       │
  │цилиндр. │ │         │   ┌───┴───┐
  │         │ │         │   │       │
  │Игольчатый││         │ Fa/Fr<1 Fa/Fr>1

                     ┌──────────┐ ┌────────┐
                     │Радиально-│ │Конический│

                     │Конический│ │Упорно- │
                     │(парами) │ │радиальный│
                     └─────────┘ └────────┘
```

## 📊 Рекомендации по выбору типа

### Только радиальная нагрузка (Fa = 0)

| Тип подшипника | Когда выбирать | Преимущества | Недостатки |
|----------------|----------------|--------------|------------|
| **Шариковый радиальный** | Стандартные условия, средние нагрузки | ✓ Универсальность<br>✓ Низкая цена<br>✓ Низкое трение | ✗ Средняя грузоподъемность |
| **Роликовый цилиндрический** | Высокие радиальные нагрузки | ✓ Высокая C<br>✓ Точность<br>✓ Жесткость | ✗ Не воспринимает Fa<br>✗ Дороже |
| **Игольчатый** | Малое радиальное пространство | ✓ Компактность<br>✓ Высокая C | ✗ Малая скорость<br>✗ Чувствителен к перекосам |
| **Роликовый сферический** | Большие нагрузки + перекосы вала | ✓ Самоустановка<br>✓ Очень высокая C | ✗ Дорогой<br>✗ Большие габариты |

---

### Только осевая нагрузка (Fr = 0 или мала)

| Тип подшипника | Когда выбирать | Преимущества | Недостатки |
|----------------|----------------|--------------|------------|
| **Упорный шариковый** | Малые/средние Fa, средние скорости | ✓ Дешевый<br>✓ Компактный | ✗ Низкая скорость<br>✗ Не воспринимает Fr |
| **Упорный роликовый** | Большие Fa, малые скорости | ✓ Очень высокая C₀<br>✓ Жесткость | ✗ Очень низкая скорость<br>✗ Не воспринимает Fr |

**⚠️ Важно:** Упорные подшипники **НЕ МОГУТ** работать под радиальной нагрузкой!

---

### Комбинированная нагрузка (Fr + Fa)

#### Когда Fa/Fr < 1 (радиальная преобладает)

| Тип подшипника | Когда выбирать | Угол контакта | Fa/Fr допустимое |
|----------------|----------------|---------------|------------------|
| **Шариковый радиально-упорный** | Средние Fr + небольшие Fa | 15°-40° | < 0.5 |
| **Роликовый конический** | Высокие Fr + средние Fa | 10°-30° | < 1.5 |

#### Когда Fa/Fr > 1 (осевая преобладает)

| Тип подшипника | Когда выбирать | Особенности |
|----------------|----------------|-------------|
| **Упорно-радиальный** | Большие Fa + малые Fr | Fr до 25% от Fa |
| **Конический (с большим углом)** | Fa >> Fr | Устанавливаются парами |

## 🧮 Расчет эквивалентной нагрузки

### Зачем нужна эквивалентная нагрузка?

**Проблема:** Подшипник работает под **комбинированной** нагрузкой (Fr + Fa), но в каталоге указана грузоподъемность **C** для чисто радиальной нагрузки.

**Решение:** Рассчитать **эквивалентную динамическую нагрузку P**, которая учитывает обе составляющие.

### Формула

```
P = X·Fr + Y·Fa

где:
P  — эквивалентная динамическая нагрузка [Н или кН]
Fr — радиальная нагрузка [Н или кН]
Fa — осевая нагрузка [Н или кН]
X  — коэффициент радиальной нагрузки
Y  — коэффициент осевой нагрузки
```

**Коэффициенты X и Y** зависят от:
- Типа подшипника
- Соотношения Fa/Fr
- Fa/(C₀·e), где C₀ — статическая грузоподъемность, e — коэффициент

---

### Пример 1: Шариковый радиальный подшипник

**Дано:**
- Подшипник: 6205
- Fr = 2000 Н
- Fa = 500 Н
- C₀ = 7800 Н (из каталога)

**Решение:**

1. Определяем Fa/Fr = 500/2000 = **0.25**

2. Для шариковых радиальных:
   - Если Fa/Fr ≤ e (обычно e ≈ 0.3):
     - X = 1, Y = 0
   - Если Fa/Fr > e:
     - X = 0.56, Y зависит от Fa/(C₀·e)

3. У нас 0.25 < 0.3, значит:
   - X = 1
   - Y = 0

4. **P = 1·2000 + 0·500 = 2000 Н**

**Вывод:** Осевая нагрузка мала, эквивалентная нагрузка равна радиальной.

---

### Пример 2: Радиально-упорный подшипник

**Дано:**
- Подшипник: 7205 (конический)
- Fr = 2000 Н
- Fa = 1500 Н
- C₀ = 14000 Н

**Решение:**

1. Fa/Fr = 1500/2000 = **0.75**

2. Для конических (e ≈ 0.35):
   - Fa/Fr = 0.75 > e
   - Используем: X = 0.4, Y = 1.6 (из таблиц каталога)

3. **P = 0.4·2000 + 1.6·1500 = 800 + 2400 = 3200 Н**

**Вывод:** Эквивалентная нагрузка выше радиальной из-за значительной осевой составляющей.

## ✅ Алгоритм выбора подшипника по нагрузке

### Шаг 1: Определите нагрузки

- [ ] Рассчитайте Fr (радиальную)
- [ ] Рассчитайте Fa (осевую)
- [ ] Определите характер (постоянная, переменная, ударная)

### Шаг 2: Определите соотношение

- [ ] Рассчитайте Fa/Fr
- [ ] Определите преобладающую нагрузку

### Шаг 3: Выберите тип по блок-схеме

- [ ] Fa = 0 → радиальный
- [ ] Fr = 0 → упорный
- [ ] Fa/Fr < 1 → радиально-упорный
- [ ] Fa/Fr > 1 → конический или упорно-радиальный

### Шаг 4: Рассчитайте эквивалентную нагрузку P

- [ ] Найдите коэффициенты X и Y
- [ ] Рассчитайте P = X·Fr + Y·Fa

### Шаг 5: Выберите размер подшипника

- [ ] Убедитесь, что C > P (с запасом!)
- [ ] Проверьте C₀ для статической нагрузки
- [ ] Проверьте ресурс по формуле L₁₀

## ⚠️ Частые ошибки

### Ошибка 1: Игнорирование осевой нагрузки

❌ **Неправильно:** "Fa мала, не буду учитывать"

✅ **Правильно:** Даже малая Fa может существенно сокращать ресурс! Всегда учитывайте.

---

### Ошибка 2: Использование упорного под радиальной нагрузкой

❌ **Неправильно:** Установить упорный подшипник под комбинированную нагрузку

✅ **Правильно:** Упорные **ТОЛЬКО** для осевой нагрузки! Для Fr + Fa нужен радиально-упорный.

---

### Ошибка 3: Неправильный расчет P

❌ **Неправильно:** P = Fr + Fa (простое сложение)

✅ **Правильно:** P = X·Fr + Y·Fa (с коэффициентами!)

---

## ❓ FAQ

**В: Что если Fr и Fa примерно равны?**
**О:** Выбирайте радиально-упорный или конический подшипник (парами).

**В: Можно ли шариковый радиальный под небольшую Fa?**
**О:** Да, если Fa/Fr < 0.3. Но лучше взять с запасом или радиально-упорный.

**В: Что если нагрузка ударная?**
**О:** Увеличьте расчетную нагрузку в 1.5-3 раза (коэффициент динамичности).
