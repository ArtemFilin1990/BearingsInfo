# Новая структура базы знаний (WIP)

## Цель
Создать единую русскоязычную структуру с разделами 01–09 и техническими слоями данных.

## Дерево разделов
```
/
├── 01_Подшипники_Общая_информация/
├── 02_Стандарты_и_маркировка/
├── 03_Типы_и_элементы/
├── 04_Расчет_и_параметры/
├── 05_Эксплуатация_и_обслуживание/
├── 06_Отказы_и_диагностика/
├── 07_Бренды_и_производители/
├── 08_Специальные_исполнения_и_справка/
├── 09_Сопутствующие_изделия/
├── _tables/
├── _data/
├── _db/
├── _tools/
├── _sources/
├── _assets/
└── _trash_review/
```

## Принципы
- Каждый подраздел — это папка с `README.md`.
- В `README.md` раздела подразделы перечислены ссылками по одной строке.
- Технические слои используют имена с подчёркиванием.
