# 1.8.3 Признаки неисправности ступичных подшипников

## Симптомы

1. **Гул, шум** при движении (усиливается с ростом скорости)
2. **Люфт колеса** (покачивание в вертикальной плоскости)
3. **Вибрация руля/кузова**
4. **Перегрев ступицы**
5. **Неравномерный износ шин**

## Проверка
- Поднять автомобиль
- Покачать колесо в вертикальной и горизонтальной плоскостях
- Прокрутить колесо (шум, заедание)

## Ресурс
- Оригинальные: 100000-200000 км
- Аналоги качественные: 80000-150000 км
