## Введение
Роликовые линейные опоры (cam followers) — опорные ролики на подшипниках для линейного перемещения по направляющим.

## Типы
1. **Стандартные** — цилиндрические ролики
2. **Игольчатые** — компактные
3. **Сферические** — компенсация несоосности

## Конструкция
- **Наружное кольцо** — толстостенное, закаленное
- **Подшипник** — игольчатый или роликовый
- **Ось** — с резьбой для крепления

## Применение
- Конвейеры
- Станки (каретки)
- Подъемники
- Ворота

## Преимущества
✅ Высокая грузоподъемность  
✅ Простота конструкции  
✅ Низкая стоимость  
✅ Компактность
