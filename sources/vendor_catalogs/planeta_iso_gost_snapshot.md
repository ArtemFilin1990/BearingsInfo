# planetapodshipnik.ru ISO↔ГОСТ table (offline snapshot)

## Purpose
Сохранять офлайн-таблицы соответствий ISO↔ГОСТ с planetapodshipnik.ru без внешних ссылок.

## Classification
- Независимый ресурс с таблицей соответствий.
- Используется для сопоставления обозначений и проверки d/D/B.

## Designation
- `SNAP_SRC_PLANETA_ISO_GOST` — выгрузка таблицы соответствий.

## Prefixes/Suffixes
- Префикс `SNAP_SRC_` — офлайн-слепок.
- Суффикс `_ISO_GOST` — таблица соответствий ISO↔ГОСТ.

## Examples
- Сохранить строки для серии 6200 с указанием ISO и ГОСТ обозначений, фиксировать дату выгрузки.

## Notes & Limitations
- Онлайн-доступ блокирован (403); требуется офлайн-выгрузка.
- Указывать контрольные суммы и дату при каждом обновлении.
