# Обозначение внутреннего диаметра подшипников

## Определение

Внутренний диаметр подшипника (d) — диаметр отверстия внутреннего кольца подшипника, соответствующий диаметру вала. Обозначается в маркировке подшипника последними цифрами основного условного обозначения по специальной системе кодирования.

## Правила кодирования

### Для подшипников с d < 10 мм

**Структура обозначения:**
```
X X X X 0 Y Z
        │ │ └─ Внутренний диаметр (1 цифра)
        │ └─── Серия диаметров
        └───── Обязательный ноль
```

**Примеры:**
- **25** → d = 5 мм
- **1000094** → d = 4 мм
- **1068** → d = 8 мм

### Для подшипников с d ≥ 10 мм

**Структура обозначения:**
```
X X X X Y Z Z
        │ └─┴─ Внутренний диаметр (2 цифры)
        └───── Серия диаметров
```

**Примеры:**
- **205** → 05 → d = 05×5 = **25 мм**
- **32205** → 05 → d = 05×5 = **25 мм**
- **6310** → 10 → d = 10×5 = **50 мм**

### Специальные диаметры

**Малые диаметры (0,6; 1,5; 2,5 мм):**

Обозначаются через дробь **"/число"**

**Примеры:**
- **7601/2,5** → d = 2,5 мм
- **4-107609/1,5ЮУТ** → d = 1,5 мм

**Диаметры 22, 28, 32 мм:**

Обозначаются через дробь **"/число"**

**Пример:**
- **602/32** → d = 32 мм

**Крупногабаритные (≥500 мм):**

Обозначаются через дробь **"/число"**

**Пример:**
- **10079/710** → d = 710 мм

## Примеры расчета диаметра

### Пример 1: Диаметр 1-9 мм

**Подшипник: 1008**
- Последняя цифра: **8**
- **d = 8 мм**

### Пример 2: Диаметр 10, 12, 15, 17 мм

**Подшипник: 201**
- Последние две цифры: **01**
- Код 01 → **d = 12 мм**

**Подшипник: 60102**
- Последние две цифры: **02**
- Код 02 → **d = 15 мм**

### Пример 3: Диаметр кратный 5 (от 20 мм)

**Подшипник: 205**
- Последние две цифры: **05**
- **d = 05 × 5 = 25 мм**

**Подшипник: 6310**
- Последние две цифры: **10**
- **d = 10 × 5 = 50 мм**

**Подшипник: 32220**
- Последние две цифры: **20**
- **d = 20 × 5 = 100 мм**

### Пример 4: Некратный диаметр с цифрой 9

**Подшипник: 60902**
- Цифра 9 на 3-м месте
- Код 02 → ближайший d = 15 мм
- Но цифра 9 → это d = **16 мм** (некратное значение)

### Пример 5: Специальные диаметры

**Подшипник: 602/32**
- Знак "/" → специальный диаметр
- **d = 32 мм**

**Подшипник: 10079/710**
- Знак "/" → крупногабаритный
- **d = 710 мм**

## Измерение внутреннего диаметра

### Инструменты

| Инструмент | Точность | Применение |
|------------|----------|------------|
| **Штангенциркуль** | ±0,05 мм | Подшипники класса 0, 6 |
| **Микрометр** | ±0,01 мм | Подшипники класса 5, 4 |
| **Нутромер** | ±0,005 мм | Прецизионные подшипники |
| **Координатно-измерительная машина** | ±0,001 мм | Класс 2, Т |

### Метод измерения

1. **Очистить подшипник** от смазки и загрязнений
2. **Установить внутреннее кольцо** на ровную поверхность
3. **Измерить диаметр** в двух взаимно перпендикулярных направлениях
4. **Усреднить результаты**

**Важно:** Измерять нужно именно **внутреннее кольцо**, а не наружное!

## Связь с обозначением вала

### Соответствие подшипника и вала

| Подшипник | Внутренний диаметр | Диаметр вала (номинал) |
|-----------|-------------------|------------------------|
| 201 | 12 мм | 12 мм |
| 202 | 15 мм | 15 мм |
| 6204 | 20 мм | 20 мм |
| 6205 | 25 мм | 25 мм |
| 6310 | 50 мм | 50 мм |

**Примечание:** При посадке с натягом реальный диаметр вала может быть на 0,005-0,030 мм больше (зависит от поля допуска).

## Примечания

**Важные особенности:**

1. **Коды 00, 01, 02, 03** — только для диаметров 10, 12, 15, 17 мм
2. **Цифра 9** на 3-м месте — признак некратного диаметра
3. **Знак "/"** — специальные диаметры (малые, средние особые, крупные)
4. **Диаметры кратные 5** — самая распространенная группа
5. Внутренний диаметр **нельзя путать** с наружным диаметром!

**Точность изготовления:**

- Класс 0: допуск ±10-15 мкм
- Класс 6: допуск ±5-7 мкм
- Класс 5: допуск ±3-5 мкм
- Класс 4: допуск ±2-3 мкм

**Практические советы:**

- При расшифровке всегда смотрите на **последние 2 цифры**
- Если есть знак **"/"** — диаметр указан явно
- Для диаметров **20-495 мм** умножайте на **5**
- При сомнении проверяйте по каталогу производителя
