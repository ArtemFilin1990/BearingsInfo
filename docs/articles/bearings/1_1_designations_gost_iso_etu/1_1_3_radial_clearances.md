# 1.1.3 Радиальные зазоры в подшипниках качения

## Цель

Описать систему обозначений радиальных зазоров в подшипниках качения по ГОСТ и ISO для правильного выбора подшипников в зависимости от условий эксплуатации (температура, посадка, скорость).

## Ключевые термины

- **Радиальный зазор** — расстояние, на которое можно переместить одно кольцо подшипника относительно другого в радиальном направлении
- **Группа зазора** — диапазон допустимых значений радиального зазора (C1, C2, C0, C3, C4, C5)
- **Рабочий зазор** — радиальный зазор в подшипнике после установки и в условиях эксплуатации
- **Посадка** — характер соединения подшипника с валом и корпусом

## Суть

## Группы радиальных зазоров

Радиальный зазор — расстояние, на которое можно переместить одно кольцо подшипника относительно другого в радиальном направлении.

### Обозначения по ГОСТ

| Группа | Величина зазора | Применение |
|--------|----------------|------------|
| **C1** | Меньше C2 | Специальные применения |
| **C2** | Меньше нормального | Прецизионные узлы, малые температуры |
| **C0** | Нормальный (без обозначения) | Стандартные применения |
| **C3** | Больше нормального | Высокие температуры, натяг на валу |
| **C4** | Больше C3 | Очень высокие температуры |
| **C5** | Больше C4 | Экстремальные температуры |

## Выбор зазора

### Влияние температуры
При нагреве внутреннее кольцо расширяется сильнее → зазор уменьшается.

**Рекомендации**:
- Δt < 10°C → C0
- Δt = 10-30°C → C3
- Δt > 30°C → C4

### Влияние посадки
Плотная посадка на вал уменьшает зазор.

### Типичные применения

- **C2**: Прецизионные станки, подшипники качения
- **C0**: Электродвигатели, насосы, редукторы
- **C3**: Электродвигатели с повышенной температурой, вентиляторы
- **C4**: Сушильные барабаны, печи

## Измерение зазора

Замеряется в свободном состоянии (до установки) индикатором часового типа.

## Примеры

**Пример 1: Электродвигатель общего назначения**
- Обозначение: `6205` (C0 не указывается)
- Зазор: C0 (нормальный)
- Температура: стандартная (Δt < 10°C)

**Пример 2: Электродвигатель с повышенной температурой**
- Обозначение: `6205-C3`
- Зазор: C3 (увеличенный)
- Температура: Δt = 10-30°C

**Пример 3: Сушильный барабан**
- Обозначение: `6308-C4`
- Зазор: C4 (большой)
- Температура: Δt > 30°C
