# 11.3 Технические жидкости

## Типы технических жидкостей

### 1. Гидравлические масла
- **Минеральные**: И-20А, И-40А, И-50А (ГОСТ 20799-88)
- **Синтетические**: высокая термостойкость
- Применение: гидросистемы

### 2. Моторные масла
- **Классификация**: SAE (вязкость), API (качество)
- **Пример**: 5W-40, 10W-40
- Применение: двигатели

### 3. Трансмиссионные масла
- **ГОСТ 23652-79**: ТАД-17, ТАП-15В
- **SAE**: 75W-90, 80W-90
- Применение: коробки передач, редукторы

### 4. Смазки пластичные
- **Литиевые**: Литол-24, ЦИАТИМ-221
- **NLGI**: классы 0-3 (консистенция)
- Применение: подшипники, шарниры

### 5. Охлаждающие жидкости
- **Антифризы**: на основе этиленгликоля
- **Тосол**: российская марка
- Температура: -40°C...-65°C

## Выбор масла
Учитывать: температуру, нагрузку, скорость, совместимость с материалами.

## Хранение
- Закрытая тара
- Защита от влаги и грязи
- Температура: +5...+25°C
