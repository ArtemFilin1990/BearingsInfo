## Определение

Предельная частота вращения — максимальная скорость, при которой подшипник может работать длительно без перегрева.

## Факторы влияния

1. **Тип подшипника**: шариковые > роликовые
2. **Размер**: меньше → выше скорость
3. **Смазка**: масло > пластичная смазка
4. **Нагрузка**: меньше → выше скорость
5. **Охлаждение**: лучше → выше скорость

## Типичные значения

| Тип | d=25мм | d=50мм |
|-----|--------|--------|
| Радиальный шариковый (6xxx) | 16000 об/мин | 10000 |
| Роликовый цилиндр. (Nxxx) | 12000 | 8000 |
| Конический (7xxx) | 8000 | 5000 |

## Параметр скорости (n×d)

```
n×d = частота × средний диаметр

n×d < 300000 → пластичная смазка
n×d > 300000 → жидкая смазка
n×d > 1000000 → масляный туман
```

## Превышение предельной скорости

Приводит к:
- Перегреву
- Разрушению смазки
- Быстрому износу
- Выходу из строя
