# 1.1.4 Обозначение момента трения подшипников

## Цель

Описать момент трения в подшипниках качения, его составляющие, факторы влияния и обозначения в маркировке для правильного выбора подшипников с требуемыми характеристиками трения.

## Ключевые термины

- **Момент трения** — момент сопротивления вращению подшипника
- **M₀** — момент трения от нагрузки (упругое гистерезисное трение)
- **M₁** — момент трения от вязкости смазки (гидродинамическое сопротивление)
- **Предварительный натяг** — усилие, создающее дополнительное трение в подшипнике
- **Вязкость смазки** — сопротивление жидкости течению, влияющее на M₁

## Суть

## Момент трения подшипника

Момент трения (M) — момент сопротивления вращению подшипника.

### Составляющие момента трения

1. **Упругое гистерезисное трение** в зонах контакта
2. **Трение качения** тел качения
3. **Трение скольжения** в сепараторе
4. **Гидродинамическое сопротивление** смазки

## Формула момента трения

```
M = M₀ + M₁

M₀ — момент от нагрузки
M₁ — момент от вязкости смазки
```

## Обозначение в маркировке

Специальные подшипники с пониженным моментом трения могут иметь суффиксы:
- **L** — низкий момент трения (Low friction)
- **LT** — низкий момент и температура

## Факторы, влияющие на момент

1. **Тип подшипника**: шариковые < роликовые
2. **Нагрузка**: больше нагрузка → больше момент
3. **Скорость**: при высоких скоростях растет M₁
4. **Вязкость смазки**: более вязкая → больше M₁
5. **Температура**: снижает вязкость → уменьшает M₁
6. **Предварительный натяг**: увеличивает M₀

## Типичные значения

| Тип подшипника | Момент трения (Н·мм) |
|----------------|---------------------|
| 6205 (шариковый) | 2-5 |
| NU 205 (роликовый) | 8-15 |
| 7205 (конический) | 10-20 |

## Измерение

Измеряется динамометрическим ключом или специальными стендами при:
- Нулевой нагрузке (M₁)
- Рабочей нагрузке (M = M₀ + M₁)

## Примеры

**Пример 1: Стандартный шариковый подшипник**
- Обозначение: `6205`
- Момент трения: 2-5 Н·мм
- Применение: Электродвигатель

**Пример 2: Подшипник с низким моментом трения**
- Обозначение: `6205-L`
- Момент трения: 1-3 Н·мм (снижен на 30-40%)
- Применение: Высокоскоростные шпиндели

**Пример 3: Роликовый подшипник**
- Обозначение: `NU 205`
- Момент трения: 8-15 Н·мм
- Применение: Редуктор
