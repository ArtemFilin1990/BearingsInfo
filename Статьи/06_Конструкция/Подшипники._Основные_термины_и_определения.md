**Подшипники. Основные термины и определения.** 


| базовый типоразмер подшипника | Типоразмер подшипника, имеющий наиболее широко применяемую внутреннюю конструкцию и основное условное обозначение в соответствии с ГОСТ |
| --- | --- |
| базовый торец | Торец подшипника качения, по отношению к которому задают допуск расположения или определяют отклонение расположения |
| базовый торец кольца | Торец, спроектированный изготовителем как базовый торец подшипника, который может служить базой для измерений. Базовым торцом для измерения кольца обычно является немаркированный торец. Для симметричных колец, когда невозможно определить базовый торец, считается, что допуски относятся к любому торцу... |
| двойной упорный шариковый (роликовый) подшипник качения | Упорный шариковый (роликовый) подшипник качения, предназначенный для восприятия осевой нагрузки в обоих направлениях |
| двухрядный подшипник качения | Двухрядный подшипник, Подшипник качения с двумя рядами тел качения |
| действительная ширина (монтажная высота) подшипника | (радиального и радиально-упорного подшипников, когда один торец внутреннего кольца и один торец наружного кольца ограничивают ширину подшипника) : Расстояние между точками пересечения оси подшипника с двумя плоскостями, касательными к действительным базовым торцам колец, предназначенным для ограниче... |
| действительный размер | Размер детали, полученный путем измерения (например, диаметр, ширина и т.д.). |
| диаметр конического отверстия | Диаметр окружности сечения конуса радиальной плоскостью, касательной к широкому торцу внутреннего кольца подшипника |
| диаметр контакта дорожки качения | Диаметр окружности, проходящей через номинальные точки контакта на дорожке качения. В роликовых подшипниках номинальными точками контакта считаются, главным образом, точки в середине длины ролика. |
| допуск | Разность между наибольшим и наименьшим предельными размерами или абсолютное значение алгебраической разности между верхним и нижним отклонениями |
| единичная длина ролика | Расстояние между двумя точками торцев в пределах нормируемого участка, лежащими на одной прямой, параллельной оси ролика |
| единичная плоскость | Любая радиальная или осевая плоскость, в которой могут проводиться измерения. |
| единичная ширина кольца | Расстояние между двумя точками торцев кольца, лежащими на одной прямой, параллельной оси кольца |
| единичная ширина кольца | Расстояние между точками пересечения двух действительных торцов кольца с прямой, перпендикулярной к плоскости, касательной к базовому торцу кольца. |
| единичная ширина упорного борта наружного кольца | Расстояние между точками пересечения двух действительных торцов упорного борта наружного кольца с прямой, перпендикулярной к плоскости, касательной к опорному торцу упорного борта. |
| единичное сечение | Сечение радиальной плоскостью, расположенное в любом месте нормируемого участка |
| единичный диаметр отверстия | Расстояние между двумя параллельными линиями, касательными к линии пересечения действительной поверхности отверстия любой радиальной плоскостью. |
| единичный диаметр отверстия (наружной цилиндрической поверхности) | Расстояние между двумя произвольно расположенными в радиальной плоскости параллельными линиями, касательными к контуру единичного сечения отверстия (наружной цилиндрической поверхности) |
| единичный диаметр отверстия в единичной плоскости | Единичный диаметр отверстия, относящийся к конкретной радиальной плоскости. |
| единичный диаметр отверстия комплекта тел качения | радиальный подшипник без внутреннего кольца, Расстояние между двумя параллельными линиями, касательными к линии пересечения огибающего профиля, вписанного в комплект тел качения, и радиальной плоскости. |
| единичный диаметр тела качения | Расстояние между двумя произвольно расположенными в радиальной плоскости параллельными линиями, касательными к контуру единичного сечения тела качения. У шарика единичный диаметр измеряют в сечении центральной плоскостью. |
| единичный наружный диаметр | Расстояние между двумя параллельными линиями, касательными к линии пересечения действительной наружной поверхности радиальной плоскостью. |
| единичный наружный диаметр в единичной плоскости | Единичный наружный диаметр, относящийся к конкретной радиальной плоскости. |
| единичный наружный диаметр комплекта тел качения | радиальный подшипник без наружного кольца, Расстояние между двумя параллельными линиями, касательными к линии пересечения огибающего профиля, описанного вокруг комплекта тел качения, и радиальной плоскостью. |
| единичный размер | Любое отдельное расстояние, измеренное между любыми двумя противоположными точками (например, диаметр, ширина и т.д.). Этот параметр рассматривается также как «локальный действительный размер» |
| закрытый подшипник качения | Подшипник качения с одним или двумя уплотнениями, с одной или двумя защитными шайбами или одним уплотнением и одной защитной шайбой. |
| игольчатый роликовый подшипник качения | Роликовый подшипник качения с игольчатыми роликами в качестве тел качения |

| измерительное усилие | Усилие, прилагаемое наконечником индикатора или самописца к измеряемой поверхности. |
| комбинированный подшипник качения | Комбинированный подшипник, Подшипник качения, представляющий собой сочетание радиального и упорного подшипников и предназначенный для восприятия радиальной и осевой нагрузок |
| комплектный подшипник качения | Один из подшипников качения, входящих в состав сдвоенного подшипника или в комплект подшипников. |
| конический роликовый подшипник качения | Конический роликоподшипник, Роликовый подшипник качения с коническими роликами в качестве тел качения |
| конус | Поверхность вращения, образованная прямой линией, пересекающей ось. |
| крайнее сечение кольца или ролика | Сечение радиальной плоскостью, ограничивающее нормируемый участок и расположенное от торца кольца или ролика на расстоянии, равном удвоенной координате фаски, а у длинных роликов - одной десятой длины |
