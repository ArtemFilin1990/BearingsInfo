# podshipnikon.ru reference (offline snapshot)

## Purpose
Хранить офлайн-выгрузки таблиц размеров podshipnikon.ru без внешних ссылок.

## Classification
- Независимый справочник по размерам и диапазонам.
- Используется для проверки d/D/B и сопоставлений.

## Designation
- `SNAP_SRC_PODSHIPNIKON_REF` — выгрузка таблиц/поиска по диапазонам.

## Prefixes/Suffixes
- Префикс `SNAP_SRC_` — офлайн-слепок.
- Суффикс `_REF` — справочник.

## Examples
- Сохранить таблицу с диапазоном d 20–30 мм и сопутствующие D/B, указать дату выгрузки.

## Notes & Limitations
- Онлайн-доступ блокирован (403), загрузка требуется офлайн.
- Указывать источник и контрольные суммы при обновлениях.

## Standard References
- Домен источника: podshipnikon.ru (раздел размеров подшипников).
- Локальные файлы: этот `.md` + выгрузки/CSV в той же директории.
