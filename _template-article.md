## 4. Таблица аналогов ГОСТ ⇄ ISO **(ОБЯЗАТЕЛЬНО)**

*Этот раздел присутствует ВСЕГДА.*

| ГОСТ | ISO | Степень соответствия | Тип соответствия | Комментарий |
|------|-----|----------------------|------------------|-------------|
| | | полное/частичное/отсутствует/спорное | прямой/функциональный/параметрический/— | |

**Степень соответствия:**
- `полное` — стандарты идентичны
- `частичное` — есть различия
- `отсутствует` — аналога нет
- `спорное` — есть противоречия

**Тип соответствия:**
- `прямой` — перевод/адаптация
- `функциональный` — одна задача, разные методы
- `параметрический` — совпадают параметры, различаются требования

*Если тема не связана со стандартами:*
```
Не применимо — тема не регламентируется отдельными стандартами.
```
