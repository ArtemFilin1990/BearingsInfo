# SKF Needle Bearings Catalog (offline snapshot)

## Purpose
Специализированный каталог игольчатых подшипников SKF с детальными техническими характеристиками и областями применения.

## Classification
- Официальный PDF каталог производителя SKF.
- Специализированный источник по игольчатым подшипникам.

## Designation
- `SNAP_SRC_SKF_NEEDLE_PDF` — файл `Igolchatye-podshipniki-SKF.pdf` + данный `.md` с метаданными.

## Content Overview
Каталог игольчатых подшипников SKF содержит:
- Игольчатые радиальные подшипники с сепаратором
- Игольчатые подшипники без внутреннего кольца
- Игольчатые радиально-упорные подшипники
- Комбинированные игольчатые подшипники
- Технические характеристики (размеры, грузоподъемность)
- Области применения и примеры использования
- Рекомендации по монтажу

## Prefixes/Suffixes
- Префикс `SNAP_SRC_` — идентификатор офлайн-слепка.
- Суффикс `_NEEDLE_PDF` — указывает на каталог игольчатых подшипников.

## Examples
Типовые обозначения игольчатых подшипников SKF:
- **RNA** — игольчатые подшипники с полными комплектами игл
- **NK** — игольчатые подшипники с сепаратором
- **NKI** — игольчатые подшипники с внутренним кольцом
- **NKIS** — игольчатые подшипники с уплотнениями
- **HK** — вытяжные игольчатые подшипники
