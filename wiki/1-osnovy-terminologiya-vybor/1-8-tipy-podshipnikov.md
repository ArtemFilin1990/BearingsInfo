# Типы подшипников

## Определение

Тип подшипника — конструктивная разновидность подшипника качения, определяемая формой тел качения, направлением воспринимаемой нагрузки и особенностями конструкции. Обозначается цифрой **4** в основном условном обозначении (четвертая позиция справа).

## Таблица типов подшипников (0-9)

| Цифра | Тип подшипника | Тела качения | Направление нагрузки | Примеры обозначений |
|-------|----------------|--------------|----------------------|-------------------|
| **0** | Шариковый радиальный | Шарики | Радиальная (основная) + малая осевая | 205, 6305, 180108 |
| **1** | Шариковый радиальный сферический (двухрядный) | Шарики | Радиальная + компенсация перекосов | 1205, 1309 |
| **2** | Роликовый радиальный с короткими цилиндрическими роликами | Ролики цилиндр. | Радиальная (только!) | 2205, 32205, NJ205 |
| **3** | Роликовый радиальный со сферическими роликами (двухрядный) | Ролики сферич. | Радиальная + высокие перекосы | 3205, 23205 |
| **4** | Роликовый радиальный с длинными цилиндрическими или игольчатыми роликами | Иголки | Радиальная, компактный | 4074103, ИК25×30×10 |
| **5** | Роликовый радиальный с витыми роликами | Ролики витые | Радиальная (редкий тип) | 5205 (редко) |

| **8** | Шариковый упорный или упорно-радиальный | Шарики | Осевая (основная) | 8205, 688 |
| **9** | Роликовый упорный или упорно-радиальный | Ролики | Осевая (основная), высокая C₀ | 9205, 81205 |

## Подробная характеристика типов

### Тип 0: Шариковый радиальный

**Конструкция:**
- Однорядный, два кольца, шарики
- Самый распространенный тип (>60% всех подшипников)

**Преимущества:**
- Универсальность
- Низкая цена
- Низкое трение
- Высокая скорость вращения

**Недостатки:**
- Средняя грузоподъемность
- Ограниченная осевая нагрузка

**Применение:** Электродвигатели, насосы, вентиляторы, автомобили

### Тип 1: Шариковый сферический

**Конструкция:**
- Двухрядный, шарики на общей сферической дорожке
- Внутреннее кольцо самоустанавливается

**Преимущества:**
- Компенсация перекосов до 2-3°
- Радиальная нагрузка в двух направлениях

**Недостатки:**
- Не воспринимает осевую нагрузку
- Ниже скорость, чем тип 0

**Применение:** Сельхозтехника, конвейеры, дробилки

### Тип 2: Роликовый цилиндрический

**Конструкция:**
- Ролики короткие цилиндрические
- Модификации: N, NU, NJ, NUP (различия в бортах)

**Преимущества:**
- Высокая радиальная грузоподъемность
- Высокая жесткость
- Точность вращения

**Недостатки:**
- **НЕ воспринимает осевую нагрузку!**
- Чувствителен к перекосам

**Применение:** Станки, редукторы, железнодорожный транспорт

### Тип 3: Роликовый сферический

**Конструкция:**
- Двухрядный, ролики бочкообразные (сферические)
- Самоустанавливающийся

**Преимущества:**
- Очень высокая грузоподъемность
- Компенсация перекосов до 2-4°
- Ударные нагрузки

**Недостатки:**
- Крупные габариты
- Высокая стоимость
- Средняя скорость

**Применение:** Металлургия, горнодобыча, тяжелое машиностроение

### Тип 4: Игольчатый

**Конструкция:**
- Ролики длинные тонкие (иголки)
- Часто без колец

**Преимущества:**
- Компактность (малый радиальный размер)
- Высокая C при малых габаритах

**Недостатки:**
- Низкая скорость
- Чувствителен к перекосам и загрязнению

**Применение:** Карданы, коробки передач, шатуны

### Тип 6: Радиально-упорный шариковый

**Конструкция:**
- Однорядный, угол контакта α (15°, 25°, 40°)
- Воспринимает Fr + Fa

**Преимущества:**
- Комбинированная нагрузка
- Высокая точность
- Высокая скорость

**Недостатки:**
- Только одностороннее действие Fa (нужна пара)
- Дороже типа 0

**Применение:** Шпиндели станков, насосы, компрессоры

### Тип 7: Роликовый конический

**Конструкция:**
- Ролики конические, угол контакта 10-30°
- Разборный (кольца раздельные)

**Преимущества:**
- Высокая Fr + высокая Fa
- Регулируемый зазор
- Высокая жесткость

**Недостатки:**
- Требует парной установки
- Сложный монтаж
- Средняя скорость

**Применение:** Автомобильные ступицы, редукторы, железные дороги

### Тип 8: Упорный шариковый

**Конструкция:**
- Шарики между двумя кольцами
- Только осевая нагрузка!

**Преимущества:**
- Компактная высота
- Малое трение

**Недостатки:**
- **Не воспринимает радиальную нагрузку!**
- Очень низкая скорость

**Применение:** Поворотные столы, домкраты, винтовые передачи

### Тип 9: Упорный роликовый

**Конструкция:**
- Ролики цилиндрические или конические
- Только осевая нагрузка!

**Преимущества:**
- Очень высокая осевая C₀
- Жесткость

**Недостатки:**
- **Не воспринимает радиальную нагрузку!**
- Очень низкая скорость (n < 1000)

**Применение:** Прессы, краны, буровые установки

## Таблица выбора типа по нагрузке

| Тип нагрузки | Тип подшипника | Когда использовать |
|--------------|----------------|-------------------|
| **Fr (только радиальная)** | 0 — Шариковый радиальный | Стандартные условия, универсальный |
| | 2 — Роликовый цилиндр. | Высокие радиальные нагрузки |
| | 4 — Игольчатый | Ограниченное радиальное пространство |
| | 3 — Роликовый сферический | Очень высокие нагрузки + перекосы |
| **Fa (только осевая)** | 8 — Упорный шариковый | Малые/средние Fa |
| | 9 — Упорный роликовый | Высокие Fa, малая скорость |
| **Fr + Fa (комбинир.)** | 6 — Радиально-упорный | Fa/Fr < 0.5, высокая скорость |
| | 7 — Конический | Fa/Fr < 1.5, высокие нагрузки |

## Примечания

**Важные особенности:**

1. **Тип 0** — самый универсальный и распространенный
2. **Типы 2, 4** — НЕ воспринимают осевую нагрузку
3. **Типы 8, 9** — НЕ воспринимают радиальную нагрузку
4. **Типы 1, 3** — самоустанавливающиеся (компенсируют перекосы)
5. **Типы 6, 7** — требуют парной установки для двусторонней Fa

**Выбор по скорости (от высокой к низкой):**
```
0 > 6 > 2 > 4 > 7 > 3 > 1 > 8 > 9
```

**Выбор по грузоподъемности (C):**
```
3 > 9 > 7 > 2 > 6 > 4 > 8 > 1 > 0
```
