## Как пользоваться каталогом

- **d** - внутренний диаметр (диаметр отверстия), мм
- **D** - наружный диаметр, мм
- **B** - ширина подшипника, мм (для радиальных)
- **H** - высота подшипника, мм (для конических и упорных)
- **m** - масса, кг (приблизительная)
- **C** - динамическая грузоподъемность, кН
- **C₀** - статическая грузоподъемность, кН
- **n (об/мин)** - предельная частота вращения

## Примечания

1. **Масса (m)** указана приблизительная для открытого подшипника без уплотнений
2. **Динамическая грузоподъемность (C)** - для расчета ресурса L₁₀
3. **Статическая грузоподъемность (C₀)** - максимальная статическая нагрузка без остаточной деформации
4. **Предельная частота вращения (n)** - для смазки консистентной смазкой, для масляной выше на 20-30%
5. Данные приведены на основе каталогов SKF, FAG, NSK, российских производителей

## Источники данных

- Каталоги SKF (Швеция)
- Каталоги FAG/INA Schaeffler Group (Германия)
- Каталоги NSK (Япония)

- ГОСТ 520-2011 "Подшипники качения. Общие технические условия"
