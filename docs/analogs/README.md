# Аналоги ГОСТ ↔ ISO и корпусные узлы

## Куда смотреть
- `data/analogs/gost_iso.csv` — соответствия ГОСТ ↔ ISO с комментариями по применимости.
- `data/analogs/units.csv` — кросс-ссылки корпусных узлов NTN ↔ SKF/FAFNIR/INA/FYH/NSK/SEAL/FAG/RHP/ASAHI.
- `data/analogs/housings.csv` — корпуса UCFC/UELF/AELP и их аналоги по брендам.
- Источники: `sources/analogs/Таблица аналагов.docx`, `docs/bearings/analogues/bearing_units.md`, `Аналоги/ГОСТ_ISO.md`.

## Правила замены
1. Проверяйте размеры d/D/B/H перед заменой (см. `data/gost/dimensions.csv` и `data/iso/dimensions.csv`).
2. Учитывайте конструктивные признаки: смазочные канавки W33, усиление EA/CA/CC, коническое отверстие K/K30.
3. Для корпусных узлов сверяйте шаг и диаметр крепежных отверстий, материал корпуса и наличие эксцентриковых втулок.
4. Запрещены дубликаты по ключам `(gost, iso)`; при наличии нескольких брендов указывается список в поле `brand`.

## Примеры
- **7205 → 30205** — конический роликовый аналог; после замены требуется настройка преднатяга.
- **2205 → 22205** — сферический роликовый шире цилиндрического: нужна проверка посадок (строка помечена в `notes`).
- **UCFL → FYTB-TF / FYTJ-TF** — фланцевые корпуса SKF, шаг отверстий должен совпадать с узлом NTN.
- **AELF → ENF (NSK)** — самоцентрирующийся корпусный узел; важно проверить материал корпуса и уплотнения.

## Контроль качества
- Стабильная сортировка по `(gost, iso, brand)` проверяется тестами `tests/test_dimensions.py` и `tests/test_dedup.py`.
- Все примечания хранятся в поле `notes`; пустые значения оставляются пустыми строками.
- Обновление выполняется через `python scripts/update_repo.py`, что гарантирует одинаковый порядок строк.
