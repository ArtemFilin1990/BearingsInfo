# ISO rolling bearings: structure and references

## Purpose
- Зафиксировать выделенный раздел для материалов по стандартам ISO без смешения с ГОСТ и таблицами аналогов.
- Указать опорные источники и связь с существующей документацией `docs/bearings/designations/iso*.md`.

## Classification
- Диапазон ISO охватывает радиальные, радиально-упорные и упорные подшипники шариковые и роликовые; конические роликовые регламентированы отдельной серией.
- Базовые серии типоразмеров и граничных размеров описаны в ISO 15 (радиальные) и ISO 355 (конические роликовые) и используются как основа для каталогов производителей.

## Designation
- Номера строятся по ISO 15: тип/серия ширины/серия диаметров/код отверстия; код 00=10 мм, 01=12 мм, 02=15 мм, 03=17 мм, далее код × 5.
- Для конических роликовых подшипников применяется серия 3XX или 302XX/303XX по ISO 355 с теми же правилами кода отверстия.
- Детализированные таблицы и суффиксы отражены в `docs/bearings/designations/iso.md` и `docs/bearings/designations/iso_suffixes.md`.

## Prefixes/Suffixes
- ISO использует суффиксы для зазоров (C2/C3/C4/C5), классов точности (P0/P6/P5/P4/P2), материалов сепараторов и защит (Z, 2Z, RS, 2RS и др.).
- При миграции данных сюда следует сохранять оригинальные обозначения производителей и проверять совместимость суффиксов с ISO 492/ISO 199/ISO 76.

## Examples
- **6205** (ISO 15): глубокий радиальный шариковый подшипник, серия диаметров 2 (легкая), серия ширины 0 (нормальная), код отверстия 05 → 25 мм.
- **30206** (ISO 355): конический роликовый подшипник, серия 02, код отверстия 06 → 30 мм; применяется в ступицах и редукторах.

## Notes & Limitations
- Суффиксы ISO не тождественны ГОСТ: перед переносом данных нужно сверять значения зазора, класса точности и материала с документацией производителя.
- Каталожные коды могут включать фирменные индексы (например, «J», «K», «TN9»); они должны сопровождаться ссылкой на каталог.
