# Справочник по подбору подшипников

## Цель
Пошаговый алгоритм выбора подшипника: от требований и стандартов ГОСТ/ISO до выбора аналога и производителя.

## 1. Определение типа и размера по ГОСТ
1. Определите направление нагрузки и тип подшипника (радиальный, упорный, радиально‑упорный).
2. Выберите тип тел качения (шариковый, роликовый, игольчатый).
3. По диаметрам вала и посадочного места определите размерный ряд и типоразмер по ГОСТ. Для многих подшипников последние две цифры основного обозначения, умноженные на пять, дают внутренний диаметр【371647085035825†L67-L148】.

## 2. Нахождение аналога по ISO
- Используйте таблицы «ГОСТ ↔ ISO» и «Аналоги» (каталог `docs/bearings/analogues`) для поиска ближайшего номера ISO.
- Например, подшипник 3632 соответствует ISO 22332EAW33【59918111166298†L3-L17】.
- При подборе проверьте:
  - размеры (внутренний диаметр, наружный диаметр, ширина);
  - серию (диаметрический и ширинный ряд);
  - радиальный зазор (группы Н, 6, 7, 8, 9 по ГОСТ или C0‑C5 по ISO)【371647085035825†L67-L148】;
  - класс точности (0, 6, 5, 4 по ГОСТ или P0, P6, P5, P4 по ISO)【371647085035825†L19-L22】.

## 3. Выбор бренда
- Сопоставьте найденный ISO‑тип с каталогами производителей:
  - **Премиум**: SKF, FAG, NTN‑SNR, INA, Koyo, IKO, Timken, THK — высокое качество и надёжность【271470310052723†L0-L18】.
  - **Средний сегмент**: NSK, NACHI/Nippon Bearing, FYH, RHP — баланс цены и ресурса.
  - **Бюджет**: TSC, ISB, KDF, NBS, LSK, FBJ, FLT, DKF DDR — применяются в менее критичных узлах【271470310052723†L112-L132】.

## 4. Проверка дополнительных параметров
- Уплотнения и защитные щитки: суффиксы 2RS, 2Z, 2RSR, 2LS обозначают наличие резиновых уплотнений или металлических щитков【874842551192622†L7-L23】.
- Радиальный зазор: суффиксы C3‑C5 по ISO означают увеличенный зазор; аналогичные группы в ГОСТ отмечаются цифрами 7, 8, 9.
- Класс точности: P6/P5/P4 по ISO соответствуют 6/5/4 по ГОСТ【371647085035825†L19-L22】.
- Материал сепаратора и исполнение: литеры J, E, TVP, M и др. (см. `iso/suffixes.md`).

## 5. Итоговый выбор
1. Сформируйте список возможных вариантов (типоразмер + бренд).
2. Сравните цены, сроки поставки и наличие на складе.
3. При необходимости согласуйте использование аналога с инженером (допуски, зазоры, исполнение).

## Примечание
В сложных условиях (высокие скорости, температура, вибрации) обращайтесь к специалистам или документации производителей.
