# Таблицы аналогов ГОСТ ↔ ISO

На этой странице представлены таблицы соответствия подшипников по стандартам ГОСТ и ISO. Аналоги полностью взаимозаменяемы по размерам (d, D, B/H) и основным характеристикам.

## Важные замечания

⚠️ **Перед заменой проверьте**:
1. Точное совпадение размеров (d, D, B или H)
2. Класс точности (может отличаться)
3. Динамическую и статическую грузоподъемность
4. Радиальный зазор
5. Тип уплотнений и смазки

✅ **Полностью взаимозаменяемые параметры**:
- Внутренний диаметр (d)
- Наружный диаметр (D)
- Ширина/высота (B/H)

## Примеры полной замены

### Пример 1: Электродвигатель
- **ГОСТ**: 6205-2RS C3
- **ISO аналог**: 6205-2RS C3 (SKF, FAG, NSK)
- **Полностью взаимозаменяемы**

### Пример 2: Редуктор
- **ГОСТ**: 2305 Е
- **ISO аналог**: NU 305 E (или NU 305 P)
- **Примечание**: Е (ГОСТ) = E/P (ISO) - пластиковый сепаратор

### Пример 3: Ступица автомобиля
- **ГОСТ**: 7205
- **ISO аналог**: 30205
- **Полностью взаимозаменяемы**

### Пример 4: Высокоточное оборудование
- **ГОСТ**: 6205-6 C3
- **ISO аналог**: 6205 P6 C3
- **Примечание**: Класс точности 6 (ГОСТ) ≈ P6 (ISO)
