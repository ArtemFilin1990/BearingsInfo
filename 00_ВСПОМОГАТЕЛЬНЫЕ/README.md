# Вспомогательные файлы

Эта директория содержит вспомогательные файлы и утилиты, которые не являются частью основного содержания книги о подшипниках.

## Содержимое

### Утилиты обработки архивов
- **AUTO_EXTRACT_GUIDE.md** - руководство по автоматической распаковке архивов
- **extract_archives.sh** - скрипт для распаковки архивных файлов

### Служебные файлы
- **MANIFEST.csv** - манифест файлов репозитория
- **instructions.md** - общие инструкции по работе с проектом
- **00_DATA/** - директория с промежуточными данными

## Назначение

Эти файлы используются для:
- Автоматизации процессов обработки данных
- Документирования структуры проекта
- Хранения метаданных и вспомогательной информации

---

*Основное содержание книги находится в директории `Подшипники/` и исходных директориях с префиксами 02-12, 90, 99.*
