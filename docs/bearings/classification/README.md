# Классификация подшипников качения

## Основные типы подшипников

Подшипники качения классифицируются по нескольким признакам: конструкции, направлению воспринимаемой нагрузки, форме тел качения и другим параметрам.

## По направлению воспринимаемой нагрузки

### Радиальные подшипники
- **Назначение**: Воспринимают радиальные нагрузки (перпендикулярные к оси вращения)
- **Типы**: Шариковые однорядные, роликовые с короткими цилиндрическими роликами, игольчатые
- **Применение**: Электродвигатели, редукторы, насосы, вентиляторы

### Радиально-упорные подшипники
- **Назначение**: Воспринимают одновременно радиальные и осевые нагрузки
- **Типы**: Шариковые радиально-упорные, роликовые конические
- **Применение**: Коробки передач автомобилей, станки, сельхозтехника

### Упорные подшипники
- **Назначение**: Воспринимают осевые нагрузки (параллельные оси вращения)
- **Типы**: Упорные шариковые, упорно-радиальные шариковые
- **Применение**: Вертикальные валы, подъемники, краны

## По форме тел качения

### Шариковые подшипники
- **Тела качения**: Шарики
- **Преимущества**: 
  - Низкое трение
  - Высокие скорости вращения
  - Низкий уровень шума
  - Меньшая стоимость
- **Недостатки**: Меньшая грузоподъемность по сравнению с роликовыми
- **Типы**:
  - Радиальные однорядные (тип 0, ГОСТ)
  - Радиальные двухрядные (тип 1)
  - Радиально-упорные (тип 6, 7)
  - Упорные (тип 5)
  - Самоустанавливающиеся (тип 2)

### Роликовые подшипники
- **Тела качения**: Ролики (цилиндрические, конические, бочкообразные, игольчатые)
- **Преимущества**:
  - Высокая грузоподъемность
  - Большая жесткость
  - Увеличенный ресурс при высоких нагрузках
- **Недостатки**: Выше трение и шум, ниже предельные скорости
- **Типы**:
  - С короткими цилиндрическими роликами (тип 2, 3)
  - Конические (тип 7)
  - Сферические (самоустанавливающиеся, тип 3)
  - Игольчатые (тип 4)

## По конструктивным особенностям

### Однорядные подшипники
- Одно кольцо тел качения
- Наиболее распространены
- Примеры: 6205, 205, N205

### Двухрядные подшипники
- Два кольца тел качения
- Повышенная грузоподъемность
- Примеры: 1204, 3204

### Многорядные подшипники
- Три и более рядов тел качения
- Применяются в специальных условиях
- Примеры: прокатные станы, тяжелое машиностроение

### Сферические (самоустанавливающиеся) подшипники
- Компенсируют перекос валов
- Сферическая дорожка качения на наружном кольце
- Примеры: 1200, 3500 серии

## По наличию защиты и смазки

### Открытые подшипники
- **Обозначение**: Без дополнительных индексов
- **Применение**: Системы с внешним смазыванием
- **Преимущества**: Легко очищаются и обслуживаются

### С защитными шайбами
- **Обозначение**: Z (одна шайба), ZZ (две шайбы)
- **Применение**: Защита от крупных частиц при умеренных скоростях
- **Особенность**: Неконтактная защита

### С уплотнениями
- **Обозначение**: RS (одно уплотнение), 2RS (два уплотнения)
- **Применение**: Пыльная среда, влажность
- **Особенность**: Контактная защита, препятствует вытеканию смазки

### С встроенной смазкой
- **Обозначение**: Часто с индексами типа 2RS/C3
- **Применение**: Необслуживаемые узлы
- **Ресурс**: Закладывается смазка на весь срок службы

## Серии подшипников

### По ширине (серия диаметров)
- **Сверхлегкая серия**: 8, 9 (ISO: 18, 19)
- **Особо легкая серия**: 1 (ISO: 10)
- **Легкая серия**: 2 (ISO: 02)
- **Средняя серия**: 3 (ISO: 03)
- **Тяжелая серия**: 4 (ISO: 04)

### По высоте (серия ширин)
- **Узкая серия**: 7, 8, 9
- **Нормальная серия**: 0, 1, 2, 3
- **Широкая серия**: 4, 5, 6

## Области применения

### Промышленное оборудование
- Электродвигатели: 6201-6210 (легкая серия)
- Редукторы: 6305-6320 (средняя серия)
- Насосы: 6203, 6204 (с уплотнениями)
- Вентиляторы: 6308-6312

### Автомобильная техника
- Ступицы колес: конические 7805, 7905
- Коробки передач: 6205-6210
- Генераторы: 6203, 6303
- Кондиционеры: 6201, 6202

### Сельхозтехника
- Почвообрабатывающие машины: 6305-6312
- Комбайны: сферические 3500 серии
- Косилки: 205-210

### Бытовая техника
- Стиральные машины: 6204-6206 (с уплотнениями)
- Пылесосы: 6201-6203
- Электроинструмент: 6200-6204

### Железнодорожный транспорт
- Буксы вагонов: роликовые конические 42726
- Тяговые двигатели: 6320-6330

## Дополнительные классификации

### По материалу
- **Стальные**: Стандартные, из стали ШХ15
- **Нержавеющие**: Коррозионностойкие среды
- **Керамические**: Высокие температуры, агрессивные среды
- **Гибридные**: Керамические тела качения + стальные кольца

### По точности изготовления (ГОСТ 520-2011)
- **Класс 0**: Нормальная точность (без обозначения)
- **Класс 6**: Повышенная точность
- **Класс 5**: Высокая точность
- **Класс 4, 2**: Прецизионные подшипники

### По радиальному зазору (ГОСТ 24810-2013)
- **C2**: Уменьшенный зазор
- **Нормальный**: Без обозначения
- **C3**: Увеличенный зазор
- **C4, C5**: Большой и очень большой зазор
