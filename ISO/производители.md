# Производители ISO

## Определение
Производители подшипников, работающие по системе ISO, используют единые базовые обозначения и добавляют собственные суффиксы и префиксы для уточнения конструкции, материалов и комплектности.

## Назначение
- Быстрый выбор бренда с учётом страны и специализации.
- Понимание фирменных суффиксов при расшифровке маркировки.
- Сопоставление обозначений при подборе аналогов между каталогами.

## Конструкция / особенности
- Базовая структура обозначения сохраняется у всех производителей (префикс – основное число – суффиксы).
- Фирменные суффиксы добавляют данные о смазке, специальной термостабилизации, комплектности и точности.
- При сравнении необходимо учитывать, что одинаковая буква может означать разные свойства у разных брендов.

## Обозначения и расшифровка
| Производитель | Страна/группа | Специализация и особенности обозначений |
|---------------|---------------|-----------------------------------------|
| SKF | Швеция | Серия Explorer (суффикс E), высокотемпературные суффиксы HT/S1–S4, коды смазок MT/VT, обозначения комплектов DB/DF/DT, специальные исполнения для высоких вибраций VA201/VA208/VA405/VA407. |
| FAG/INA (Schaeffler) | Германия | Серии B/C/E, полиамидные сепараторы (P), высокотемпературные версии W64, усиленные конструкции CA/CC, использование суффиксов P5/P4 и ABEC-эквивалентов. |
| NSK | Япония | Уплотнения DDU/DU, высокоточные обозначения HG, комплектность DB/DF/DT, использование серии HR для повышенной жёсткости. |
| NTN | Япония | Уплотнения VV, неконтактные LLB, высокотемпературные серии TS1–TS4, специальные смазки VT-кодов, применение стандартных C-групп зазоров. |
| KOYO (JTEKT) | Япония | Уплотнения SR/SRL, специальные суффиксы ROVS для виброоборудования, обозначения комплектов QB/QF/QU. |
| NKE | Австрия | Стандартные ISO-обозначения, серии для железнодорожного транспорта, применение стандартных классов точности P6/P5/P4. |
| ISB | Италия | Стандартные ISO-обозначения, нержавеющие серии (SS) для пищевой промышленности, упор на базовую структуру без сложных фирменных суффиксов. |

## Примеры
- **6308-2RS1 C3 (SKF)** — двойные уплотнения RS1, увеличенный зазор C3, стандартное исполнение SKF.
- **NU205/P5 (FAG)** — цилиндрический роликовый подшипник с точностью P5 и латунной клеткой по умолчанию.
- **6205-DDU (NSK)** — двухсторонние контактные уплотнения NSK, базовый типоразмер 25×52×15.
- **6205-LLB (NTN)** — две неконтактные манжеты LLB с нормальным зазором для высоких скоростей.
- **22336EAW33 (KOYO/SNR)** — усиленная конструкция EA со смазочной канавкой и отверстиями W33, используется для сферических роликовых подшипников.

## Примечания и ограничения
- При подборе межбрендовых аналогов сверяйте фирменные суффиксы: одинаковое обозначение (например, E или P) может различаться по материалу или точности.
- Для специализированных отраслей (железная дорога, пищевое производство, высокие температуры) проверяйте наличие отраслевых серий у производителя.
- На упаковке может указываться как полное ISO-обозначение, так и фирменное с расширенным набором суффиксов; использовать необходимо полную форму.
