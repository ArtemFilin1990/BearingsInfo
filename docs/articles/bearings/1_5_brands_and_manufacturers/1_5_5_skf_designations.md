# 1.5.5 Обозначения подшипников SKF

## Цель

Объяснить систему обозначений подшипников компании SKF (Svenska Kullagerfabriken) — одного из крупнейших мировых производителей подшипников качения, чтобы специалисты могли правильно расшифровывать маркировку и подбирать аналоги.

## Ключевые термины

- **SKF** — шведская компания, основанная в 1907 году, мировой лидер в производстве подшипников
- **Explorer** — высокопроизводительная серия подшипников SKF с улучшенными характеристиками
- **Базовое обозначение** — основной код подшипника по системе ISO
- **Префикс** — буквы перед основным обозначением (E2, GE, QJ и др.)
- **Суффикс** — буквенно-цифровой код после основного обозначения
- **VA-подшипники** — специальные модификации для виброустойчивых применений

## Суть

### Структура обозначения SKF

```
[Префикс] [Базовое обозначение] [Суффикс]
   │              │                  │
   │              │                  └─ Дополнительные характеристики
   │              └──────────────────── Основное обозначение по ISO
   └─────────────────────────────────── Конструктивные особенности
```

### Базовое обозначение

SKF в основном использует систему обозначений ISO, схожую с ГОСТ:

| Пример | Расшифровка |
|--------|-------------|
| **6205** | Шариковый радиальный, легкая серия, d=25 мм |
| **NU 308** | Роликовый цилиндрический, средняя серия, d=40 мм |
| **22205** | Сферический роликовый, легкая серия, d=25 мм |
| **7205** | Радиально-упорный шариковый, d=25 мм |
| **30205** | Конический роликовый, d=25 мм |

### Основные префиксы SKF

| Префикс | Значение | Пример |
|---------|----------|--------|
| **E2** | Энергоэффективная серия Explorer (с 2010-х гг.) | E2.6205-2Z |
| **GE** | Шарнирный подшипник (сферический) | GE 20 ES |
| **QJ** | Четырёхточечный радиально-упорный | QJ 205 |
| **BS** | Упорный шариковый с плоскими шайбами | BS 2562 |
| **K** | Подшипник с коническим отверстием | K 6205 |
| **NJ** | Роликовый цилиндрический с фланцем | NJ 308 |
| **NUP** | Роликовый с разъёмным фланцем | NUP 308 |
| **RNA** | Игольчатый без колец | RNA 4905 |
| **NKIS** | Игольчатый с одним кольцом | NKIS 40 |
| **YAR**, **YET** | Y-подшипники (корпусные) | YAR 205-2F |

### Основные суффиксы SKF

#### Уплотнения и защитные шайбы

| Суффикс | Описание | Температура | Скорость |
|---------|----------|-------------|----------|
| **2Z** | Две металлические шайбы | -30 до +110°C | Высокая |
| **2RS1** | Два контактных резиновых уплотнения (стандарт) | -30 до +110°C | Средняя |
| **2RS2** | Два контактных уплотнения (улучшенная смазка) | -30 до +110°C | Средняя |
| **2RSH** | Два резиновых уплотнения (высокая температура) | -30 до +150°C | Средняя |
| **2RSL** | Два низкомоментных уплотнения | -40 до +110°C | Высокая |
| **2RZ** | Два неконтактных уплотнения | -40 до +120°C | Очень высокая |

#### Радиальный зазор

| Суффикс | Зазор |
|---------|-------|
| **C1** | Меньше C2 |
| **C2** | Меньше нормального |
| **CN** | Нормальный (обычно не указывается) |
| **C3** | Больше нормального |
| **C4** | Больше C3 |
| **C5** | Больше C4 |

#### Класс точности

| Суффикс | Класс точности | Применение |
|---------|----------------|------------|
| **(без)** | P0 (нормальный) | Общепромышленное |
| **P6** | Повышенный | Станки, редукторы |
| **P5** | Высокий | Шпиндели |
| **P4** | Прецизионный | Высокоточные станки |
| **P2** | Сверхпрецизионный | Измерительное оборудование |

#### Материал клетки (сепаратора)

| Суффикс | Материал | Применение |
|---------|----------|------------|
| **J** | Штампованная стальная клетка | Стандарт |
| **M** | Массивная латунная клетка | Тяжёлые нагрузки |
| **MA** | Массивная латунная (модифицированная) | Специальное применение |
| **TN9** | Полиамидная клетка (армированная) | Высокие скорости, низкий шум |
| **TVP** | Полиамидная клетка (стекловолокно) | Высокие температуры |
| **TVPB** | Полиамидная клетка (синяя) | Пищевая промышленность |

#### Специальные обозначения

| Суффикс | Описание |
|---------|----------|
| **/C3** | Радиальный зазор C3 (наиболее частый суффикс) |
| **N** | Канавка на наружном кольце |
| **NR** | Канавка + стопорное кольцо |
| **H** | Адаптерная втулка (для монтажа на цилиндрический вал) |
| **K** | Конический хвостовик (для монтажа на конический вал 1:12) |
| **VA405** | Специальная обработка для виброустойчивых применений |
| **W64** | Специальная смазка для высоких температур |
| **HT** | Высокотемпературное исполнение |
| **LHT** | Низко-высокотемпературное исполнение (-40 до +150°C) |
| **VT** | Низкотемпературное исполнение |
| **/DB** | Дуплекс спина-к-спине (back-to-back) |
| **/DF** | Дуплекс лицом-к-лицу (face-to-face) |
| **/DT** | Дуплекс тандем |

### Серия SKF Explorer

**Explorer** — высокопроизводительная линейка подшипников SKF, отличающаяся:
- Увеличенным ресурсом (до 2 раз)
- Оптимизированной внутренней геометрией
- Улучшенной термообработкой
- Низким уровнем шума и вибрации

**Обозначение**: Префикс **E2**

Примеры:
- E2.6205-2Z/C3 — Explorer, шариковый радиальный
- E2.NU 308 ECP/C3 — Explorer, роликовый цилиндрический

### Y-подшипники SKF (корпусные подшипники)

Специальная линейка для подшипниковых узлов с корпусом:

| Серия | Описание |
|-------|----------|
| **YAR** | С коническим отверстием и втулкой |
| **YET** | С эксцентриковым запирающим кольцом |
| **YAT** | Упорный Y-подшипник |
| **YSA** | Самоустанавливающийся Y-подшипник |
| **YSP** | Плунжерный Y-подшипник |

Пример: **YAR 205-2F** — Y-подшипник, d=25 мм, с двумя уплотнениями Taconite

## Примеры

### Пример 1: E2.6205-2Z/C3

- **E2** — серия Explorer (энергоэффективная)
- **6205** — шариковый радиальный, легкая серия, d=25 мм
- **2Z** — две металлические защитные шайбы
- **C3** — увеличенный радиальный зазор

**Применение**: Электродвигатели общего назначения.

**Аналог ГОСТ**: 6205-2Z C3 (базовое исполнение без оптимизации Explorer)

### Пример 2: NU 308 ECP/C3

- **NU** — роликовый цилиндрический с двумя бортами на наружном кольце
- **308** — средняя серия, d=40 мм
- **E** — усиленная конструкция (оптимизированная геометрия)
- **CP** — полиамидная клетка
- **C3** — увеличенный зазор

**Применение**: Редукторы, станки.

**Аналог ГОСТ**: 2308 или NU 308

### Пример 3: 22205 E/VA405

- **22205** — сферический роликовый двухрядный, d=25 мм
- **E** — усиленная конструкция
- **VA405** — специальная обработка для виброустойчивых применений

**Применение**: Вибрационное оборудование, грохоты, дробилки.

**Аналог ГОСТ**: 3205

### Пример 4: 6308-2RS1/C3 HT

- **6308** — шариковый радиальный, d=40 мм
- **2RS1** — два резиновых уплотнения (стандартные)
- **C3** — увеличенный зазор
- **HT** — высокотемпературное исполнение (до +150°C)

**Применение**: Сушильные барабаны, печи.

**Аналог ГОСТ**: 308-2RS C3 (без высокотемпературного исполнения)

### Пример 5: 7208 BECBP/DB

- **7208** — радиально-упорный шариковый, d=40 мм
- **BE** — угол контакта 40°
- **C** — зазор (специальный)
- **BP** — полиамидная клетка (прецизионная)
- **DB** — дуплекс back-to-back (спина-к-спине)

**Применение**: Шпиндели станков.

**Аналог ГОСТ**: 66208 (угол 26°) — требуется пересчёт

### Пример 6: YAR 206-2F

- **YAR** — Y-подшипник с коническим отверстием
- **206** — размер (d=30 мм с втулкой)
- **2F** — два уплотнения Taconite (для тяжёлых условий)

**Применение**: Корпусные подшипниковые узлы в сельхозтехнике, конвейерах.
