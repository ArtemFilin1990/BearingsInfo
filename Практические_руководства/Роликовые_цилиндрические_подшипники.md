# Роликовые цилиндрические подшипники

## Общие сведения

Роликовые цилиндрические подшипники — тип подшипников качения, в которых в качестве тел качения используются цилиндрические ролики. Они предназначены для восприятия больших радиальных нагрузок и обеспечивают высокую жесткость подшипникового узла.

## Конструктивные особенности

### Основные элементы

- **Внутреннее кольцо** — с дорожкой качения и, возможно, бортами
- **Наружное кольцо** — с дорожкой качения и бортами
- **Цилиндрические ролики** — тела качения
- **Сепаратор** — удерживает ролики (штампованный, массивный или оконный)

### Типы бортов

Роликовые цилиндрические подшипники различаются по наличию и расположению бортов:

- **NU** — без бортов на внутреннем кольце, два борта на наружном
- **N** — два борта на внутреннем кольце, без бортов на наружном
- **NJ** — один борт на внутреннем кольце, два борта на наружном
- **NUP** — два борта на внутреннем кольце (один съемный), два борта на наружном
- **NH** — два борта на внутреннем кольце (оба съемные), два борта на наружном

## Обозначение по ГОСТ

Роликовые цилиндрические подшипники обозначаются следующими цифрами в коде типа:

- **2** — роликовые с короткими цилиндрическими роликами
- **12** — двухрядные роликовые с короткими цилиндрическими роликами
- **32** — роликовые с длинными цилиндрическими роликами или игольчатые

### Примеры обозначений

- **2102** — роликовый цилиндрический, серия 0, d=15 мм
- **12313** — двухрядный роликовый, серия 3, d=65 мм
- **42305** — роликовый цилиндрический, серия 2, d=25 мм
- **32308** — роликовый с длинными роликами, серия 2, d=40 мм

## Модификации по конструкции

### По типу исполнения

| Тип | Борта на внутреннем кольце | Борта на наружном кольце | Применение |
|-----|----------------------------|--------------------------|------------|
| NU | Нет | Два | Радиальное фиксирование без осевого |
| N | Два | Нет | Свободный осевой ход наружного кольца |
| NJ | Один | Два | Фиксирование в одном направлении |
| NUP | Два (один съемный) | Два | Полное осевое фиксирование |
| NF | Два | Два | Стандартная конструкция |

### По количеству рядов

- **Однорядные** — стандартная конструкция
- **Двухрядные** — для восприятия больших радиальных нагрузок
- **Четырехрядные** — для прокатных станов и тяжелой промышленности

## Характеристики и возможности

### Преимущества

✓ Высокая радиальная грузоподъемность  
✓ Высокая жесткость подшипникового узла  
✓ Возможность осевого смещения вала относительно корпуса  
✓ Разъемная конструкция (некоторые типы)  
✓ Длительный срок службы  
✓ Высокая скорость вращения  
✓ Малое трение  

### Ограничения

✗ Не воспринимают осевые нагрузки (кроме типов NJ, NUP)  
✗ Чувствительны к перекосам  
✗ Требуют точной соосности  
✗ Больший вес по сравнению с шариковыми  

## Воспринимаемые нагрузки

- **Радиальные нагрузки** — очень большие (основное предназначение)
- **Осевые нагрузки** — не воспринимают (типы NU, N) или воспринимают ограниченные (типы NJ, NUP)
- **Комбинированные нагрузки** — только совместно с упорными подшипниками

## Типы сепараторов

### Штампованные (Т)
- Из стального листа
- Легкие, недорогие
- Для средних скоростей

### Массивные (М, Б, Л)
- Из латуни (Л), бронзы (Б), стали
- Высокая прочность
- Для высоких скоростей и нагрузок

### Оконного типа (Е)
- Из стеклотекстолита или полиамида
- Низкое трение
- Для высоких скоростей

## Области применения

### Промышленное оборудование
- Электродвигатели средней и большой мощности
- Коробки передач
- Редукторы
- Станки (шпиндели)
- Насосы и компрессоры

### Транспорт
- Железнодорожные буксы
- Тяговые двигатели
- Редукторы электровозов
- Оси вагонов

### Энергетическое оборудование
- Турбины
- Генераторы
- Насосные станции

### Металлургия
- Прокатные станы
- Кранов-балки
- Конвейеры

## Специальные исполнения

### Полнороликовые
- Без сепаратора
- Максимальная грузоподъемность
- Пониженная скорость вращения
- Применение в тяжелых условиях

### С модифицированным контактом роликов
- Логарифмический профиль роликов
- Снижение краевых напряжений
- Увеличение долговечности
- Маркировка: E (у SKF), TN (у FAG)

### Высокоскоростные
- Специальная конструкция сепаратора
- Оптимизированная смазка
- Прецизионная точность

## Рекомендации по выбору

### Критерии подбора

1. **Величина радиальной нагрузки** — основной фактор
2. **Необходимость осевого смещения** — выбор типа бортов
3. **Частота вращения** — проверка предельных оборотов
4. **Температура эксплуатации** — выбор материала сепаратора
5. **Точность вращения** — класс точности

### Рекомендуемые посадки

**Для внутреннего кольца (на вал):**
- Легкие нагрузки: js6, k6
- Нормальные нагрузки: m6, n6
- Тяжелые нагрузки: p6

**Для наружного кольца (в корпус):**
- Нормальные условия: H7
- С температурным расширением: G7, F7
- Повышенная точность: Js7, K7

## Монтаж и эксплуатация

### Правила монтажа

1. Использовать горячую посадку для больших подшипников
2. Обеспечить соосность посадочных мест
3. Контролировать зазоры
4. Не допускать перекосов
5. Использовать специальный инструмент

### Смазка

**Пластичная смазка:**
- Заполнение 30-40% свободного объема
- Интервал пополнения зависит от скорости и нагрузки

**Жидкая смазка:**
- Масляная ванна — для низких и средних скоростей
- Циркуляционная — для высоких скоростей и нагрузок
- Масляный туман — для высокоскоростных применений

### Контроль состояния

- Мониторинг вибрации
- Контроль температуры
- Анализ смазки
- Периодическая ревизия

## Дефекты и неисправности

### Основные причины отказов

1. **Усталостное выкрашивание** — естественный износ
2. **Износ дорожек качения** — абразив в смазке
3. **Задиры** — недостаток смазки, перегрузка
4. **Разрушение сепаратора** — перегрузки, высокие ускорения
5. **Перекосы** — неточность монтажа

### Признаки неисправности

- Повышенный шум
- Нагрев узла выше нормы
- Вибрация
- Увеличение осевого люфта (для типов NJ, NUP)

## Стандарты и нормативная база

- **ГОСТ 8328-75** — Подшипники роликовые радиальные с короткими цилиндрическими роликами

- **ГОСТ 520-2011** — Подшипники качения. Общие технические условия

## Аналоги

### ГОСТ ↔ ISO

Российские обозначения обычно соответствуют международным с некоторыми отличиями в дополнительных знаках:

- ГОСТ 2102 → ISO NU 202
- ГОСТ 42305 → ISO N 305
- ГОСТ 12313 → ISO NNU 4913
