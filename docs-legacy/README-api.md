# bearing-api - FastAPI приложение для работы с подшипниками

*Этот файл является заглушкой для документации из репозитория ArtemFilin1990/bearing-api*

## Оригинальное содержимое

Репозиторий bearing-api содержал:
- FastAPI приложение
- Пайплайн импорта в PostgreSQL
- Docker-контейнеризация
- GitHub Actions автоматизация

## Текущее размещение

Содержимое из репозитория bearing-api теперь находится в:
- `api/` - FastAPI приложение и все связанные модули
- `api/app/` - основное приложение
- `api/scripts/` - скрипты импорта и обработки
- `api/sql/` - SQL скрипты инициализации
- `data/inbox/` - входящие файлы для импорта

См. [MIGRATION_GUIDE.md](../MIGRATION_GUIDE.md) для подробностей.
