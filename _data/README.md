# Данные для БД (WIP)

Единый слой структурированных данных для последующей загрузки в БД.

## Структура
- `schema/` — описания схем и форматов
- `reference/` — справочники и классификаторы
- `mappings/` — сопоставления и кросс-ссылки
- `dimensions/` — размерные и параметрические таблицы
- `indexes/` — индексы и вспомогательные списки
