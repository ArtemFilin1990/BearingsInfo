# Класс точности подшипников ГОСТ, ISO, ABEC

## Определение

Класс точности подшипника — нормированная совокупность предельных отклонений размеров, формы и расположения поверхностей подшипника, характеризующая точность его изготовления. Чем выше класс, тем меньше допуски и выше эксплуатационные характеристики.

## Классификация по ГОСТ

### Ряд классов точности (по возрастанию)

**Для шариковых и роликовых радиальных:**
```
8 → 7 → 0 → 6 → 5 → 4 → Т → 2
      (нормальный)              (сверхпрецизионный)
```

**Для роликовых конических:**
```
8 → 7 → 0 → 6Х → 6 → 5 → 4 → 2
```

**Для упорных:**
```
8 → 7 → 0 → 6 → 5 → 4 → 2
```

### Характеристика классов

| Класс | Наименование | Применение | Точность изготовления |
|-------|--------------|------------|----------------------|
| **8** | Ниже нормального | Неответственные узлы | Пониженная |
| **7** | Ниже нормального | Сельхозмашины, конвейеры | Пониженная |
| **0** | **Нормальный** | Общее машиностроение | Нормальная |
| **6Х** | Повышенный (конич.) | Конические подшипники | Повышенная |
| **6** | Повышенный | Автомобили, станки | Повышенная |
| **5** | Высокий | Прецизионное оборудование | Высокая |
| **4** | Прецизионный | Шпиндели станков | Прецизионная |
| **Т** | Особо прецизионный | Координатно-расточные станки | Особо прецизионная |
| **2** | Сверхпрецизионный | Измерительные приборы | Сверхпрецизионная |

## Влияние класса точности на эксплуатационные характеристики

| Параметр | Класс 0 | Класс 6 | Класс 5 | Класс 4 | Класс 2 |
|----------|---------|---------|---------|---------|---------|
| **Максимальная скорость** | 100% | +10% | +50% | +100% | +100% |
| **Вибрация** | Высокая | Средняя | Низкая | Очень низкая | Минимальная |
| **Шум** | Высокий | Средний | Низкий | Очень низкий | Минимальный |
| **Радиальное биение** | Высокое | Среднее | Низкое | Очень низкое | Минимальное |
| **Тепловыделение** | Высокое | Среднее | Низкое | Очень низкое | Минимальное |
| **Стоимость** | Базовая | +20% | +50% | +150% | +300% |

### Влияние на предельную частоту вращения

**Коэффициенты увеличения n_lim:**
- **Класс 5:** частота увеличивается в **1,5 раза** (шариковые, роликовые цилиндрические)
- **Класс 4:** частота увеличивается в **2 раза** (шариковые, роликовые цилиндрические)
- **Класс 5:** частота увеличивается в **1,1 раза** (конические, упорные)
- **Класс 4:** частота увеличивается в **1,2 раза** (конические, упорные)

## Таблица соответствия ГОСТ ↔ ISO ↔ ABEC

| ГОСТ 520-2011 | ISO 492 | ABEC (USA) | DIN (старый) | Точность изготовления |
|---------------|---------|------------|--------------|---------------------|
| **Класс 0** (нормальный) | P0 | ABEC1 | — | Нормальная |
| **Класс 6** | P6 | ABEC3 | Р | Повышенная |
| **Класс 5** | P5 | ABEC5 | А | Высокая |
| **Класс 4** | P4 | ABEC7 | В | Прецизионная |
| **Класс 2** | P2 | ABEC9 | С | Сверхпрецизионная |

**Примечания:**
- ISO P0 — обозначается как Normal или не указывается
- ABEC — стандарт США, распространен в скейтбордах, роликах
- Для класса Т (ГОСТ) нет прямого соответствия в ISO/ABEC

|------------------------|----------------------------|----------------------|
| **Н** | **0** или не указывается | Нормальный |
| **П** | **6** | Повышенный |
| **ВП** | Нет соответствия | Особо повышенный промежуточный |
| **В** | **5** | Высокий |
| **АВ** | Нет соответствия | Особо высокий промежуточный |
| **А** | **4** | Прецизионный |
| **СА** | **Т** | Особо прецизионный |
| **С** | **2** | Сверхпрецизионный |

**Пример старого обозначения:** С-236207е  
**Современное обозначение:** 2-36207Е

## Класс точности игольчатых подшипников

Для роликов игольчатых подшипников установлены **три степени точности** по ГОСТ 6870:

| Степень точности | Точность | Обозначение в маркировке |
|------------------|----------|--------------------------|
| **5** | Нормальная | Не указывается |
| **3** | Повышенная | Цифра 3 перед обозначением |
| **2** | Высокая | Не указывается (высшая) |

**Пример:** 3КК30×35×46Е  
- 3 — степень точности роликов (повышенная)
- КК — двухрядный без колец
- 30×35×46 — размеры d×D×B
- Е — сепаратор из текстолита

**Примечание:** Степень точности 2 в условном обозначении не проставляется (высшая точность).

## Класс точности тел качения

### Шарики (ГОСТ 3722-2014)

Установлено **11 степеней точности** (в порядке снижения):

```
G3 → G5 → G10 → G16 → G20 → G24 → G28 → G40 → G60 → G100 → G200
(высшая)                                                    (низшая)
```

**DIN 5401:** G700, G600, G500, G300, G200, G100, G80, G40, G28, G20, G16, G10, G5, G3

### Ролики цилиндрические

**Короткие ролики:** IV, IIIA, III, IIA, II, I (в порядке повышения точности)  
**Длинные ролики:** III, II, I (в порядке повышения точности)

### Иголки

**Три степени точности:** 4, 3, 2 (в порядке повышения точности)

## Дополнительное обозначение "У"

Буква **"У"** после класса точности означает **повышенную точность конических роликовых подшипников по монтажной высоте**.

**Пример:** 6У-7608  
- 6 — класс точности
- У — повышенная точность по монтажной высоте
- 7608 — конический подшипник

| Отрасль | Типичные классы | Примеры оборудования |
|---------|----------------|---------------------|
| **Общее машиностроение** | 0, 6 | Насосы, вентиляторы, конвейеры |
| **Автомобилестроение** | 0, 6, 5 | Двигатели, коробки передач, ступицы |
| **Станкостроение** | 5, 4 | Шпиндели токарных, фрезерных станков |
| **Прецизионное оборудование** | 4, 2, Т | Координатно-расточные станки |
| **Авиация** | 5, 4, 2 | Двигатели, системы управления |
| **Электроника** | 5, 4 | Электродвигатели, сервоприводы |
| **Измерительная техника** | 2, Т | Приборы, гироскопы |

## Примечания

**Важные особенности:**

1. **Класс 0** при отсутствии специальных требований **не указывается** в обозначении
2. Для конических подшипников класс **6Х** обозначается только буквой **Х**
3. С повышением класса точности **требования ко всем элементам** возрастают (внутренние и внешние допуски)
4. Подшипники классов **4 и 2** называют **прецизионными**
5. Точность деталей определяет класс всего подшипника (по наименее точной детали)

**Рекомендации:**

- Для **скоростных узлов** — класс 5 или выше
- Для **повышения точности вращения** — класс 5, 4, 2
- Для **снижения вибрации** — класс 6 или выше
- Для **общепромышленного применения** — класс 0 достаточен
- Высокие классы существенно **дороже** (в 1.5-3 раза и более)
