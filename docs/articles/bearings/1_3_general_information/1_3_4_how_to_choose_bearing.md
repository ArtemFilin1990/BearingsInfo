## Порядок подбора

### 1. Определить исходные данные
- Радиальная нагрузка Fr (Н)
- Осевая нагрузка Fa (Н)
- Частота вращения n (об/мин)
- Требуемый ресурс Lh (часы)

### 2. Выбрать тип подшипника

| Условие | Тип |
|---------|-----|
| Только Fr | Радиальный шариковый (6xxx), цилиндрический (Nxxx) |
| Fr + Fa | Конический (7xxx), радиально-упорный (36xxx) |
| Только Fa | Упорный (8xxx, 9xxx) |
| Перекосы | Сферический (1xxx, 3xxx) |

### 3. Определить размер

По диаметру вала d выбрать из стандартного ряда.

### 4. Рассчитать грузоподъемность

```
P = X×Fr + Y×Fa  (эквивалентная нагрузка)

C = P × (Lh×n×60/10⁶)^(1/p)

p = 3 (шариковые), 10/3 (роликовые)
```

### 5. Выбрать подшипник из каталога

C(подшипника) ≥ C(расчетное)

### 6. Проверить скорость

n ≤ n(предельная)

### 7. Определить дополнительные параметры
- Класс точности
- Радиальный зазор
- Уплотнения

## Пример

**Условия**: Fr=2000Н, n=1500 об/мин, d=25мм, Lh=10000ч

**Решение**:
- Тип: радиальный шариковый
- Размер: 6205 (d=25мм)
- C(требуемая) = 2000×(10000×1500×60/10⁶)^(1/3) = 11.4 кН
- C(6205) = 14.8 кН ✓
- n(предельное, 6205) = 16000 об/мин ✓

**Ответ**: 6205
