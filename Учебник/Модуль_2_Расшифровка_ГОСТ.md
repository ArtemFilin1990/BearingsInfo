# Модуль 2: Расшифровка обозначений ГОСТ (Ключевой модуль)

## Цель модуля

Научиться **разбирать обозначения подшипников ГОСТ** на составляющие элементы, используя реальные данные из CSV-файлов и справочных статей репозитория.

---

## 2.1. Теоретическая основа

### Структура обозначения подшипника ГОСТ

Согласно [`../docs/articles/bearings/1_1_designations_gost_iso_etu/1_1_1_gost_designation_system.md`](../docs/articles/bearings/1_1_designations_gost_iso_etu/1_1_1_gost_designation_system.md):

```
[Префикс] X-XXYYZZ [Суффикс]
          │  │ │ │
          │  │ │ └─── Код внутреннего диаметра (2 цифры)
          │  │ └───── Серия ширины (1 цифра)
          │  └─────── Серия диаметров (1 цифра)
          └────────── Тип подшипника (1-2 цифры)
```

### Компоненты обозначения

1. **Тип подшипника** (1-2 цифры слева)
2. **Серия диаметров** (1 цифра)
3. **Серия ширины** (1 цифра)
4. **Код внутреннего диаметра** (2 цифры справа)
5. **Дополнительные обозначения** (префиксы/суффиксы)

## 2.3. Расшифровка кода внутреннего диаметра

Источник: [`../docs/articles/bearings/1_1_designations_gost_iso_etu/1_1_6_bore_diameter_designation.md`](../docs/articles/bearings/1_1_designations_gost_iso_etu/1_1_6_bore_diameter_designation.md)

### Правило расчета

| Код | Внутренний диаметр d (мм) |
|-----|---------------------------|
| **00** | d = 10 мм |
| **01** | d = 12 мм |
| **02** | d = 15 мм |
| **03** | d = 17 мм |
| **04** | d = 20 мм |
| **05 и выше** | d = код × 5 (например, 05 = 25 мм) |

**Примеры:**
- Код **05** → d = 5 × 5 = **25 мм**
- Код **06** → d = 6 × 5 = **30 мм**
- Код **10** → d = 10 × 5 = **50 мм**

### Расширенная справочная таблица суффиксов ISO

Ниже приведена детальная справочная информация по суффиксам, используемым различными производителями подшипников.

#### Обозначения уплотнений

| Суффикс | Производитель | Описание | Эквивалент SNR |
|---------|---------------|----------|----------------|
| **2RS** | FAG / GMN / INA / KOYO / RHP | Двухстороннее резиновое уплотнение | EE |
| **2RS1** | SKF | Двухстороннее резиновое уплотнение | EE |
| **2RS1N** | SKF | Двухстороннее уплотнение + канавка для стопорного кольца | N EE |
| **2RS2** | SKF | Двухстороннее уплотнение для высокой температуры | EE3 |
| **2RSR** | FAG / RHP | Двухстороннее резиновое уплотнение | EE |
| **2LS** | SKF | Двухстороннее уплотнение | EE |
| **2T** | TORRINGTON | Двухстороннее уплотнение | EE |
| **ZZ** | Различные | Два металлических пыльника | Z |
| **Z** | Различные | Один металлический пыльник | Z |

#### Обозначения смазки

| Суффикс | Производитель | Описание | Код смазки |
|---------|---------------|----------|------------|
| **15A** | NTN | Специальная смазка | D109 |
| **1D** | NTN | Специальная смазка | D59 |
| **1E** | NTN | Специальная смазка | D109 |

#### Обозначения класса точности и специальных исполнений

| Суффикс | Производитель | Описание |
|---------|---------------|----------|
| **2MM** | FAFNIR | Шариковый радиально-упорный, класс точности ABEC 7, угол контакта 15° |
| **P5** | Различные | Класс точности ISO 5 (соответствует ABEC 5) |
| **P4** | Различные | Класс точности ISO 4 (соответствует ABEC 7) |
| **P2** | Различные | Класс точности ISO 2 (сверхвысокая точность) |

#### Обозначения комплектов подшипников

| Суффикс | Производитель | Описание |
|---------|---------------|----------|
| **2TB2T** | RHP | Шариковый радиально-упорный, комплект из 4 подшипников |
| **2TF** | RHP | Комплект из 3 подшипников, схема "X" (наружные кольца обращены узкими торцами) |
| **2TF2T** | RHP | Комплект из 4 подшипников |
| **Q** | Различные | Обозначение комплектов (используется в сочетании с другими суффиксами) |

#### Обозначения зазоров

| Суффикс | Описание |
|---------|----------|
| **C2** | Зазор меньше нормального |
| **CN** | Нормальный зазор (обычно не указывается) |
| **C3** | Зазор больше нормального |
| **C4** | Зазор значительно больше нормального |
| **C5** | Зазор намного больше нормального |

### Применение суффиксов при заказе подшипников

**Примеры полных обозначений:**

1. **6205-2RS C3** 
   - Базовое: 6205 (радиальный шариковый, d=25 мм)
   - 2RS: Два резиновых уплотнения
   - C3: Увеличенный зазор

2. **6308-ZZ P6**
   - Базовое: 6308 (радиальный шариковый, d=40 мм)
   - ZZ: Два металлических пыльника
   - P6: Повышенная точность (класс 6)

3. **6205-2RS1N**
   - Базовое: 6205
   - 2RS1: Уплотнение SKF
   - N: Канавка для стопорного кольца на наружном кольце

### Важные замечания

> ⚠️ **Суффиксы разных производителей могут отличаться!**
> 
> Один и тот же тип уплотнения может обозначаться по-разному:
> - SKF: 2RS1
> - FAG: 2RS
> - SNR: EE
> 
> При заказе всегда уточняйте обозначение у конкретного производителя!

**Для точной расшифровки:**
- Используйте каталог конкретного производителя (Модуль 4)
- Сверяйте с техническими справочниками производителей
- Проверяйте технические требования вашего применения

## 2.7. Алгоритм расшифровки

**Шаг 1:** Выделите базовое обозначение (без префиксов/суффиксов)  
**Шаг 2:** Определите тип подшипника (первая цифра) → проверьте в [`../data/gost/bearings.csv`](../data/gost/bearings.csv)  
**Шаг 3:** Выделите серию диаметров и ширины  
**Шаг 4:** Вычислите внутренний диаметр по коду (последние 2 цифры × 5)  
**Шаг 5:** Найдите размеры D, B в [`../data/gost/dimensions.csv`](../data/gost/dimensions.csv)  
**Шаг 6:** Расшифруйте суффиксы (класс точности, уплотнения, зазор)
