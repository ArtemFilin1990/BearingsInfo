# Аналоги подшипников: сопоставление ГОСТ ↔ ISO

## Purpose
- Обозначить зону хранения таблиц соответствия и правил сопоставления между ГОСТ и ISO без смешения с первичными стандартами.
- Задать базовые проверки (габариты, серия, зазор, исполнение) для безопасного подбора аналогов.

## Classification
- Типовые группы соответствий: глубокие радиальные шариковые (серии 60/62/63/64), конические роликовые (302/303 серии), сферические роликовые (222/223/232/238), игольчатые.
- Сопоставление выполняется по граничным размерам (d/D/B) и назначению нагрузки: радиальные ↔ радиальные, радиально-упорные ↔ радиально-упорные, упорные ↔ упорные.

## Designation
- Базовое правило: к трёхзначному обозначению ГОСТ добавляется ведущая цифра серии ISO. Примеры: 205 → 6205 (лёгкая серия), 305 → 6305 (средняя серия), 405 → 6405 (тяжёлая серия).
- При подборе аналога учитываются серия ширины/диаметров, код отверстия (по ISO 15) и требования к посадкам/зазору по ГОСТ 24810-81.
- Полные таблицы соответствий и частные случаи (например, упорные или игольчатые подшипники) должны ссылаться на исходные каталоги производителей.

## Prefixes/Suffixes
- Прямой перенос обозначений допускается только для базового номера. Суффиксы по зазору (C2/C3/C4/C5), материалам сепараторов (E, J, TVP, и др.) и защитам (Z/ZZ/RS/2RS) необходимо сверять с каталогами.
- Префиксы ГОСТ (группа зазора, класс точности) не всегда имеют прямой аналог в ISO; при отсутствии явного соответствия требуется подбор по допустимым биениям и зазорам.

## Examples
- **205 ↔ 6205**: d=25 мм, D=52 мм, B=15 мм (источник: TD IRBIS). Прямой аналог без изменения серии.
- **305 ↔ 6305**: d=25 мм, D=62 мм, B=17 мм (источник: TD IRBIS). Средняя серия для повышенной нагрузки.
- **405 ↔ 6405**: d=25 мм, D=80 мм, B=21 мм (источник: aprom.by). Тяжёлая серия.

## Notes & Limitations
- Совпадение габаритов не гарантирует совместимости по зазору, классу точности и исполнению уплотнений; эти параметры должны уточняться по ГОСТ 3189-89 и ISO 15/355.
- Номера с нестандартными префиксами/суффиксами (например, увеличенный зазор, нестандартный сепаратор, смазка) требуют прямой проверки в каталоге производителя.
- Желательно фиксировать источник данных в таблицах (ГОСТ, ISO, каталог завода) для каждой пары аналогов.

## Standard References
- ГОСТ 3189-89 — система обозначений ГОСТ.
- ГОСТ 520-89 / 520-2011 — классы точности и требования к биениям.
- ГОСТ 24810-81 — группы радиального зазора.
- ISO 15 — Rolling bearings — Radial bearings — Boundary dimensions.
- ISO 355 — Rolling bearings — Tapered roller bearings — Boundary dimensions.
- Каталоги TD IRBIS (размерные таблицы) и aprom.by (тяжёлая серия 6400).
