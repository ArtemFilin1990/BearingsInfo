# GOST - Стандарты ГОСТ

Директория содержит стандарты ГОСТ на подшипники и сопутствующие изделия.

## Содержание

### Стандарты на подшипники качения
- Общие технические условия
- Классификация и типы
- Размеры и допуски
- Обозначения
- Методы испытаний

### Стандарты на резинотехнические изделия (РТИ)
- Манжеты и сальники
- Уплотнительные кольца
- Рукава высокого давления
- Технические пластины
- Ремни приводные

### Стандарты на приводные элементы
- Ремни клиновые
- Ремни плоские
- Цепи приводные
- Звездочки

## Основные стандарты

### Подшипники
- ГОСТ 520-2002 - Подшипники качения. Общие технические условия
- ГОСТ 3478-2012 - Основные нормы взаимозаменяемости. Подшипники качения
- ГОСТ 8338-75 - Подшипники шариковые радиальные однорядные

### РТИ и уплотнения
- ГОСТ 8752-79 - Манжеты резиновые армированные для валов
- ГОСТ 9833-73 - Кольца резиновые уплотнительные
- ГОСТ 6286-73 - Рукава резиновые высокого давления

### Приводные ремни
- ГОСТ 1284.1-89 - Ремни приводные клиновые нормальных сечений
- ГОСТ 23831-79 - Ремни плоские приводные

## Применение

Стандарты используются для:
- Проектирования механизмов
- Подбора компонентов
- Обеспечения взаимозаменяемости
- Контроля качества
- Составления технической документации
