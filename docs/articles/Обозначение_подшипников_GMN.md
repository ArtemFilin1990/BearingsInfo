## Расшифровка обозначения высокоточных прецизионных радиально-упорных шариковых подшипников  GMN.

материал		HY - керамические шарики без символа - стандартный подшипник	
	серия		S60 S619 S62	
	внутренний диаметр		00 - 10мм 01 - 12мм 02 - 15мм 03 - 17мм 04 - 20мм Свыше 04: Внутренний диаметр подшипника = номер диаметра х5 (мм) 	
	бесконтактные уплотнения		без символа - открытый подшипник X-2Z - закрытый подшипник	
	угол контакта		C - 15 град E - 25 град	
	сепаратор		TA - текстолитовый сепаратор TB - текстолитовый сепаратор TAM - текстолитовый сепаратор TXM - PEEK	
	класс точности		P4 - 4 класс точности по ISO P2 - 2 класс точности по ISO A7 (ABEC7) - 4 класс точности по ISO A9 (ABEC9) - 2 класс точности по ISO	
	схема установки		U - универсальный подшипник DU - универсальный дуплекс TU - универсальный триплекс DB - "спина к спине" DF - "лицо к лицу" DT - "тандем"	
	преднатяг		L - легкий преднатяг M - средний преднатяг S - тяжелый преднатяг	
	смазка		L252 - стандартная смазка без символа - подшипник без смазки
