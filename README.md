## О проекте

Данный репозиторий представляет собой справочную систему для работы с подшипниками качения по ГОСТ. Проект предназначен для инженеров, конструкторов и специалистов в области машиностроения.

### Назначение

- Быстрый поиск технических характеристик подшипников
- Справочная информация по маркировке и классификации
- База данных для проектирования механических систем
- Обучающий материал для студентов технических специальностей

## Типы подшипников

### Шариковые подшипники
- Радиальные однорядные
- Радиальные сферические двухрядные
- Радиально-упорные однорядные
- Радиально-упорные двухрядные
- Упорные однорядные
- Упорные двухрядные

### Роликовые подшипники
- Радиальные с короткими цилиндрическими роликами
- Радиальные сферические двухрядные
- Радиальные с витыми роликами
- Радиально-упорные конические
- Упорные с цилиндрическими роликами
- Упорные сферические однорядные

### Игольчатые подшипники
- Радиальные с сепаратором
- Радиальные без внутреннего кольца
- Радиально-упорные

## Классификация

### По типу тел качения
1. **Шариковые** - используют шарики в качестве тел качения
2. **Роликовые** - используют ролики различной формы
3. **Игольчатые** - используют игольчатые ролики

### По направлению воспринимаемой нагрузки
- **Радиальные** - воспринимают радиальную нагрузку
- **Упорные** - воспринимают осевую нагрузку
- **Радиально-упорные** - комбинированная нагрузка

### По количеству рядов тел качения
- Однорядные
- Двухрядные
- Многорядные

## Технические характеристики

### Основные параметры

#### Геометрические размеры
- **d** - внутренний диаметр (мм)
- **D** - наружный диаметр (мм)
- **B** - ширина подшипника (мм)
- **C** - монтажная высота (мм)

#### Эксплуатационные характеристики
- **Cr** - динамическая грузоподъемность (кН)
- **C0r** - статическая грузоподъемность (кН)
- **n предельная** - предельная частота вращения (об/мин)
- **Масса** - вес подшипника (кг)

### Точность изготовления (классы)
- Класс 0 (нормальный)
- Класс 6 (повышенный)
- Класс 5 (высокий)
- Класс 4 (особо высокий)
- Класс 2 (прецизионный)

### Радиальные зазоры
- Группа C2 (меньше нормального)
- Группа C0 (нормальный)
- Группа C3 (больше нормального)
- Группа C4 (большой)
- Группа C5 (особо большой)

## Маркировка

### Структура обозначения подшипника

```
XXXXXX YY ZZ
│      │  │
│      │  └─ Дополнительные характеристики
│      └──── Серия диаметров и типа

```

### Примеры маркировки

- **205** - шариковый радиальный однорядный, серия диаметров 2, внутренний диаметр 25 мм
- **7305** - радиально-упорный шариковый, угол контакта 15°, серия 03, d=25 мм
- **3182120** - радиально-упорный роликовый конический, серия 2, d=100 мм

### Префиксы и суффиксы
- **C** - сепаратор из латунной ленты
- **E** - усиленная конструкция
- **K** - конический
- **N** - наличие канавки на наружном кольце
- **H** - закрепительная втулка

## Установка и использование

### Требования к монтажу

1. **Подготовка посадочных мест**
   - Проверка чистоты поверхностей
   - Контроль геометрии валов и отверстий
   - Соблюдение допусков и посадок

2. **Правила установки**
   - Использование специализированного инструмента
   - Равномерное приложение усилия
   - Контроль соосности

3. **Смазка подшипников**
   - Пластичные смазки
   - Жидкие масла
   - Твердые смазочные материалы

### Эксплуатация

- Регулярный контроль состояния
- Периодическое пополнение смазки
- Мониторинг температурного режима
- Контроль уровня вибрации
