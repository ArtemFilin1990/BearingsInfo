# Источники данных и файлы с данными подшипников

## Описание

Этот каталог содержит исходные материалы, скрипты и структурированные данные для базы знаний по подшипникам.

## Структура файлов

### Документация источников

### Файлы с данными (JSON)

#### Размерные данные подшипников

##### bearing_dimensions_6000_series.json
Технические характеристики подшипников серии 6000 (особо легкая серия).

**Охват:** 11 подшипников серии 6000 (6000-6010)

**Источники:** SKF, NSK, podshipnik.info

##### bearing_dimensions_6200_series.json
Технические характеристики подшипников серии 6200 (легкая серия, самая популярная).

**Охват:** 13 подшипников серии 6200 (6200-6212)

**Источники:** SKF, NSK, IRBIS

##### bearing_dimensions_6300_series.json
Технические характеристики подшипников серии 6300 (средняя серия).

**Охват:** 13 подшипников серии 6300 (6300-6312)

**Источники:** SKF, NSK, каталог

##### bearing_dimensions_cylindrical_rollers.json
Технические характеристики роликовых цилиндрических подшипников.

**Охват:** 14 подшипников серий NU 200 и NU 300

**Источники:** SKF, NSK, каталог

##### bearing_dimensions_tapered_rollers.json
Технические характеристики роликовых конических подшипников.

**Охват:** 16 подшипников серий 30200, 30300, 32000

**Источники:** Timken, SKF, каталог

##### bearing_dimensions_spherical.json
Технические характеристики сферических подшипников (шариковые и роликовые самоустанавливающиеся).

**Охват:** 11 подшипников серий 1200, 22200, 22300

**Источники:** SKF, NSK, каталог

##### bearing_dimensions_angular_contact.json
Технические характеристики радиально-упорных шариковых подшипников.

**Охват:** 5 подшипников серии 7200 (углы контакта 15° и 40°)

**Источники:** SKF, NSK, каталог

##### bearing_dimensions_thrust.json
Технические характеристики упорных шариковых подшипников.

**Охват:** 8 подшипников серий 51100 и 51200

**Источники:** SKF, NSK, каталог

#### Таблицы соответствий ГОСТ ↔ ISO

##### gost_iso_correspondence.json
Таблица соответствий обозначений подшипников между стандартами ГОСТ и ISO.

**Охват:** 26 соответствий, включая:
- Шариковые радиальные (серии 6000, 6200, 6300)
- Роликовые цилиндрические (серия NU)
- Роликовые конические (серии 30200, 30300)
- Роликовые сферические (серия 22000)
- Упорные шариковые (серии 51100, 51200)

**Источники:** planetapodshipnik.ru, podshipnik.info, Timken

## Использование данных

### Для разработчиков

Данные в JSON формате могут быть использованы для:
- Создания веб-приложений для подбора подшипников
- Разработки API для доступа к справочной информации
- Интеграции с системами проектирования (CAD/CAM)
- Автоматизации процессов подбора и заказа

### Пример загрузки данных (Python)

```python
import json

# Загрузка размерных данных серии 6200
with open('bearing_dimensions_6200_series.json', 'r', encoding='utf-8') as f:
    bearings_6200 = json.load(f)

# Поиск подшипника по внутреннему диаметру
target_d = 25
result = [b for b in bearings_6200 if b['d_mm'] == target_d]
print(f"Найдено подшипников с d={target_d}мм: {len(result)}")
```

### Пример загрузки данных (JavaScript)

```javascript
// Node.js
const fs = require('fs');

// Загрузка таблицы соответствий ГОСТ-ISO
const correspondence = JSON.parse(
  fs.readFileSync('gost_iso_correspondence.json', 'utf-8')
);

// Поиск ISO аналога по ГОСТ обозначению
const gostDesignation = '6205';
const isoEquivalent = correspondence.find(c => c.gost === gostDesignation);
console.log(`ГОСТ ${gostDesignation} = ISO ${isoEquivalent.iso}`);
```

## Источники данных

Все данные собраны из проверенных источников:

### Официальные производители
- **SKF** (Швеция) - мировой лидер производства подшипников
- **NSK** (Япония) - крупнейший японский производитель
- **NTN** (Япония) - технические каталоги и спецификации
- **Timken** (США) - специализация на конических подшипниках

### Справочные порталы
- **podshipnik.info** - каталог подшипников ГОСТ
- **irbis.ua** - таблицы размеров (2600+ типоразмеров)
- **planetapodshipnik.ru** - таблицы соответствий ISO/GOST
- **kugellager-express.de** - международный каталог

### Стандарты
- ГОСТ 520-2002 - Подшипники качения. Общие технические условия
- ГОСТ 3478-2012 - Основные нормы взаимозаменяемости
- ISO 15 - Rolling bearings – Radial bearings – Boundary dimensions

## Расширение базы данных

### Добавление новых данных

1. **Проверьте источник** - используйте только официальные каталоги или проверенные справочники
2. **Соблюдайте структуру** - следуйте существующему формату JSON
3. **Указывайте источники** - всегда добавляйте поле "source" с перечислением источников
4. **Проверяйте данные** - сверьте данные минимум из двух независимых источников

### Примеры расширения

Для добавления новой серии подшипников:
1. Создайте файл `bearing_dimensions_XXXX_series.json`
2. Используйте существующую структуру данных
3. Документируйте источники
4. Обновите этот README.md

## Валидация данных

Перед использованием данных рекомендуется:
- Проверить корректность размеров (d < D)
- Убедиться в логичности грузоподъемности (C > C0 обычно)
- Сверить с официальными каталогами для критических применений
