# Технические характеристики подшипников

Раздел посвящен основным техническим характеристикам и параметрам подшипников.

## Содержание раздела

Материалы о:
- Размерных рядах и сериях
- Нагрузочных характеристиках
- Скоростных параметрах
- Специальных характеристиках

## Основные материалы

- **Обозначение размерных серий подшипников** - система обозначения габаритов
- **Обозначение внутреннего диаметра подшипников** - кодирование диаметра отверстия
- **Обозначение категорий подшипников** - классификация по назначению
- **Нагрузка на подшипники** - расчет и оценка нагрузок
- **Обозначение момента трения подшипников** - характеристики трения
- **Высокотемпературные подшипники** - подшипники для работы при повышенных температурах
- **Подшипники в виброустойчивом исполнении** - конструкции для вибронагруженных узлов
- **Угол контакта - подшипники шариковые радиально-упорные однорядные** - параметры радиально-упорных подшипников

## Основные параметры

- Внутренний диаметр (d)
- Наружный диаметр (D)
- Ширина (B, T, C)
- Динамическая грузоподъемность (C)
- Статическая грузоподъемность (C₀)
- Предельная частота вращения
- Угол контакта (для радиально-упорных)

## Серии подшипников

- Сверхлегкая серия (1)
- Легкая серия (2)
- Средняя серия (3)
- Тяжелая серия (4)
