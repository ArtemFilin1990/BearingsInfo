## Типы нагрузок

### Радиальная нагрузка (Fr)
Перпендикулярна оси вращения

### Осевая нагрузка (Fa)
Вдоль оси вращения

### Комбинированная
Fr + Fa одновременно

## Эквивалентная динамическая нагрузка

```
P = X×Fr + Y×Fa

X, Y — коэффициенты (зависят от типа подшипника и Fa/Fr)
```

### Для радиальных шариковых:
- Fa/Fr ≤ e: P = Fr
- Fa/Fr > e: P = 0.56×Fr + Y×Fa

## Расчет ресурса

```
L₁₀ = (C/P)³ × 10⁶ оборотов (шариковые)
L₁₀ = (C/P)^(10/3) × 10⁶ оборотов (роликовые)
```

## Факторы безопасности

Для надежности умножают P на коэффициенты:
- 1.0-1.2: равномерная нагрузка
- 1.5: умеренные толчки
- 2.0-3.0: сильные удары
