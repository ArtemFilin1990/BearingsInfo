# 1.3.2 Из чего состоит подшипник

## Цель

Описать основные конструктивные элементы подшипника качения, материалы изготовления и их функциональное назначение для понимания работы и правильной эксплуатации подшипников.

## Ключевые термины

- **Внутреннее кольцо** — кольцо с дорожкой качения, устанавливаемое на вал
- **Наружное кольцо** — кольцо с дорожкой качения, устанавливаемое в корпус
- **Тела качения** — шарики или ролики, передающие нагрузку между кольцами
- **Сепаратор (клетка)** — элемент, удерживающий тела качения на равном расстоянии
- **Дорожка качения** — рабочая поверхность колец, по которой катятся тела качения
- **Борт (бортик)** — боковая направляющая поверхность на кольце для удержания тел качения
- **Фаска** — скруглённый переход между рабочими поверхностями

## Суть

### Основные элементы подшипника качения

#### 1. Кольца (внутреннее и наружное)

**Внутреннее кольцо:**
- Устанавливается на вал (обычно с натягом)
- Имеет внутреннюю цилиндрическую или коническую посадочную поверхность
- На наружной поверхности расположена дорожка качения
- Может иметь борта для удержания тел качения

**Наружное кольцо:**
- Устанавливается в корпус или отверстие
- Имеет наружную цилиндрическую посадочную поверхность
- На внутренней поверхности расположена дорожка качения
- Может иметь борта для удержания тел качения

**Особенности:**
- В некоторых типах подшипников одно из колец может отсутствовать (игольчатые, упорные)
- Для конических роликовых подшипников кольца разъёмные
- Дорожки качения закаливаются до твёрдости HRC 58-65

#### 2. Тела качения

**Типы:**

| Тип | Форма | Применение | Преимущества |
|-----|-------|------------|--------------|
| **Шарики** | Сферическая | Радиальные, радиально-упорные | Высокие скорости, низкое трение, универсальность |
| **Цилиндрические ролики** | Цилиндр | Радиальные роликовые | Высокая радиальная грузоподъёмность |
| **Конические ролики** | Усечённый конус | Конические роликовые | Комбинированная нагрузка |
| **Бочкообразные ролики** | Выпуклая боковая поверхность | Сферические роликовые | Компенсация перекосов, высокая нагрузка |
| **Игольчатые ролики** | Тонкий длинный цилиндр (L/D ≥ 3) | Игольчатые | Компактность при высокой нагрузке |

**Материал:**
- Сталь ШХ15 (ГОСТ 801-78) — основной материал для подшипников
- Сталь ШХ15СГ — для крупных подшипников
- Нержавеющая сталь (например, AISI 440C) — для коррозионностойких
- Керамика (Si₃N₄) — для гибридных высокоскоростных подшипников

#### 3. Сепаратор (клетка)

**Назначение:**
- Разделяет тела качения на равном расстоянии
- Предотвращает контакт тел качения между собой
- Направляет движение тел качения
- Удерживает тела качения после демонтажа одного кольца

**Типы сепараторов:**

| Тип | Материал | Конструкция | Применение |
|-----|----------|-------------|------------|
| **Штампованный** | Стальной лист | Две половины с заклёпками | Стандартные подшипники, средние нагрузки |
| **Массивный** | Латунь, бронза, сталь | Механически обработанный | Тяжёлые нагрузки, высокие температуры |
| **Полимерный** | Полиамид (PA66, PA46) | Литой | Высокие скорости, низкий шум, лёгкие нагрузки |
| **Игольчатый** | Стальная лента | Штампованный | Игольчатые подшипники |

**Обозначения материалов:**
- **J** — штампованная стальная клетка
- **M** — массивная латунная клетка
- **TN9** — полиамидная клетка (стекловолокно)
- **TVP** — полиамидная клетка для высоких температур
- **Y** — латунная клетка (центрируемая по телам качения)

#### 4. Дополнительные элементы

**Уплотнения и защитные шайбы:**
- **Металлические шайбы (Z, 2Z)** — защита от пыли
- **Резиновые уплотнения (RS, 2RS)** — защита от влаги и грязи, удержание смазки
- **Лабиринтные уплотнения** — для высоких скоростей

**Стопорные элементы:**
- **Канавка + стопорное кольцо (NR)** — фиксация осевого положения
- **Эксцентриковая втулка** — для быстрого монтажа

**Смазка:**
- Предварительная закладка пластичной смазки (в закрытых подшипниках)
- Масло (в открытых подшипниках с системой смазки)

### Материалы и их характеристики

#### Подшипниковая сталь ШХ15

**Состав:**
- Углерод (C): 0.95-1.05%
- Хром (Cr): 1.30-1.65%
- Марганец (Mn): 0.20-0.40%
- Кремний (Si): 0.17-0.37%

**Свойства:**
- Твёрдость после закалки: HRC 60-66
- Высокая износостойкость
- Хорошая контактная прочность
- Ударная вязкость: средняя

**Термообработка:**
- Закалка в масле (820-840°C)
- Отпуск (150-180°C)
- Стабилизация (криогенная обработка для прецизионных)

#### Альтернативные материалы

| Материал | Применение | Особенности |
|----------|------------|-------------|
| **Нержавеющая сталь (AISI 440C)** | Влажная, агрессивная среда | Коррозионная стойкость, но ниже грузоподъёмность |
| **Керамика (Si₃N₄)** | Гибридные подшипники | Низкая плотность, высокие скорости, диэлектрик |
| **Бронза, латунь** | Сепараторы | Хорошая обрабатываемость, антифрикционные свойства |

### Конструктивные варианты

#### Открытые подшипники
- Без защитных элементов
- Требуют внешней системы смазки и уплотнений
- Подходят для высоких скоростей и температур

#### Закрытые подшипники
- С защитными шайбами (Z, 2Z) или уплотнениями (RS, 2RS)
- Предварительно смазаны на весь срок службы
- Не требуют обслуживания

#### Разборные подшипники
- Конические роликовые — кольца и сепаратор с роликами разделяются
- Цилиндрические (NU, N) — внутреннее кольцо снимается отдельно

## Примеры

### Пример 1: Шариковый радиальный однорядный 6205

**Состав:**
- Внутреннее кольцо: сталь ШХ15, d = 25 мм
- Наружное кольцо: сталь ШХ15, D = 52 мм, B = 15 мм
- Тела качения: 9 шариков Ø 7.938 мм, сталь ШХ15
- Сепаратор: штампованный стальной (тип J)

**Масса**: ~0.128 кг

### Пример 2: Роликовый цилиндрический NU 308

**Состав:**
- Внутреннее кольцо: сталь ШХ15, d = 40 мм, с двумя бортами
- Наружное кольцо: сталь ШХ15, D = 90 мм, B = 23 мм, без бортов
- Тела качения: цилиндрические ролики (обычно 12-14 шт.)
- Сепаратор: латунный массивный (тип M) или полиамидный (тип TN9)

**Особенность**: Разборный, допускает осевое перемещение вала.

### Пример 3: Конический роликовый 30205

**Состав:**
- Внутреннее кольцо с сепаратором и роликами (конус)
- Наружное кольцо (чашка)
- Тела качения: конические ролики
- Сепаратор: штампованный стальной

**Особенность**: Полностью разборный, монтируется парами с регулировкой зазора.

### Пример 4: Сферический роликовый 22205

**Состав:**
- Внутреннее кольцо: с двумя дорожками качения
- Наружное кольцо: со сферической дорожкой качения
- Тела качения: два ряда бочкообразных роликов
- Сепаратор: массивный латунный (обычно)

**Особенность**: Самоустанавливающийся, компенсирует перекосы до 2-3°.
