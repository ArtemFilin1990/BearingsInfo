# Система условных обозначений подшипников

## Определение

Система условных обозначений подшипников — унифицированная система кодирования подшипников качения по ГОСТ 3189-89, определяющая конструкцию, размеры и технические характеристики подшипника через последовательность цифр и буквенных знаков.

## Структура условного обозначения

Условное обозначение подшипника состоит из **основного обозначения** (7 цифр) и **дополнительных знаков** (буквы и цифры слева и справа).

### Общая схема обозначения

```
[категория][момент трения][зазор][точность] - [ОСНОВНОЕ] [конструкция][материал][температура][смазка][вибрация]
```

| Позиция | Элемент | Расположение | Пример |
|---------|---------|--------------|--------|
| Слева | Категория (А, В, С) | Перед основным | А, В, 2В |
| Слева | Момент трения | Перед зазором | 1, 2, 3...9 |
| Слева | Группа зазора | Перед классом | 6, 7, 8, 9 или Н |
| Слева | Класс точности | Перед "-" | 0, 6, 5, 4, 2, Т |
| Центр | Основное обозначение | 7 цифр | 7654321 |
| Справа | Дополнительные знаки | После основного | А, К, М, У, Т, С, Ш |

## Основное условное обозначение (7 цифр)

### Для подшипников с d ≥ 10 мм

```
7 6 5 4 3 2 1
│ │ │ │ │ └─┴─ Внутренний диаметр (2 цифры)
│ │ │ │ └───── Серия диаметров (1 цифра)
│ │ │ └─────── Тип подшипника (1 цифра)
│ └─┴───────── Конструктивная разновидность (2 цифры)
└───────────── Серия ширин (1 цифра)
```

**Пример:** 32205
- 3 — конструктивная разновидность
- 2 — тип (роликовый с короткими цилиндрическими роликами)
- 2 — серия диаметров
- 05 — внутренний диаметр 25 мм (05×5=25)

### Для подшипников с d < 10 мм

```
7 6 5 4 3 2 1
│ │ │ │ │ │ └─ Внутренний диаметр (1 цифра, в мм)
│ │ │ │ │ └─── Серия диаметров (1 цифра)
│ │ │ │ └───── Цифра 0 (обязательна!)
│ │ │ └─────── Тип подшипника (1 цифра)
│ └─┴───────── Конструктивная разновидность (2 цифры)
└───────────── Серия ширин (1 цифра)
```

**Пример:** 1000094
- 1 — серия ширин
- 00 — конструктивная разновидность
- 0 — тип (шариковый радиальный)
- 0 — обязательный ноль
- 9 — серия диаметров
- 4 — внутренний диаметр 4 мм

## Дополнительные знаки СЛЕВА (до "-")

### Категория подшипника (А, В, С)

- **А** — подшипники классов точности 5, 4, 2, Т с повышенными требованиями
- **В** — подшипники классов 0, 6Х, 6, 5 с дополнительными требованиями
- **С** — подшипники классов 8, 7, 0, 6 (в обозначении не указывается)

Цифра перед буквой (2В, 3А) — конкретизация дополнительных требований.

### Момент трения (1-9)

Номер ряда момента трения (1, 2, 3...9). Проставляется перед зазором.

**Пример:** 2М5-1000905
- 2 — момент трения по 2-му ряду
- М — нормальный радиальный зазор
- 5 — класс точности

### Радиальный зазор

**Обозначение цифрами:** 1, 2, 6, 7, 8, 9 (нормальная группа не указывается)

**Обозначение буквами:**
- **Н** — ненормализованный радиальный зазор
- **М** — нормальный зазор (при указании момента трения)
- **НТ/НУ** — тугая подборка

**Пример:** 75-313ЕШ2
- 7 — радиальный зазор по 7-й группе
- 5 — класс точности

### Класс точности

**Ряд классов (по возрастанию):** 8, 7, 0, 6Х, 6, 5, 4, Т, 2

- **0** — нормальный класс (при отсутствии спецтребований не указывается)
- **6Х** — для конических роликовых (обозначается как Х)
- **У** — после класса означает повышенную точность по монтажной высоте

## Дополнительные знаки СПРАВА (после основного)

### Последовательность расположения

```
ОУО - [А][Н][материал][К][М][У][Т][С][Ш][особые]
```

1. **А** — повышенная грузоподъемность
2. **Н, Н1...** — конструктивные особенности
3. **Материал:** И, Н, Р, Х, Э, Ю, Я, W
4. **К, К1...** — конструктивные изменения
5. **М, М1...** — модифицированный контакт
6. **У, У1...** — специальные технические требования
7. **Т, Т1...Т6** — температура отпуска
8. **С1...С37** — марка смазки
9. **Ш, Ш1...Ш5** — уровень вибрации
10. Особые отличия

### Основные знаки справа

| Знак | Значение | Применение |
|------|----------|------------|
| **А** | Повышенная грузоподъемность | Шариковые и роликовые |
| **Б, Б1** | Сепаратор из безоловянистой бронзы | Высокоскоростные |
| **Г, Г1** | Сепаратор из черных металлов | Штампованный сепаратор |
| **Д, Д1** | Сепаратор из алюминиевого сплава | Легкие конструкции |
| **Е, Е1-Е5** | Сепаратор из пластмассы | Е-текстолит, Е5-полиамид |
| **Л, Л1** | Сепаратор из латуни | Массивные сепараторы |
| **К, К1-К11** | Конструктивные изменения | Различные модификации |
| **М, М1** | Модифицированный контакт | Роликовые подшипники |
| **Н, Н1** | Конструктивные особенности | Зависит от типа |
| **Р, Р1** | Детали из теплостойкой стали | До 250°C |
| **Т, Т1-Т6** | Температура отпуска | От 150°C до 450°C |
| **У, У1-У3** | Специальные требования | Ужесточенные допуски |
| **Х, Х1** | Кольца из цементируемой стали | Повышенная прочность |
| **Ю, Ю1** | Детали из нержавеющей стали | Коррозионная стойкость |
| **Я, Я1** | Детали из редких материалов | Керамика, стекло |
| **Ш, Ш1-Ш5** | Уровень вибрации | Ш5 — минимальная |
| **С1-С37** | Марка смазки | Для закрытых подшипников |

## Специальные обозначения

### Игольчатые подшипники

Обозначение в виде **dхDхB** (внутренний × наружный × ширина, мм)

**Буквы перед размерами:**
- **К** — без колец
- **ИК** — с сепаратором
- **ИКВ** — без внутреннего кольца
- **КК** — двухрядный без колец
- **АК** — упорный

**Пример:** ИК25×30×10Д — игольчатый с сепаратором из алюминия

### Шарнирные подшипники

**Типы:**
- **Ш** — для подвижных соединений
- **ШМ** — для неподвижных соединений  
- **ШС** — с отверстиями для смазки
- **ШСЛ** — с разъемом наружного кольца

**Пример:** ШС40К — шарнирный с отверстиями, d=40 мм

### Особые диаметры

Диаметры, отделенные знаком дроби:
- **0,6; 1,5; 2,5 мм** — до 10 мм
- **22; 28; 32 мм** — специальные значения
- **≥500 мм** — крупногабаритные

**Пример:** 602/32 — подшипник с d=32 мм

## Примечания

**Важные особенности:**
1. Нормальный класс 0 и нормальный зазор в обозначении **не указываются**
2. Знак категории С **не проставляется**
3. Серия ширин 0 **не указывается**
4. Буквы слева — **строчные**, справа — **прописные**
5. Знак "-" отделяет дополнительные знаки слева от основного обозначения

**Ограничения:**
- Система применима к подшипникам по ГОСТ, для импортных действуют ISO/DIN
- Конкретные значения коэффициентов X, Y для расчета — в каталогах производителей
- Специальные подшипники могут иметь собственную систему обозначений
