# Маркировка подшипников по ГОСТ

## Определение

Маркировка подшипников по ГОСТ 3189-89 — это систематическая система условных обозначений, позволяющая однозначно определить тип, размеры и конструктивные особенности подшипника без обращения к каталогам.

## Назначение

- Быстрая идентификация подшипника по номеру
- Определение основных параметров (d, D, B) без измерения
- Подбор аналогов и замен
- Оформление заказов и коммерческих предложений

## Что это значит на практике

Менеджер или инженер, видя маркировку **6205-2RS C3**, должен понимать:
- Это шариковый радиальный подшипник
- Внутренний диаметр 25 мм
- Имеет два резиновых уплотнения
- Увеличенный радиальный зазор (для температурных деформаций)

## Структура основного обозначения

```
   X  X  X  X  X  X  X
   │  │  │  │  └─┴──┴─ Код внутреннего диаметра (2 цифры)
   │  │  │  └──────── Серия ширины (1 цифра)
   │  │  └─────────── Серия диаметров (1 цифра)
   │  └────────────── Тип подшипника (1 цифра)
   └───────────────── Конструктивная особенность (0-2 цифры, может отсутствовать)
```

### Что это значит на практике

При чтении обозначения справа налево:
1. **Последние 2 цифры** → внутренний диаметр
2. **3-я цифра справа** → серия диаметров (определяет наружный диаметр)
3. **4-я цифра справа** → серия ширины
4. **Остальные цифры слева** → тип и конструкция

## Расшифровка кода внутреннего диаметра

| Код | Диаметр d, мм | Формула расчёта |
|-----|---------------|----------------|
| **00** | 10 | Специальное значение |
| **01** | 12 | Специальное значение |
| **02** | 15 | Специальное значение |
| **03** | 17 | Специальное значение |
| **04** | 20 | d = код × 5 |
| **05** | 25 | d = код × 5 |
| **06** | 30 | d = код × 5 |
| **08** | 40 | d = код × 5 |
| **10** | 50 | d = код × 5 |
| **12** | 60 | d = код × 5 |
| **16** | 80 | d = код × 5 |
| **20** | 100 | d = код × 5 |

**Общее правило**: Для кодов от 04 до 96: **d = код × 5 мм**

### Пример расчёта
- Код **05** → d = 5 × 5 = **25 мм**
- Код **10** → d = 10 × 5 = **50 мм**
- Код **20** → d = 20 × 5 = **100 мм**

### Специальные случаи
Для d < 10 мм или d > 500 мм используется дробная черта:
- **/6** → d = 6 мм
- **/8** → d = 8 мм
- **/500** → d = 500 мм

## Типы подшипников (первая цифра базового обозначения)

| Тип | Название | Тела качения | Нагрузка |
|-----|----------|-------------|----------|
| **0** | Шариковый радиальный однорядный | Шарики | Радиальная |
| **1** | Шариковый самоустанавливающийся двухрядный | Шарики | Радиальная + перекосы |
| **2** | Роликовый цилиндрический | Ролики | Радиальная высокая |
| **3** | Роликовый сферический двухрядный | Ролики | Радиальная + перекосы |
| **4** | Игольчатый | Иглы | Радиальная компактная |
| **6** | Шариковый радиальный однорядный | Шарики | Радиальная + осевая малая |
| **7** | Роликовый конический | Ролики | Радиальная + осевая |
| **8** | Упорный шариковый | Шарики | Осевая |
| **9** | Упорный роликовый | Ролики | Осевая высокая |

### Что это значит на практике

- **Тип 6** — универсальный выбор для большинства применений (электродвигатели, насосы)
- **Тип 7** — для ступиц, редукторов с осевыми нагрузками
- **Тип 2** — когда нужна высокая радиальная грузоподъёмность
- **Тип 1, 3** — когда есть перекосы валов, вибрация

## Серии диаметров (определяют наружный диаметр D)

| Серия | Название | Относительный D | Применение |
|-------|----------|----------------|------------|
| **1** | Особо лёгкая | Малый D | Приборы, легкие нагрузки |
| **2** | Лёгкая | Средний D | **Стандарт** для общего машиностроения |
| **3** | Средняя | Большой D | Редукторы, повышенные нагрузки |
| **4** | Тяжёлая | Очень большой D | Тяжёлое машиностроение |

### Пример для d = 25 мм

| Обозначение | Серия | d, мм | D, мм | B, мм |
|-------------|-------|-------|-------|-------|
| **6105** | 1 | 25 | 42 | 9 |
| **6205** | 2 | 25 | 52 | 15 |
| **6305** | 3 | 25 | 62 | 17 |
| **6405** | 4 | 25 | 80 | 21 |

**Вывод**: При одинаковом d, больше серия → больше D → выше грузоподъёмность

## Серии ширины

| Серия | Размер | Применение |
|-------|--------|------------|
| **8, 9** | Узкая | Ограниченное осевое пространство |
| **0, 1, 2** | Нормальная | **Стандарт** |
| **3, 4, 5** | Широкая | Высокие осевые нагрузки |

### Пример
- **6**0**05** — нормальная ширина
- **6**3**05** — расширенная серия для радиально-упорных
- **6**8**05** — узкая серия

## Примеры полной расшифровки

### Пример 1: **6205**

**Расшифровка**:
- **6** — шариковый радиальный однорядный
- **2** — серия диаметров лёгкая
- **0** — серия ширины нормальная
- **05** — внутренний диаметр 25 мм

**Размеры**: d=25мм, D=52мм, B=15мм

**Применение**: Электродвигатели 2-15 кВт, насосы, вентиляторы

**Аналог ISO**: 6205

### Пример 2: **7205** (или **30205** по ISO)

**Расшифровка**:
- **7** — роликовый конический
- **2** — серия диаметров лёгкая
- **0** — серия ширины нормальная
- **05** — внутренний диаметр 25 мм

**Размеры**: d=25мм, D=52мм, H=16.25мм

**Применение**: Автомобильные ступицы, редукторы с осевыми нагрузками

**Особенность**: Требует установки парами с преднатягом

**Аналог ISO**: 30205

### Пример 3: **2205** (или **NU 205** по ISO)

**Расшифровка**:
- **2** — роликовый с короткими цилиндрическими роликами
- **2** — серия диаметров лёгкая
- **0** — серия ширины нормальная
- **05** — внутренний диаметр 25 мм

**Размеры**: d=25мм, D=52мм, B=15мм

**Применение**: Редукторы с высокими радиальными нагрузками

**Аналог ISO**: NU 205

### Пример 4: **3205** (или **22205** по ISO)

**Расшифровка**:
- **3** — роликовый сферический двухрядный (самоустанавливающийся)
- **2** — серия диаметров лёгкая
- **0** — серия ширины нормальная
- **05** — внутренний диаметр 25 мм

**Размеры**: d=25мм, D=52мм, B=18мм

**Применение**: Вибрационное оборудование, узлы с перекосами валов

**Особенность**: Допускает перекос до 3°

**Аналог ISO**: 22205

### Пример 5: **180205**

**Расшифровка**:
- **1** — шариковый самоустанавливающийся двухрядный
- **8** — серия диаметров особо лёгкая
- **0** — серия ширины нормальная
- **2** — дополнительная серия
- **05** — внутренний диаметр 25 мм

**Применение**: Лёгкие узлы с возможностью перекосов

**Аналог ISO**: 1205

## Суффиксы (дополнительные обозначения)

Суффиксы указываются после основного номера через дефис или пробел.

### Класс точности

| Класс | Точность | Применение |
|-------|----------|------------|
| **0** (без обозначения) | Нормальная | Общее машиностроение |
| **6** | Повышенная | Редукторы, станки |
| **5** | Высокая | Шпиндели, точные механизмы |
| **4** | Прецизионная | Металлорежущие станки |
| **2** | Сверхвысокая | Высокоточные шпиндели |

**Пример**: **6205-5** (класс точности 5)

### Радиальный зазор

| Группа | Зазор | Применение |
|--------|-------|------------|
| **C2** | Меньше нормального | Прецизионные узлы, натяг |
| **(нет)** | Нормальный | **Стандарт** |
| **C3** | Больше нормального | Высокие температуры (+80°C и выше) |
| **C4** | Большой | Очень высокие температуры |
| **C5** | Очень большой | Экстремальные температуры |

**Пример**: **6205 C3**

**Что это значит на практике**: C3 используется в электродвигателях, где посадка с натягом и нагрев уменьшают начальный зазор.

### Защита и уплотнения

| Суффикс | Тип защиты | Особенность | Ограничение скорости |
|---------|-----------|------------|---------------------|
| **Z** | Одна металлическая шайба | Неконтактная | Не снижает |
| **ZZ**, **2Z** | Две металлические шайбы | Неконтактная | Не снижает |
| **RS**, **RSR** | Одно резиновое уплотнение | Контактная | Снижает на 20% |
| **2RS**, **2RSR** | Два резиновых уплотнения | Контактная | Снижает на 30% |

**Пример**: **6205-2RS**

**Что это значит на практике**: 
- **Z/ZZ** — для чистых условий, требуется внешнее смазывание
- **RS/2RS** — для пыли и влаги, встроенная смазка на весь срок

### Материал сепаратора

| Суффикс | Материал | Применение |
|---------|----------|------------|
| **Г** | Штампованная сталь | Стандарт |
| **Л** | Латунь | Высокие нагрузки |
| **Е** | Полиамид | Низкий шум, химстойкость |
| **Д** | Алюминий | Высокие скорости |
| **Р** | Без сепаратора | Максимальная грузоподъёмность |

**Пример**: **6205 Е**

### Специальные обозначения

| Суффикс | Значение | Применение |
|---------|----------|------------|
| **N** | Канавка на наружном кольце | Фиксация осевого положения |
| **NR** | Канавка + стопорное кольцо | Фиксация в корпусе |
| **К** | Конический диаметр отверстия (1:12) | Посадка на конический вал |
| **Н** | Отверстие в наружном кольце | Для смазки |
| **Ю** | Повышенная температура | До +200°C |
| **Т** | Пониженная температура | До -60°C |

**Пример**: **6205-NR** (с канавкой и стопорным кольцом)

## Примеры полных обозначений с суффиксами

### **6205-2RS C3**
- Шариковый радиальный, d=25мм
- Два резиновых уплотнения
- Увеличенный зазор
- **Применение**: Электродвигатель в пыльной среде

### **7208 AC/P4**
- Роликовый конический, d=40мм
- Угол контакта (AC = 25°)
- Класс точности P4 (прецизионный)
- **Применение**: Шпиндель токарного станка

### **NU 306 E**
- Роликовый цилиндрический, d=30мм
- Пластиковый сепаратор (низкий шум)
- **Применение**: Промышленный редуктор

### **22205 CC/W33**
- Сферический роликовый, d=25мм
- Оптимальный зазор (CC)
- Смазочные канавки (W33)
- **Применение**: Вибрационное оборудование

## Типовые ошибки при чтении маркировки

### Ошибка 1: Путаница между типами 6 и 7

**Неправильно**: Считать, что **6205** и **7205** взаимозаменяемы
- **6205** — радиальный шариковый (осевая нагрузка минимальна)
- **7205** — конический роликовый (требует осевой нагрузки и преднатяга)

**Последствие**: Конический подшипник без осевой нагрузки быстро разрушится

### Ошибка 2: Игнорирование суффиксов

**Неправильно**: Заменить **6205 C3** на **6205** без анализа
- C3 критичен для компенсации теплового расширения

**Последствие**: Заклинивание подшипника при нагреве

### Ошибка 3: Неправильный расчёт внутреннего диаметра

**Неправильно**: Код **00** = 0 мм
**Правильно**: Код **00** = 10 мм (специальное значение)

### Ошибка 4: Путаница между 2RS и ZZ

**Неправильно**: Использовать **6205-2RS** при высоких оборотах (>10000 об/мин)
**Правильно**: Использовать **6205-ZZ** или открытый **6205**

**Последствие**: Перегрев уплотнений, разрушение смазки

## Сравнение с маркировкой ISO

| Элемент | ГОСТ | ISO | Примечание |
|---------|------|-----|------------|
| Базовый номер | 6205 | 6205 | Совпадает для типа 6 |
| Конический | 7205 | 30205 | Разные обозначения |
| Сферический | 3205 | 22205 | Разные обозначения |
| Цилиндрический | 2205 | NU 205 | ISO добавляет префикс |
| Уплотнения | 2RS | 2RS1, 2RSH | Могут различаться |
| Зазор | C3 | C3 | Совпадает |
| Точность | 5 | P5 | ISO добавляет P |

**Важно**: При замене ГОСТ на ISO всегда проверяйте полное соответствие, особенно суффиксы!

## Практический пример: Декодирование неизвестного подшипника

**Задача**: На подшипнике написано **6308-2RS C3**

**Пошаговая расшифровка**:
1. **08** → d = 8 × 5 = 40 мм
2. **3** (серия диаметров) → средняя серия
3. **0** (серия ширины) → нормальная
4. **6** (тип) → шариковый радиальный однорядный
5. **2RS** → два резиновых уплотнения
6. **C3** → увеличенный радиальный зазор

**Результат**: 
- d=40мм, D=90мм, B=23мм
- Закрытый подшипник с встроенной смазкой
- Подходит для температур до +110°C
- Применение: Электродвигатели 15-30 кВт в пыльных условиях
