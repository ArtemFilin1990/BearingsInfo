## Источники

- ГОСТ 3189-89 "Подшипники качения шариковые и роликовые. Система условных обозначений"
- Каталоги производителей: SKF, FAG/INA (Schaeffler Group), MPZ, NKE, ISB

## О проекте

База знаний содержит только проверенную информацию из официальных стандартов и каталогов производителей подшипников. Все данные о размерах, весе, аналогах и применении соответствуют реальным техническим характеристикам.
