## Основные этапы

### 1. Заготовительное производство
- Штамповка колец из прутка/трубы
- Обточка по наружному/внутреннему диаметру

### 2. Термическая обработка
- Закалка: нагрев до 820-860°C
- Отпуск: снятие напряжений
- Твердость: HRC 58-64

### 3. Шлифование
- Финишная обработка дорожек качения
- Шероховатость: Ra 0.1-0.32 мкм
- Прецизионные подшипники: суперфиниш

### 4. Изготовление тел качения

**Шарики**:
- Штамповка → термообработка → шлифование → полировка
- Класс точности по размеру: G5-G40

**Ролики**:
- Токарная обработка → термообработка → шлифование

### 5. Сборка
- Комплектация по размерам
- Установка сепаратора
- Контроль зазора

### 6. Контроль качества
- Геометрия (допуски)
- Вибрация
- Шум
- Момент трения

### 7. Консервация и упаковка
- Промывка
- Нанесение консервационной смазки
- Упаковка (индивидуальная/групповая)

## Оборудование

- Токарные автоматы
- Шлифовальные станки
- Термические печи
- Измерительные машины
