# podshipnik.info catalog (offline snapshot)

## Purpose
Сохранять офлайн-выгрузки каталога ГОСТ-подшипников podshipnik.info без внешних ссылок.

## Classification
- Независимый справочник ГОСТ типоразмеров.
- Используется для валидации d/D/B и массы.

## Designation
- `SNAP_SRC_PODSHIPNIK_INFO` — статичная выгрузка (HTML/CSV) каталога.

## Prefixes/Suffixes
- Префикс `SNAP_SRC_` — офлайн-слепок.
- Суффикс `_INFO` — справочный портал.

## Examples
- Сохранить таблицу по серии 6200 (поля: обозначение, d, D, B, масса) в этом файле или отдельном CSV.

## Notes & Limitations
- Доступ из окружения блокирован (403), требуется офлайн-выгрузка.
- При обновлении фиксировать дату и источник выгрузки.

## Standard References
- Домен источника: podshipnik.info (каталог ГОСТ-подшипников).
- Локальные файлы: этот `.md` + возможные CSV/HTML выгрузки в той же директории.
