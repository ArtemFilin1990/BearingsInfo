# Радиальные и осевые зазоры ISO

## Определение
Зазор ISO описывает величину свободного перемещения колец подшипника относительно друг друга в радиальном или осевом направлении и фиксируется буквенными суффиксами в обозначении.

## Назначение
- Выбор подходящего зазора под температуру, посадки и уровень вибраций.
- Предотвращение перегрева или люфта при изменении условий эксплуатации.
- Корректная интерпретация суффиксов C-групп без обращения к ГОСТ.

## Конструкция / особенности
- Радиальные зазоры кодируются группами C1–C5; нормальный зазор обычно не указывается (CN/C0).
- Дополнительные группы (C6, C7, CM) применяются для специальных условий, в частности для электродвигателей.
- Для упорных подшипников осевой зазор задаётся буквами CA/CB/CC.

## Обозначения и расшифровка

### Радиальный зазор
| Суффикс | Величина относительно нормального | Типовые случаи применения |
|---------|-----------------------------------|---------------------------|
| C1 | Наименьший | Преднатяг, высокая точность позиционирования |
| C2 | Меньше нормального | Точные механизмы, небольшой преднатяг |
| CN / C0 | Нормальный | Стандартные условия |
| C3 | Больше нормального | Посадки с натягом, температура выше ~70 °C, несоосность |
| C4 | Значительно больше нормального | Высокие температуры и тяжёлые натяги |
| C5 | Максимальный из стандартных | Экстремальные тепловые и вибрационные условия |
| C6 | Между C2 и CN | Специальная настройка |
| C7 | Между C3 и C4 | Специальная настройка |
| CM | Специальный зазор для электродвигателей | Контроль шума и вибраций |

### Осевой зазор для упорных подшипников
| Суффикс | Описание |
|---------|----------|
| CA | Осевой зазор группы A |
| CB | Осевой зазор группы B |
| CC | Осевой зазор группы C |

## Примеры
- **6205-C3** — увеличенный радиальный зазор для посадки с натягом на валу.
- **NU305E-C4** — цилиндрический роликовый подшипник с большим радиальным зазором для повышенных температур.
- **6308-2RS-C2** — уменьшенный зазор C2 с двухсторонними уплотнениями для точной фиксации.
- **22205-CA** — сферический роликовый подшипник с осевым зазором группы A.

## Примечания и ограничения
- При выборе зазора учитывайте фактическую посадку вала и корпуса: натяг уменьшает зазор, а температура увеличивает его.
- Для высоких оборотов и повышенных температур рекомендуется проверять допустимую частоту вращения для выбранной группы зазора.
- Неверный зазор приводит к перегреву или вибрации; всегда сверяйте суффикс C-группы при подборе аналогов.
