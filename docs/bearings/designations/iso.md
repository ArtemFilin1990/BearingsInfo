# Обозначения по ISO (международная система)

Международная система обозначений подшипников основана на стандартах ISO и используется большинством мировых производителей: SKF, FAG/INA, NSK, NTN, TIMKEN, NKE, ISB и др.

## Структура обозначения

Обозначение подшипника по ISO состоит из нескольких частей:

```
[Префикс] [Основное обозначение] [Суффикс]
```

- **Префикс**: Конструктивные особенности (опционально)
- **Основное обозначение**: Тип, серия, размер
- **Суффикс**: Уплотнения, зазоры, материалы, точность

## Основное обозначение

### Шариковые радиальные однорядные

Структура: **6 [серия] [размер]**

```
6  2  05
│  │  └─ Код внутреннего диаметра
│  └──── Серия диаметров
└─────── Тип подшипника (6 = радиальный шариковый)
```

#### Серии диаметров
| Серия | Обозначение | D относительно d | Применение |
|-------|-------------|------------------|------------|
| Особо легкая | 18, 19 | Малый | Приборы |
| Легкая | 02, 62, 63 | Средний | Стандартное применение |
| Средняя | 03 | Большой | Тяжелые нагрузки |
| Тяжелая | 04 | Очень большой | Специальное применение |

#### Примеры
- **6205**: Радиальный шариковый, легкая серия, d=25 мм
- **6305**: Радиальный шариковый, средняя серия, d=25 мм
- **6005**: Радиальный шариковый, особо легкая серия, d=25 мм

### Роликовые цилиндрические

Используют буквенные префиксы для обозначения конструкции:

| Обозначение | Конструкция | Описание |
|-------------|-------------|----------|
| **N** | Бурты на внутреннем кольце | Фиксация НК по оси |
| **NU** | Бурты на наружном кольце | Фиксация ВК по оси |
| **NJ** | Бурты на НК + бортик на ВК | Осевая фиксация в одну сторону |
| **NUP** | NU + свободное кольцо | Осевая фиксация в обе стороны |
| **NH** | Втулка вместо внутреннего кольца | Для валов без бурта |
| **NF** | Два бурта на НК + два на ВК | Полная осевая фиксация |

#### Примеры
- **NU 205**: Роликовый цилиндрический, бурты на НК, d=25 мм
- **NJ 308**: Роликовый цилиндрический, бурты на НК + бортик на ВК, d=40 мм
- **NUP 2205**: Роликовый цилиндрический с дополнительным кольцом

### Роликовые конические

Обозначение: **3 [серия] [размер]**

```
30  2  05
│   │  └─ Код внутреннего диаметра
│   └──── Серия диаметров
└──────── Тип (конический роликовый)
```

#### Серии
- **302**: Легкая серия, угол контакта ~12-15°
- **303**: Средняя серия, угол контакта ~15-18°
- **320, 322, 323**: Различные серии и углы контакта
- **329, 330**: Тяжелые серии

#### Примеры
- **30205**: Конический роликовый, легкая серия, d=25 мм
- **30305**: Конический роликовый, средняя серия, d=25 мм
- **32005**: Конический роликовый, серия 320, d=25 мм

### Двухрядные сферические роликовые

Обозначение: **2 [серия] [размер]**

```
22  2  05
│   │  └─ Код внутреннего диаметра
│   └──── Серия диаметров
└──────── Тип (сферический роликовый)
```

#### Серии
- **213**: Легкая серия
- **222**: Средняя серия
- **223**: Тяжелая серия
- **230, 231, 232**: Различные модификации

#### Примеры
- **22205**: Сферический роликовый, серия 222, d=25 мм
- **23120**: Сферический роликовый, серия 231, d=100 мм

### Радиально-упорные шариковые

Обозначение зависит от угла контакта:

| Угол контакта | Обозначение | Применение |
|---------------|-------------|------------|
| 15° | 72xx, 73xx | Низкие осевые нагрузки |
| 25° | 72xxB, 73xxB | Средние нагрузки |
| 40° | 72xxC, 73xxC | Высокие осевые нагрузки |

#### Примеры
- **7205**: Радиально-упорный, угол 15°, d=25 мм
- **7305B**: Радиально-упорный, угол 25°, d=25 мм
- **7205C**: Радиально-упорный, угол 40°, d=25 мм

### Упорные шариковые

Обозначение: **511xx, 512xx, 513xx**

- **511**: Одинарные
- **512**: Двойные односторонние
- **513**: Двойные двухсторонние

## Код внутреннего диаметра

Аналогично ГОСТ:

| Код | Диаметр d, мм | Правило |
|-----|---------------|---------|
| 00 | 10 | Специальное |
| 01 | 12 | Специальное |
| 02 | 15 | Специальное |
| 03 | 17 | Специальное |
| 04-96 | 20-480 | d = код × 5 |

Для диаметров < 10 мм используется дробь (например, /8).

## Суффиксы (дополнительные обозначения)

### Уплотнения и защита

| Суффикс | Значение | Производитель |
|---------|----------|---------------|
| **Z** | Одна защитная шайба | Общее |
| **ZZ, 2Z** | Две защитные шайбы | Общее |
| **RS, RSR** | Одно контактное уплотнение | SKF, FAG |
| **2RS, 2RSR** | Два контактных уплотнения | SKF, FAG |
| **DDU** | Два контактных уплотнения | NSK |
| **LLU** | Два неконтактных уплотнения | NSK |
| **VV** | Два контактных уплотнения | NTN |

### Радиальный зазор

| Суффикс | Зазор | Применение |
|---------|-------|------------|
| **C2** | Меньше нормального | Преднатяг, высокая точность |
| **(норм.)** | Нормальный | Стандартное применение |
| **C3** | Больше нормального | Высокие температуры, посадки с натягом |
| **C4** | Большой | Очень высокие температуры |
| **C5** | Очень большой | Экстремальные условия |

### Класс точности (ISO 492)

| Класс | Точность | Применение |
|-------|----------|------------|
| **P0** (без обозначения) | Нормальная | Общее машиностроение |
| **P6** | Повышенная | Станки, электродвигатели |
| **P5** | Высокая | Шпиндели станков |
| **P4** | Прецизионная | Высокоточные шпиндели |
| **P2** | Сверхпрецизионная | Измерительные приборы |

### Материал сепаратора

| Суффикс | Материал | Особенности |
|---------|----------|-------------|
| **J** | Стальной штампованный | Стандартный |
| **M** | Латунный массивный | Высокие нагрузки |
| **Y** | Латунный | Средние нагрузки |
| **P** | Полиамид (пластик) | Низкий шум, коррозионная стойкость |
| **F** | Полиамид армированный | Высокие скорости |
| **TN** | Полиамид (SKF) | Низкий шум |

### Специальные модификации

| Суффикс | Значение |
|---------|----------|
| **K** | Конический диаметр отверстия (1:12) |
| **K30** | Конический диаметр отверстия (1:30) |
| **N** | Канавка на наружном кольце |
| **NR** | Канавка со стопорным кольцом |
| **H** | Адаптерная втулка |
| **X** | Улучшенная конструкция |

### Температурные диапазоны

| Суффикс | Температура | Производитель |
|---------|-------------|---------------|
| **HT** | Высокая температура (до +300°C) | SKF |
| **S0** | Нержавеющая сталь | SKF |
| **W64** | Высокая температура | FAG |
| **S1** | Стабилизация до +200°C | FAG |

### Смазка

| Суффикс | Тип смазки | Производитель |
|---------|------------|---------------|
| **L** | Консистентная смазка | Общее |
| **LHT** | Высокотемпературная смазка | SKF |
| **MT47** | Пищевая смазка | SKF |
| **GJN** | Смазка общего назначения | NSK |

## Примеры полных обозначений ISO

### 6205-2RS C3
- **6205**: Радиальный шариковый, легкая серия, d=25 мм
- **2RS**: Два контактных уплотнения
- **C3**: Увеличенный радиальный зазор
- **Применение**: Электродвигатели, пыльные условия

### NU 305 E C3
- **NU**: Роликовый цилиндрический (бурты на НК)
- **305**: Средняя серия, d=25 мм
- **E**: Пластиковый сепаратор
- **C3**: Увеличенный радиальный зазор
- **Применение**: Высокие радиальные нагрузки

### 30205 J2 Q
- **30205**: Конический роликовый, легкая серия, d=25 мм
- **J2**: Стальной штампованный сепаратор
- **Q**: Улучшенная внутренняя геометрия
- **Применение**: Автомобильные ступицы

### 6308 ZZ C4
- **6308**: Радиальный шариковый, d=40 мм
- **ZZ**: Две защитные шайбы
- **C4**: Большой радиальный зазор
- **Применение**: Высокие температуры

### 7205 BECBP
- **7205**: Радиально-упорный, d=25 мм
- **B**: Угол контакта 40°
- **E**: Оптимизированная внутренняя конструкция
- **C**: Полиамидный сепаратор
- **BP**: Высокая точность (SKF)
- **Применение**: Шпиндели станков

## Особенности производителей

### SKF (Швеция)
- Широкое использование суффиксов для спецификации
- Серия Explorer: улучшенные характеристики (суффикс E)
- Особые обозначения для пищевой промышленности

### FAG/INA (Schaeffler Group, Германия)
- Серии B, C, E для различных модификаций
- Суффикс P для полимерного сепаратора
- Специальные высокотемпературные версии (W64)

### NSK (Япония)
- DDU для двойных уплотнений
- LLU для неконтактных уплотнений
- Серия HR: высокая жесткость

### NTN (Япония)
- VV для контактных уплотнений
- LLB для неконтактных уплотнений
- Серия UCF для подшипниковых узлов

### NKE (Австрия)
- Стандартные обозначения ISO
- Специальные серии для железнодорожного транспорта

### ISB (Италия)
- Стандартные обозначения ISO
- Нержавеющие серии для пищевой промышленности (SS)

## Сравнение с ГОСТ

| Параметр | ISO | ГОСТ |
|----------|-----|------|
| Радиальный шариковый | 6205 | 6205 (совпадает) |
| Роликовый цилиндрический | NU 205 | 2205 |
| Конический роликовый | 30205 | 7205 |
| Сферический роликовый | 22205 | 3205 |
| Уплотнения | 2RS | 2RS |
| Зазор | C3 | C3 (совпадает) |
