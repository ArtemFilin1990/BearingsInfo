# Данные по ISO

Структурированные данные о подшипниках по стандартам ISO.

## Файлы

### bearings.csv
Основной каталог подшипников по ISO

### dimensions.csv
Таблицы размеров по ISO

### prefixes.csv
Префиксы обозначений ISO

### suffixes.csv
Суффиксы обозначений ISO

## Формат

- Кодировка: UTF-8
- Разделитель: запятая
- Заголовки: в первой строке

*Источник: Baza/data/iso/*
