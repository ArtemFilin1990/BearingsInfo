# 5.4 Соответствие размеров ГОСТ и импортных ремней

## Классические ремни

| ГОСТ | Импорт (ISO) | Сечение |
|------|--------------|---------|
| О | Z/10 | 10×6 мм |
| А | A/13 | 13×8 мм |
| Б | B/17 | 17×11 мм |
| В | C/22 | 22×14 мм |
| Г | D/32 | 32×19 мм |
| Д | E/38 | 38×23 мм |

## Узкие ремни
- **SPZ** = СПЗ
- **SPA** = СПА
- **SPB** = СПБ
- **SPC** = СПЦ

## Длина
Импортные ремни маркируются по наружной длине (Lа), отечественные — по внутренней (Li).

**Преобразование**: Lа = Li + k×h
где k — коэффициент (≈3.5-4), h — высота сечения
