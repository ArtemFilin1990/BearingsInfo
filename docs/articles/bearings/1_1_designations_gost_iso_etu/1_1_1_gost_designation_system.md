# 1.1.1 Система условных обозначений подшипников (ГОСТ)

## Структура обозначения ГОСТ

Обозначение подшипника состоит из основного условного обозначения и дополнительных знаков (слева и справа).

### Основное обозначение (7 цифр)

```
X XX YY ZZ
│ │  │  │
│ │  │  └─ Внутренний диаметр (последние 2 цифры)
│ │  └──── Серия диаметров (2 цифры)
│ └─────── Тип подшипника (2 цифры)
└───────── Конструктивное исполнение
```

## Типы подшипников (первая/вторые цифры)

| Код | Тип подшипника |
|-----|----------------|
| 0 | Двухрядный радиально-упорный шариковый |
| 1 | Шариковый сферический двухрядный |
| 2 | Роликовый с короткими цилиндрическими роликами |
| 3 | Роликовый сферический двухрядный |
| 4 | Роликовый игольчатый |
| 5 | Роликовый с витыми роликами |
| 6 | Шариковый радиальный однорядный |
| 7 | Роликовый конический |
| 8 | Упорный шариковый |
| 9 | Упорный роликовый |

## Серии диаметров

- **Легкая**: 1, 9
- **Сверхлегкая**: 7, 8
- **Средняя**: 2, 0
- **Тяжелая**: 3, 4

## Обозначение внутреннего диаметра

- **d = 10-17 мм**: последние 2 цифры = 00, 01, 02, 03
- **d = 20-495 мм**: делим на 5 (например, 05 = 25 мм)
- **d < 10 мм**: прямое обозначение

## Примеры

**6205**: 
- 6 = радиальный шариковый
- 2 = средняя серия
- 05 = d=25 мм

**7305**:
- 7 = конический роликовый  
- 3 = тяжелая серия
- 05 = d=25 мм
