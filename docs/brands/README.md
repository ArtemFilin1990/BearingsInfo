# Классификация брендов

## Данные
- `data/brands/brands.csv` — сегмент (бюджет/средний/премиум), страна, ключевые категории и примечания.
- Источники: `Бренды/бюджет.md`, `Бренды/средний_класс.md`, `Бренды/премиум.md` и каталоги в `sources/brands`/`sources/skf`.

## Сегменты
- **Бюджет**: NKE, ISB — стандартные ISO-исполнения, акцент на унификацию и нержавеющие серии.
- **Средний**: NTN, KOYO, SNR — фирменные уплотнения (VV/LLB/2RS/SR), температурные серии, комплектность DB/DF/DT.
- **Премиум**: SKF, FAG/INA, NSK — прецизионные классы P4/P5/P2, смазки MT/VT/W64, специальные серии VA/Explorer.

## Использование
- Для выбора бренда сопоставляйте требования к точности (P-класс), зазору (C-группа) и температуре (S1–S4/HT).
- В `categories` перечислены отличительные признаков (например, W33, VA**, Explorer); сверяйте с каталогом производителя в `sources/brands` или `sources/skf`.
- Источник данных фиксирован в колонке `source`; обновления выполняются через `scripts/update_repo.py`.
