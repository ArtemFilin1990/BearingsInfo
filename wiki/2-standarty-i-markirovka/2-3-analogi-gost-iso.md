# Аналоги подшипников ГОСТ – ISO

## Определение

Таблица соответствия подшипников по системе ГОСТ → ISO — справочная таблица для поиска международного эквивалента отечественного подшипника. Для каждого обозначения ГОСТ указан эквивалент ISO и тип конструкции подшипника.

## Назначение таблицы

Таблица используется для:
- Поиска импортного аналога российского подшипника
- Проверки взаимозаменяемости при закупке
- Сопоставления технической документации ГОСТ и ISO
- Составления заявок на поставку

## Принципы соответствия

**Взаимозаменяемость по габаритам:**
- Подшипники ГОСТ и ISO с одинаковыми d, D, B/T физически взаимозаменяемы
- Обозначения различаются, но размеры стандартизованы

**Учёт дополнительных требований:**
- Классы точности могут различаться
- Зазоры требуют проверки
- Материалы сепараторов могут отличаться

## Примечание

В номере и аналоге подшипника два правых знака 00 меняются на реальные значения цифр (обозначение диаметра подшипника).

**Пример:** ГОСТ 200 = ISO 6200
- Подшипник 205 (ГОСТ) = 6205 (ISO)
- Подшипник 210 (ГОСТ) = 6210 (ISO)

## Ограничения таблицы

**Таблица не охватывает:**
- Специальные конструкции подшипников
- Подшипники с нестандартными размерами
- Модифицированные исполнения

**При замене учитывать:**
- Дополнительные знаки обозначения (класс точности, зазоры)
- Материалы сепараторов (могут различаться)
- Смазку и уплотнения
- Условия эксплуатации
