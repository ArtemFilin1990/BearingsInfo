# Inbox - Входящие файлы

Директория для входящих файлов, ожидающих обработки и импорта.

## Назначение

Файлы помещаются сюда для автоматической обработки через пайплайн импорта.

## Поддерживаемые форматы

- ZIP, RAR, 7Z архивы
- CSV файлы
- XLSX файлы
- PDF каталоги (для извлечения данных)

## Обработка

Запустите скрипт обработки:
```bash
python ../api/scripts/unpack_and_classify.py
```

Скрипт:
1. Распакует архивы
2. Классифицирует файлы по типу
3. Переместит обработанные файлы
4. Подготовит данные для импорта

## Статус файлов

- `pending/` - ожидают обработки
- `processing/` - в процессе обработки
- `processed/` - обработаны
- `errors/` - ошибки обработки

*Источник: bearing-api/data/inbox/*
