# Номенклатура подшипников по производителям

В этой директории содержатся файлы с номенклатурой подшипников различных производителей и заводов.

## Структура

Файлы организованы по производителям и заводам:

- **Российские производители**: ГПЗ-1, ГПЗ-2, ГПЗ-3, МПЗ, ЕПК и др.
- **Международные производители**: SKF, FAG, NSK, NTN, KOYO, TIMKEN и др.

## Использование

Для поиска подшипников конкретного производителя откройте соответствующий файл.

Основные производители:
- [SKF](./SKF.md) - шведский производитель
- [FAG](./FAG.md) - немецкий производитель (Schaeffler Group)
- [NSK](./NSK.md) - японский производитель
- [ГПЗ-1](./ГПЗ-1.md) - Первый государственный подшипниковый завод (Москва)
- [ГПЗ-3](./ГПЗ-3.md) - Волжский подшипниковый завод (Самара)
