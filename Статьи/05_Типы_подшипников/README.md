# Типы подшипников

Раздел содержит классификацию и описание различных типов подшипников качения.

## Содержание раздела

Материалы о:
- Классификации подшипников по конструкции
- Типах подшипников и их применении
- Специальных типах подшипников
- Характеристиках различных конструкций

## Основные материалы

- **Классификация подшипников** - систематизированное описание всех типов подшипников
- **Типы подшипников** - детальное описание конструктивных особенностей
- **Сферические роликовые подшипники** - особенности конструкции и применения
- **Гибридные подшипники** - подшипники с керамическими телами качения
- **Миниатюрные подшипники** - малогабаритные подшипники
- **Шариковая опора** - конструкция и применение шаровых опор
- **Роликовые линейные опоры качения** - линейные системы перемещения
- **Шарнирные головки** - шарнирные наконечники и их аналоги
- **Обозначение и размеры подшипников SKF Y-типа** - корпусные подшипники

## Типы по конструкции

- Шариковые радиальные
- Роликовые цилиндрические
- Роликовые конические
- Роликовые сферические
- Игольчатые
- Упорные
- Радиально-упорные
- Шарнирные

## Применение

Каждый тип подшипника оптимален для определенных условий работы и нагрузок.
