# Каталоги производителей

Эта директория содержит каталоги производителей подшипников в формате Markdown.

## Содержание

Каталоги ведущих мировых и региональных производителей подшипников, включая:

### Мировые лидеры
- SKF (Швеция)
- FAG/INA (Германия)
- NSK, NTN, KOYO, NACHI (Япония)
- TIMKEN, RBC (США)

### Европейские производители
- NKE (Австрия)
- SNFA (Франция)
- SNR (Франция)
- ZVL (Словакия)
- KINEX (Словакия)
- FKL (Сербия)
- FERSA (Испания)

### Российские и СНГ
- Московский подшипниковый завод (МПЗ)
- Самарский подшипник
- Вологодский подшипниковый завод (ВПЗ)
- ЕПК (Единая подшипниковая компания)
- APB (для станкостроения)

### Азиатские производители
- FBJ (Япония)
- IKO (Япония)
- ASAHI (Япония)
- KYK (Южная Корея)

### Специализированные производители
- Myonic - сверхпрецизионные подшипники
- Kaydon - тонкостенные подшипники
- GAMET BEARINGS - специальные подшипники
- BECO - высокотемпературные подшипники
- FLURO - шарнирные подшипники

### Линейные системы и комплектующие
- ROLLON - линейные направляющие
- HIWIN - шарико-винтовые пары
- THK, ABBA - системы линейного перемещения

### Приводная техника
- SATI - приводная техника
- Contitech, Optibelt, RUBENA - приводные ремни
- MEGADYNE - ремни и цепи
- MARKES - конвейерные ролики

### Другие компоненты
- TTO - сальники
- Sandvik Coromant - твердосплавный инструмент

## Формат каталогов

Каталоги представлены в формате Markdown и содержат:
- Информацию о производителе
- Типы выпускаемой продукции
- Системы обозначений
- Размерные ряды
- Технические характеристики

## Использование

Каталоги можно использовать для:
- Подбора подшипников конкретного производителя
- Изучения ассортимента
- Понимания систем обозначений
- Поиска аналогов
