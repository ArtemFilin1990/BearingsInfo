- `—` — не применимо

**Комментарий:**
- Указать конкретные различия (допуски, материалы, методы испытаний)

### 7) Методика / расчёт / правила применения (если применимо)

#### Алгоритм действий

**Для подбора подшипника качения:**
1. Определить тип нагрузки (радиальная, осевая, комбинированная)
2. Рассчитать эквивалентную нагрузку по формуле
3. Определить требуемый ресурс (в часах или оборотах)
4. Рассчитать требуемую динамическую грузоподъёмность
5. Выбрать подшипник из каталога с C ≥ C_требуемая
6. Проверить по статической грузоподъёмности
7. Проверить по предельной частоте вращения

#### Формулы

**Расчёт эквивалентной динамической нагрузки:**

```
P = X·Fr + Y·Fa

где:
P  — эквивалентная динамическая нагрузка, Н
Fr — радиальная нагрузка, Н
Fa — осевая нагрузка, Н
X  — коэффициент радиальной нагрузки (из таблиц стандарта)
Y  — коэффициент осевой нагрузки (из таблиц стандарта)
```

**Расчёт ресурса (по ГОСТ 18855-2013):**

```
L₁₀ = (C / P)^p · 10⁶

где:
L₁₀ — номинальный ресурс, оборотов (с вероятностью безотказной работы 90%)
C   — динамическая грузоподъёмность, Н
P   — эквивалентная нагрузка, Н
p   — показатель степени (3 для шариковых, 10/3 для роликовых)
```

**Перевод ресурса в часы:**

```
Lh = L₁₀ / (60 · n)

где:
Lh — ресурс в часах
n  — частота вращения, об/мин
```

#### Условия применимости

- Формулы применимы для стандартных условий эксплуатации
- Температура: -40...+120 °C
- Нормальная смазка и чистота
- Отсутствие вибраций и ударов
- При отклонении от стандартных условий применяются поправочные коэффициенты

#### Типовые ошибки

1. **Использование номинальной грузоподъёмности вместо динамической** при расчёте ресурса
2. **Игнорирование осевой нагрузки** при выборе радиального подшипника
3. **Неучёт коэффициента безопасности** для ответственных узлов
4. **Превышение предельной частоты вращения** для выбранного типа смазки
5. **Неправильный выбор посадок** валов и отверстий корпуса

### 8) Практические примеры

#### Пример 1: Выбор подшипника для электродвигателя

**Условия:**
- Мощность электродвигателя: 11 кВт
- Частота вращения: 1500 об/мин
- Радиальная нагрузка: Fr = 2800 Н
- Осевая нагрузка: Fa = 400 Н
- Требуемый ресурс: 20 000 ч

**Применение:**
1. Тип подшипника: радиальный шариковый однорядный
2. Расчёт эквивалентной нагрузки: P = 1·2800 + 0,5·400 = 3000 Н
3. Требуемый ресурс в оборотах: L₁₀ = 20000·60·1500 = 1,8·10⁹ об
4. Требуемая грузоподъёмность: C = P·(L₁₀/10⁶)^(1/3) = 3000·(1800)^(1/3) = 36 600 Н

**Эффект / результат:**
- Выбран подшипник 6312 (C = 81 900 Н > 36 600 Н)
- Запас по грузоподъёмности: 2,24
- Ожидаемый ресурс: >20 000 ч

#### Пример 2: Проверка соответствия ГОСТ ⇄ ISO

**Условия:**
- Требуется подшипник 180 мм (внутренний диаметр)
- Проектирование по ГОСТ, поставка по ISO

**Применение:**
1. По ГОСТ 8338-2014: серия 300, обозначение 6036
2. По ISO 15:2017: обозначение 6036 (идентично)
3. Проверка размеров: d=180 мм, D=280 мм, B=46 мм (совпадают)
4. Проверка допусков: класс точности Normal (6) — эквивалентен

**Эффект / результат:**
- Полная взаимозаменяемость подтверждена
- Можно использовать любого производителя (ГОСТ или ISO)
- Различия только в маркировке упаковки

#### Пример 3: Выявление несоответствия стандартов

**Условия:**
- Требуется конический роликоподшипник
- Сравнение требований ГОСТ и ISO к смазке

**Применение:**
1. ГОСТ 27365-2017: требует консистентную смазку типа Литол-24
2. ISO 355:2007: указывает на консистентную смазку без конкретного типа
3. Температурный диапазон по ГОСТ: -40...+120 °C
4. Температурный диапазон по ISO: -30...+110 °C (по умолчанию)

**Эффект / результат:**
- Выявлено частичное несоответствие по требованиям к смазке
- Различия в температурных диапазонах: ГОСТ более широкий
- Рекомендация: использовать требования ГОСТ для российских условий

### 9) Связанные термины

**Термины, необходимые для корректного понимания темы:**

- **Дорожка качения** — профилированная поверхность на кольцах подшипника, по которой катятся тела качения
- **Тело качения** — шарик или ролик, передающий нагрузку между кольцами
- **Сепаратор** — деталь, разделяющая и направляющая тела качения
- **Грузоподъёмность динамическая (C)** — постоянная нагрузка, при которой ресурс подшипника составляет 10⁶ оборотов
- **Грузоподъёмность статическая (C₀)** — нагрузка, вызывающая контактные напряжения определённой величины
- **Класс точности** — совокупность допусков на размеры и форму деталей подшипника
- **Радиальный зазор** — расстояние, на которое одно кольцо может перемещаться относительно другого в радиальном направлении
- **Посадка** — характер соединения вала с внутренним кольцом и корпуса с наружным кольцом
- **Эквивалентная нагрузка** — постоянная радиальная нагрузка, под действием которой подшипник имеет такой же ресурс, как и при реальных условиях нагружения
- **Серия подшипника** — группа подшипников с одинаковым углом контакта и отношением размеров

**Связанные стандарты:**
- ГОСТ 18855-2013 — Расчёт ресурса
- ГОСТ 520-2011 — Общие технические условия
- ISO 281:2007 — Расчёт динамической грузоподъёмности и ресурса
- ISO 76:2006 — Расчёт статической грузоподъёмности

```
- Копии стандартов ГОСТ и ISO
- Таблицы данных
- Справочные материалы
```

#### Стандарты (упоминаемые, но не обязательно имеющиеся в репо):

- **ГОСТ 520-2011** — Подшипники качения. Общие технические условия
- **ISO 15:2017** — Rolling bearings — Radial bearings — Boundary dimensions, general plan
- (и т.д., если они упоминаются в статье)

**Важно:** Различать файлы, которые **есть в репо** (первый список) и стандарты, которые **упоминаются** (второй список).

### 11) Ограничения и неопределённости

#### Что не нормировано

- **Конкретные марки смазок** — ГОСТ указывает тип, но не конкретную марку
- **Методы монтажа и демонтажа** — упоминаются общие требования, но нет детальных инструкций
- **Диагностика состояния** — методы контроля вибрации и температуры не стандартизированы в полной мере
- **Утилизация** — требования к переработке и утилизации отработанных подшипников не нормированы

#### Где ГОСТ и ISO расходятся

| Аспект | ГОСТ | ISO | Критичность |
|--------|------|-----|-------------|
| Температурный диапазон | -40...+120 °C | -30...+110 °C | Средняя |
| Требования к смазке | Конкретные типы (Литол-24, и т.д.) | Общие характеристики | Низкая |
| Маркировка | Цифровое обозначение | Буквенно-цифровое | Низкая |
| Методы испытаний | Подробно регламентированы | Общие требования | Средняя |
| Упаковка и хранение | Детальные требования | Общие рекомендации | Низкая |

#### Какие данные критично уточнить

**При проектировании:**
1. **Точные условия эксплуатации** — температура, загрязнённость среды, вибрации
2. **Характер нагрузки** — постоянная, переменная, ударная
3. **Требуемый ресурс** — в часах или оборотах, с учётом коэффициента безопасности
4. **Тип смазки** — консистентная или жидкая, периодичность обслуживания
5. **Класс точности** — для высокоскоростных или прецизионных применений

**При выборе аналогов:**
1. **Полное обозначение стандарта** — включая год издания и поправки
2. **Производитель** — для проверки соответствия каталожных данных
3. **Сертификаты соответствия** — особенно для ответственных применений
4. **Допуски и посадки** — для обеспечения взаимозаменяемости

**При эксплуатации:**
1. **Фактические условия работы** — нагрузки, температуры, скорости
2. **Состояние смазки** — периодичность замены, тип используемой смазки
3. **Наличие дефектов** — шум, вибрация, повышенная температура
