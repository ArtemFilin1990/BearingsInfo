# ГОСТ-стандарты по подшипникам

## ГОСТ 18854‑2013 — Номинальное статическое радиальное/осевое нагрузки

- Стандарт заменяет ГОСТ 18854‑94 и модифицирован на основе ISO 76:2006【588444937252912†L64-L78】.
- Вводятся определения статической эквивалентной радиальной нагрузки \(P_0\) и осевой нагрузки \(P_{0a}\) — это нагрузки, вызывающие ту же остаточную деформацию, что и фактическая нагрузка【588444937252912†L303-L316】.
- Определён статический коэффициент запаса \(S_0\) — отношение статической грузоподъёмности к эквивалентной нагрузке【588444937252912†L317-L322】.
- Даны термины: диаметр тела качения, эффективная длина ролика, номинальный угол контакта и диаметр делительной окружности【588444937252912†L323-L356】.
- Статическая грузоподъёмность соответствует остаточной деформации 0{.}0001 диаметра тела качения【588444937252912†L300-L302】.
- Перечислены схемы установки подшипников (tandem, back‑to‑back, face‑to‑face) и другие термины【588444937252912†L372-L396】.

## ГОСТ 7242 — Радиальные шариковые подшипники с защитными шайбами

- Распространяется на однорядные подшипники с защитными шайбами серий 60000 (1 шайба) и 80000 (2 шайбы)【630543591891967†L48-L51】.
- Шайбы не должны выступать за торцы колец и не должны вращаться относительно кольца; радиальный зазор измеряется до запрессовки шайб【630543591891967†L70-L85】.
- Подшипники серии 80000 заполняются смазкой и поставляются закрытыми; серия 60000 поставляется без смазки【630543591891967†L85-L92】.
- Защитные шайбы обрабатываются противокоррозионными покрытиями; посадки по валу и корпусу — по ГОСТ 3325, остальные требования — по ГОСТ 520【630543591891967†L107-L110】.

## ГОСТ 13218.1‑80 — Корпусные подшипниковые узлы ( ШМ )

- Стандарт распространяется на неразъёмные широкие корпусные узлы типа ШМ с сквозными отверстиями под подшипники диаметром 47‑150 мм【49887014073694†L25-L53】.
- Корпуса предназначены для радиально-упорных подшипников серий 1200, 1300, 1500, 11200, 11300, 11500, 11600 и 1600 по ГОСТ 5720‑75 и ГОСТ 8545‑75【49887014073694†L25-L53】.
- Предусмотрено два исполнения: с выемкой под необработанную поверхность (нагрузка от опоры) и без выемки для обработанных поверхностей (нагрузка к опоре)【49887014073694†L73-L81】.
- Основные размеры, обозначения и технические требования приведены в таблицах; рекомендации по выбору и расчёту нагрузки приведены в приложении 1【49887014073694†L83-L92】.

## ГОСТ 7872‑89 — Шариковые упорные подшипники

- Описывает однорядные (серия 8000), двухрядные (серия 38000), самоцентрирующиеся с отдельным свободным кольцом (серия 28000) и самоцентрирующиеся с седловой шайбой (серия 18000) подшипники【119850338611188†L43-L52】.
- Подшипники предназначены для восприятия только осевых нагрузок. Примеры обозначений: 8206 — одинарный упорный подшипник, 38206 — двойной, 28210/18210 — самоустанавливающиеся【119850338611188†L118-L127】.
- Технические требования и контроль качества корреспондируют ГОСТ 520, посадки — по ГОСТ 3325【119850338611188†L133-L135】.

## ГОСТ 8328‑75 — Радиальные роликовые подшипники с короткими цилиндрическими роликами

- Стандарт распространяется на роликовые подшипники, предназначенные для радиальной нагрузки【980716322292262†L53-L55】.
- Формируется по типам: 2000 (без бортов), 12000 (наружное кольцо с одним бортом), 32000 (внутреннее кольцо без бортов), 42000/52000 (наружное кольцо с двумя бортами, внутреннее с одним), 62000 (два фланца), 92000 и 102000 (с кольцами и упорными шайбами)【980716322292262†L60-L78】.
- Основные размеры (\(d, D, B\)) и радиус фаски \(r\) соответствуют таблицам; чертежи не задают конструктивное исполнение【980716322292262†L79-L80】【980716322292262†L158-L173】.

## ГОСТ 520‑2011 — Общие технические условия на подшипники качения

- Национальный стандарт, разработанный на основе ISO 492:2002 и ISO 199:2005; заменяет ГОСТ 520‑2002【110128295511652†L75-L85】.
- Определяет классы точности подшипников в порядке возрастания точности: нормальный (0), 6, 5, 4, T, 2【371647085035825†L19-L22】.
- Для конических подшипников буква «N» обозначает нормальный класс; другие классы обозначаются цифрами, например 5, 4; радиальные зазоры соответствуют ГОСТ 24810 и могут изменяться по согласованию【371647085035825†L67-L148】.
- Стандарт описывает терминологию, обозначения, размеры, допуски, технические требования, методы контроля, маркировку и упаковку.

## ГОСТ 5721‑2022 — Радиальные сферические двухрядные роликовые подшипники с асимметричными роликами

- Устанавливает типоразмеры, основные характеристики и требования к сферическим двухрядным роликовым подшипникам с асимметричными роликами, изготавливаемым по ГОСТ 520【39943091853125†L155-L170】.
- Документ включает рекомендации по выбору, эксплуатации и монтажу; ссылается на ГОСТ 520, ГОСТ 3189 (cистема обозначений), ГОСТ 3325 (посадки) и ГОСТ 3478 (монтажные размеры)【39943091853125†L155-L170】.

## ГОСТ 8338‑75 — Подшипники роликовые конические однорядные

- Стандарт устанавливает основные размеры и технические требования для конических роликовых подшипников.
- Эти подшипники предназначены для восприятия комбинированных радиально-осевых нагрузок.
- Широко применяются в автомобильных ступицах, редукторах и других узлах с высокими нагрузками.

## ГОСТ 10058‑90 — Подшипники шариковые радиально-упорные однорядные

- Стандарт распространяется на радиально-упорные шариковые подшипники с различными углами контакта.
- Подшипники предназначены для одновременного восприятия радиальных и осевых нагрузок.
- Могут устанавливаться парами для увеличения осевой грузоподъемности.

## ГОСТ 8882‑75 — Подшипники шариковые радиальные однорядные с уплотнениями

- Стандарт распространяется на подшипники с защитными уплотнениями (типы 60000 с одним уплотнением и 80000 с двумя уплотнениями).
- Уплотнения предназначены для защиты подшипников от загрязнений и удержания смазки.
- Подшипники серии 80000 поставляются с предварительно заложенной смазкой и не требуют дополнительного обслуживания.
- Включает таблицы значений статической (C₀) и динамической (C) грузоподъемности для подшипников различных серий (0, 1, 2, 3, 5, 6).

## ГОСТ 18854‑2013 — Подшипники роликовые радиальные сферические двухрядные (дополнение)

- Устанавливает основные размеры и технические требования для сферических роликовых подшипников.
- Эти подшипники обладают высокой радиальной грузоподъемностью и способностью компенсировать значительные перекосы.

## ГОСТ 18855‑2013 — Подшипники роликовые радиальные с витыми роликами

- Распространяется на подшипники с витыми (игольчатыми) роликами.
- Компактная конструкция позволяет использовать их в узлах с ограниченным радиальным пространством.
