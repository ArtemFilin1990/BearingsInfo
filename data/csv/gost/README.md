# Данные по ГОСТ

Структурированные данные о подшипниках по стандартам ГОСТ.

## Файлы

### bearings.csv
Основной каталог подшипников по ГОСТ
- Обозначение
- Тип
- Размеры (d, D, B)
- Грузоподъемность
- Серия

### dimensions.csv
Таблицы размеров по сериям

### series.csv
Описание серий подшипников ГОСТ

### tolerances.csv
Допуски и посадки по ГОСТ

## Формат

- Кодировка: UTF-8
- Разделитель: запятая
- Заголовки: в первой строке
