## Основные варианты

### 1. По бортам на кольцах

**Роликовые цилиндрические**:
- **N**: без бортов на наружном кольце
- **NU**: с двумя бортами на наружном кольце
- **NJ**: с бортом и направляющим фланцем
- **NUP**: со съемным бортом

### 2. По защитным устройствам

- **Открытые**: без защиты
- **Z**: одна металлическая шайба
- **ZZ**: две металлические шайбы
- **RS**: одно резиновое уплотнение
- **2RS**: два резиновых уплотнения

### 3. По сепаратору

- Стальной штампованный (стандарт)
- Латунный массивный (М)
- Полиамидный (TN9, TVP)

### 4. По точности

- Класс 0 (нормальный)
- P6, P5, P4, P2 (повышенная точность)

## Специальные исполнения

- **Гибридные**: керамические тела качения
- **Из нержавеющей стали**: для влажных сред
- **С покрытиями**: для вакуума, агрессивных сред
