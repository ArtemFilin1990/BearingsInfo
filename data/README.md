# Data - Данные

Директория содержит структурированные данные в формате CSV.

## Структура

### Основные категории данных

#### ГОСТ
- `gost/bearings.csv` - Подшипники по ГОСТ
- `gost/dimensions.csv` - Размеры подшипников ГОСТ
- `gost/series.csv` - Серии подшипников
- `gost/tolerances.csv` - Допуски и посадки

#### ISO
- `iso/bearings.csv` - Подшипники по ISO
- `iso/dimensions.csv` - Размеры подшипников ISO
- `iso/prefixes.csv` - Префиксы в обозначениях
- `iso/suffixes.csv` - Суффиксы в обозначениях

#### Аналоги
- `analogs/gost_iso.csv` - Таблица соответствий ГОСТ ↔ ISO
- `analogs/units.csv` - Подшипниковые узлы
- `analogs/housings.csv` - Корпуса подшипниковых узлов

#### Бренды
- `brands/brands.csv` - База производителей и брендов подшипников

#### Отчеты
- `reports/` - Автоматически генерируемые отчеты об обновлениях данных

## Формат данных

Все данные представлены в формате CSV (Comma-Separated Values) с:
- Заголовками в первой строке
- Кодировкой UTF-8
- Разделителем запятая (,)

## Использование

Данные можно использовать для:
- Подбора аналогов подшипников
- Поиска характеристик
- Интеграции в базы данных
- Анализа и обработки
- Автоматизации процессов подбора

## Обновление данных

Данные обновляются на основе:
- Официальных каталогов производителей
- Стандартов ГОСТ и ISO
- Проверенных технических источников
