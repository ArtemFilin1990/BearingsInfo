# Номенклатура производителей подшипников

## Описание

Справочник производителей подшипников, работающих по стандартам ГОСТ и ISO. Данный документ содержит полный перечень производителей, представленных в базе данных.

## Назначение

- Идентификация производителя по маркировке подшипника
- Подбор аналогов между производителями
- Понимание географии производства подшипников
- Навигация по каталогам производителей

## Структура номенклатуры

Производители разделены на две основные группы:
1. **Российские заводы (ГПЗ)** - государственные подшипниковые заводы СССР и России
2. **Международные производители** - европейские, американские и азиатские компании
