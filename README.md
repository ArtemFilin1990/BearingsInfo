# База знаний по подшипникам

Структурированная база знаний по подшипникам качения с обозначениями по ГОСТ и ISO.

## Содержание

1. [Классификация подшипников](docs/bearings/classification/README.md)
2. [Системы обозначений](docs/bearings/designations/README.md)
   - [Обозначения по ГОСТ](docs/bearings/designations/gost.md)
   - [Обозначения по ISO](docs/bearings/designations/iso.md)
3. [Таблицы аналогов ГОСТ↔ISO](docs/bearings/analogues/README.md)
4. [Каталог подшипников](docs/bearings/catalog/README.md)
5. [Часто задаваемые вопросы (FAQ)](docs/bearings/faq/README.md)
6. [Подшипники ГОСТ (учебное пособие)](docs/bearings/gost/README.md)

## Источники

- ГОСТ 3189-89 "Подшипники качения шариковые и роликовые. Система условных обозначений"
- Каталоги производителей: SKF, FAG/INA (Schaeffler Group), MPZ, NKE, ISB

## О проекте

База знаний содержит только проверенную информацию из официальных стандартов и каталогов производителей подшипников. Все данные о размерах, весе, аналогах и применении соответствуют реальным техническим характеристикам.

### Цели проекта

- **Систематизация знаний** о подшипниках качения в едином справочнике
- **Упрощение подбора** подшипников для инженеров и технических специалистов
- **Облегчение замены** подшипников ГОСТ на ISO и наоборот
- **Предоставление актуальной** технической информации о размерах, весе и характеристиках
- **Помощь в обучении** студентам технических специальностей

### Для кого этот проект

- Инженеры-конструкторы и проектировщики
- Технические специалисты по обслуживанию оборудования
- Снабженцы и закупщики подшипников
- Студенты машиностроительных специальностей
- Работники ремонтных служб предприятий

### Особенности

✅ **Полное соответствие стандартам** - вся информация основана на ГОСТ 3189-89 и международных стандартах ISO  
✅ **Проверенные данные** - характеристики взяты из официальных каталогов производителей (SKF, FAG, NSK, NTN)  
✅ **Практическая направленность** - примеры применения для конкретных типов оборудования  
✅ **Удобная навигация** - структурированное изложение с перекрестными ссылками  
✅ **Таблицы аналогов** - быстрый поиск замены между ГОСТ и ISO  

### Как пользоваться

1. Для изучения основ подшипников начните с раздела [Классификация](docs/bearings/classification/README.md)
2. Чтобы расшифровать обозначение, перейдите в [Системы обозначений](docs/bearings/designations/README.md)
3. Для замены подшипника используйте [Таблицы аналогов](docs/bearings/analogues/README.md)
4. Для подбора по размерам смотрите [Каталог](docs/bearings/catalog/README.md)
5. Ответы на частые вопросы ищите в [FAQ](docs/bearings/faq/README.md)

### Отказ от ответственности

Данная база знаний предоставляется исключительно в информационных целях. Автор не несет ответственности за решения, принятые на основе информации из этого проекта. Для критически важных применений всегда консультируйтесь с официальными каталогами производителей и квалифицированными специалистами.

## Как внести вклад

Проект открыт для улучшений и дополнений. Вы можете помочь следующими способами:

- **Исправление ошибок** - если нашли неточность в данных, создайте Issue с описанием проблемы
- **Дополнение информации** - можно добавить новые типы подшипников, производителей или примеры применения
- **Улучшение документации** - исправление опечаток, улучшение формулировок, добавление иллюстраций
- **Добавление примеров** - практические кейсы подбора и применения подшипников

При добавлении новой информации обязательно указывайте источник (ГОСТ, каталог производителя с указанием года издания).

## Лицензия

Проект распространяется для свободного использования в образовательных и справочных целях. При использовании материалов желательно указывать ссылку на источник.
