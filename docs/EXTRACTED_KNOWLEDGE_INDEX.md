**Расположение**: `tests/Суффиксы ISO.docx`  
**Размер**: ~300+ записей суффиксов  
**Создан документ**: [docs/bearings/designations/manufacturer_suffixes_cross_reference.md](designations/manufacturer_suffixes_cross_reference.md)

Полная таблица кросс-референса суффиксов между различными производителями подшипников.

**Структура данных**:
- Суффикс производителя
- Производитель
- Тип подшипника (если применимо)
- Описание
- Эквивалент ISO

**Категории суффиксов**:

1. **Уплотнения и защита**
   - 2RS, RS (SKF, FAG, INA, KOYO)
   - DDU, DU (NSK)
   - Z, ZZ (общие)
   - PP, RR (FAFNIR)

2. **Канавки и стопорные кольца**
   - N, NR (все производители)
   - N2 (FAG)
   - NDU, NS7 (NSK)

3. **Классы точности**
   - P4, P5, P6 (SKF, FAG, NSK, NTN, KOYO)
   - PA3, PA5, PA7 (NSK)
   - ММ (FAFNIR)

4. **Радиальные зазоры**
   - C2, C3, C4, C5 (все производители)
   - P (FAFNIR для зазора)

5. **Сепараторы**
   - M, MA, MB (латунные)
   - TN, TN9, TV, TVH (полиамидные)
   - MG (полиамид армированный)

6. **Температурная стабилизация**
   - S1 (200°C), S2 (250°C), S3 (300°C), S4 (350°C)
   - T2 (NTN для 250°C)
   - TS1-TS4 (NTN)

7. **Специальные конструкции**
   - W33, W20 (для сферических)
   - CA, CC (усиленные)
   - E, EA (усиленные FAG/SKF)

8. **Комплектные подшипники**
   - DB, DF, DT (схемы установки)
   - QB, QF, QU (комплекты из 4 шт)
   - TDT, TFT (комплекты из 3 шт)

9. **Контроль шума и вибрации**
   - Q5, Q6 (SKF)
   - QE5, QE6 (SKF для электродвигателей)

10. **Код смазки**
    - MT33, MT37, MT47, MT57, MT59 (SKF)
    - VT105, VT120, VT131, VT162, VT164 (SKF)
    - SR, SRL (KOYO)

11. **Специальные применения**
    - VA201, VA208, VA405, VA407 (SKF для вагонеток печей)
    - W740, W800 (TORRINGTON для вибрационных установок)
    - ROVS (KOYO для вибропросеивателей)

**Применение**:
- Подбор аналогов между производителями

### 4. Подшипники SKF.docx

**Расположение**: `tests/Подшипники SKF.docx`  
**Размер**: 1699 строк  
**Создан документ**: [docs/bearings/brands/skf_designation_system.md](brands/skf_designation_system.md)

#### Содержание

Полное руководство по системе обозначений подшипников SKF, включая стандартные и специальные подшипники, комплектующие.

**Основные разделы**:

1. **Структура базового обозначения**
   - Префикс (опционально)
   - Основное обозначение (тип + серия + диаметр)
   - Суффиксы

2. **Определение диаметра отверстия**
   - Стандартные диаметры (d ≥ 20 мм): d = код × 5
   - Малые диаметры (10, 12, 15, 17 мм): коды 00, 01, 02, 03
   - Диаметры < 10 мм: прямое указание (629 = d 9 мм)
   - Диаметры ≥ 500 мм: через косую черту (618/500)
   - Нестандартные: 322/22, 322/28, 322/32
   - Произвольные: 6202/15,875

3. **Обозначения серий**
   - Дополнительные буквы A, B, C, D, E
   - Комбинации CA, CB, CC и др.

4. **Специальные подшипники**
   - По номеру чертежа (6-7 цифр)
   - С префиксом B (Bxxx-номер)
   - С префиксами F, G, K

5. **Закрепительные втулки**
   - H - стандартная метрическая
   - HA - дюймовая 1/16"
   - HE - дюймовая 1/4"
   - HS - дюймовая 1/8"
   - OH - с масляными каналами

6. **Стяжные втулки**
   - AH - стандартная
   - AHX - стандарт ISO
   - AOH - с масляными каналами
   - AOHX - ISO с масляными каналами

7. **Гайки**
   - HM - трапецеидальная резьба ISO
   - HML - с меньшим сечением
   - HMV - гидравлическая гайка

**Практические примеры**:
- 6308-2RS1-C3: шариковый ⌀40мм, уплотнения, зазор C3
- 22215 CCK/W33: сферический роликовый ⌀75мм, усиленный, конус
- H310 + 21310 CCK: втулка для вала ⌀45мм

**Применение**:
- Расшифровка обозначений SKF
- Подбор втулок и гаек
- Понимание системы SKF
- Заказ подшипников и комплектующих SKF
