# IRBIS bearing size table (offline snapshot)

## Purpose
Хранить офлайн-выгрузку таблицы размеров подшипников IRBIS без внешних ссылок.

## Classification
- Независимый каталог (2600+ типоразмеров).
- Используется для быстрой проверки d/D/B и соответствий ГОСТ/ISO.

## Designation
- `SNAP_SRC_IRBIS_TABLE` — CSV/HTML выгрузка таблицы IRBIS.

## Prefixes/Suffixes
- Префикс `SNAP_SRC_` — офлайн-слепок.
- Суффикс `_TABLE` — табличный каталог.

## Examples
- Сохранить полный CSV с колонками обозначение, d, D, B, масса и указать дату выгрузки.

## Notes & Limitations
- Прямой доступ заблокирован (403); требуется офлайн-загрузка.
- При обновлении фиксировать источник и контрольную сумму файла.

## Standard References
- Домен источника: irbis.ua (раздел размеров подшипников).
- Локальные файлы: этот `.md` + CSV выгрузка в той же директории.
