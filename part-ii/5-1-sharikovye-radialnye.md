## Описание и назначение

**Шариковые радиальные однорядные подшипники** — это наиболее распространенный тип подшипников качения, составляющий более **70% от общего объема производства** подшипников в мире.

### Основное определение

Шариковый радиальный однорядный подшипник (англ. *Single row deep groove ball bearing*) — это подшипник качения, в котором между внутренним и наружным кольцами расположен один ряд шариков, удерживаемых сепаратором, предназначенный для восприятия преимущественно радиальных нагрузок.

**Обозначение:**
- **По ГОСТ:** Тип **0** (первая цифра в обозначении)
- **По ISO:** Серии **60, 62, 63, 64** (серия зависит от наружного диаметра)
- **Примеры:** 6205, 180205, 6305, 6000-2RS, 608-2RS

### Назначение и функции

**Основные функции:**
1. **Поддержка вращающегося вала** с минимальным трением
2. **Восприятие радиальных нагрузок** (перпендикулярно оси)
3. **Восприятие небольших осевых нагрузок** (до 50% от статической радиальной грузоподъемности)
4. **Точное позиционирование вала** в радиальном направлении
5. **Обеспечение высоких скоростей вращения** (до 50 000 об/мин для малых размеров)

**Сферы применения:** Электродвигатели, насосы, редукторы, бытовая техника, автомобили, станки

## Конструктивные особенности

### Основные компоненты

| Элемент | Описание | Характеристики | Материал |
|---------|----------|----------------|----------|
| **Внутреннее кольцо** | Посажено на вал | Дорожка качения — желобчатая, фаски для монтажа | ШХ15, твердость 58-63 HRC |
| **Наружное кольцо** | Устанавливается в корпус | Дорожка качения на внутренней поверхности, возможна стопорная канавка | ШХ15, твердость 58-63 HRC |
| **Шарики** | Тела качения | Количество: 6-12 шт (зависит от размера), сферичность < 0,5 мкм | ШХ15, твердость 60-66 HRC |
| **Сепаратор** | Удерживает шарики | Штампованный (стандарт), массивный (скорости), полимерный (шум) | Сталь 10, латунь Л63, полиамид PA66 |

### Геометрия дорожек качения

**Особенность конструкции:** Глубокие желобчатые дорожки качения обеспечивают:
- Радиус дорожки ≈ 52% от диаметра шарика
- Угол контакта α ≈ 0° в ненагруженном состоянии
- Под нагрузкой: α = 10-15° (позволяет воспринимать осевую нагрузку)

**Зазор между шариком и дорожкой:**
- Осциллирующий зазор (osculation): 2-4% от диаметра шарика
- Обеспечивает равномерное распределение контактных напряжений
- Позволяет роликам катиться с минимальным проскальзыванием

## Обозначение по ГОСТ и ISO

### Система обозначений ГОСТ 520-2011

**Структура обозначения:**
```
[Префиксы] TZSS [Суффиксы]
```

Где:
- **T** — тип подшипника (0 для радиальных однорядных)
- **Z** — серия диаметров (0, 1, 2, 3, 4)
- **SS** — код внутреннего диаметра

**Пример расшифровки: 6205**

1. **6** — тип подшипника (шариковый радиально-упорный, но для однорядных радиальных используется **0**, а **6** для радиально-упорных)
   - *Примечание:* Для чисто радиальных: **0**205, но часто пишут **6**205 (ISO стиль)
2. **2** — серия диаметров (легкая серия)
3. **05** — код внутреннего диаметра: 05 × 5 = **25 мм**

**Полная расшифровка:** Шариковый радиальный однорядный подшипник легкой серии с внутренним диаметром 25 мм.

### Система обозначений ISO

**Серии по ISO:**

| Серия ISO | Серия ГОСТ | Тип | D (при d=20мм) | Применение |
|-----------|------------|-----|----------------|------------|
| **60** | 10 | Сверхлегкая | 42 мм | Приборы, малые нагрузки |
| **62** | 02 | Легкая | 47 мм | Общего назначения (самая распространенная) |
| **63** | 03 | Средняя | 52 мм | Повышенные нагрузки |
| **64** | 04 | Тяжелая | 62 мм | Большие нагрузки |
| **160** | - | Особо легкая узкая | 42 мм | Специальное применение |

**Примеры ISO обозначений:**
- **6205** = серия 62, d = 05 × 5 = 25 мм
- **6308** = серия 63, d = 08 × 5 = 40 мм
- **6010** = серия 60, d = 10 × 5 = 50 мм

### Суффиксы (модификации)

| Суффикс | Значение | Пример |
|---------|----------|--------|
| **Z** | Защитная шайба с одной стороны | 6205-Z |
| **2Z (ZZ)** | Защитные шайбы с двух сторон | 6205-2Z |
| **RS** | Резиновое уплотнение с одной стороны | 6205-RS |
| **2RS** | Резиновые уплотнения с двух сторон (герметичный) | 6205-2RS |
| **C3** | Увеличенный радиальный зазор | 6205-C3 |
| **/P5** | Класс точности 5 (ISO) | 6205/P5 |
| **M** | Массивный латунный сепаратор | 6205M |
| **TN** | Полимерный сепаратор (полиамид) | 6205-TN |
| **NR** | Стопорное кольцо на наружном кольце | 6205-NR |

**Комбинированный пример: 6205-2RS-C3/P5**
- 6205 — тип и размер
- 2RS — два резиновых уплотнения
- C3 — увеличенный зазор
- /P5 — класс точности 5

## Технические характеристики

### Грузоподъемность

**Динамическая грузоподъемность (C):**  
Определяется по ISO 281, зависит от размера подшипника.

**Пример для подшипника 6205:**
- C = 14 000 Н (14 кН)
- C₀ = 6 900 Н (6,9 кН)

**Формула расчета срока службы:**
```
L₁₀ = (C / P)³ [млн. оборотов]
```

**Пример расчета:**
- При нагрузке P = 1,4 кН:
  - L₁₀ = (14 / 1,4)³ = 1000 млн. оборотов
- При частоте вращения n = 1500 об/мин:
  - L₁₀h = 1000 × 10⁶ / (60 × 1500) = **11 111 часов** ≈ 463 дня непрерывной работы

### Предельные частоты вращения

**Факторы, определяющие максимальную скорость:**
- Тип смазки
- Температура окружающей среды
- Нагрузка
- Точность изготовления
- Тип сепаратора

**Предельные частоты (пример для 6205):**

| Условия | Частота вращения |
|---------|------------------|
| Пластичная смазка | 9 000 об/мин |
| Масляная смазка | 13 000 об/мин |
| Масляная смазка + полимерный сепаратор | 16 000 об/мин |
| Прецизионный класс P5 + масло + охлаждение | до 20 000 об/мин |

**Сравнение с другими типами (относительная скорость):**

| Тип подшипника | Относительная макс. скорость |
|----------------|------------------------------|
| Шариковый радиальный однорядный | 100% (базовый уровень) |
| Радиально-упорный шариковый | 80-90% |
| Цилиндрический роликовый | 70-80% |
| Конический роликовый | 50-60% |

### Точность изготовления

**Классы точности по ГОСТ/ISO:**

| Класс ГОСТ | ISO | ABEC | Радиальное биение, мкм | Применение |
|------------|-----|------|------------------------|------------|
| **0** | Normal (P0) | ABEC-1 | 15 | Общее машиностроение (80% применений) |
| **6** | P6 | ABEC-3 | 10 | Электродвигатели, редукторы |
| **5** | P5 | ABEC-5 | 7 | Станки, насосы высокой точности |
| **4** | P4 | ABEC-7 | 5 | Прецизионные станки, шпиндели |
| **2** | P2 | ABEC-9 | 2,5 | Высокоточные шпиндели, измерительные приборы |

**Влияние класса точности:**
- Биение вала уменьшается в 2 раза при переходе на следующий класс
- Уровень шума снижается на 3-5 дБ
- Максимальная скорость увеличивается на 20-30%
- Стоимость возрастает в 2-3 раза на каждый класс

## Преимущества и недостатки

### Преимущества

| Преимущество | Значение | Обоснование |
|--------------|----------|-------------|
| **Низкий коэффициент трения** | 0,0015-0,0020 | Точечный контакт шариков с дорожками |
| **Высокая частота вращения** | До 50 000 об/мин | Малая масса шариков, низкое трение |
| **Малый вес** | На 20-40% легче роликовых | Меньше материала в телах качения |
| **Низкий уровень шума** | 40-60 дБ | Плавное качение шариков |
| **Простота монтажа** | Стандартные инструменты | Симметричная конструкция |
| **Широкий диапазон размеров** | d от 1 до 500 мм | Полная стандартизация |
| **Взаимозаменяемость** | 100% | Точные стандарты ГОСТ/ISO |
| **Низкая стоимость** | Базовый уровень | Массовое производство |

### Ограничения и недостатки

| Недостаток | Значение | Решение |
|------------|----------|---------|
| **Меньшая грузоподъемность** | На 30-50% ниже роликовых | Использовать роликовые подшипники |
| **Чувствительность к перекосам** | Допустимо до 4' (угловых минут) | Точная соосность, самоустанавливающиеся типы |
| **Ограниченная ударная прочность** | Могут появиться вмятины | Сферические роликовые для ударных нагрузок |
| **Осевая нагрузка** | Не более 50% от C₀ | Радиально-упорные подшипники |
| **Шум при высоких скоростях** | Увеличивается с ростом скорости | Прецизионные классы, керамические шарики |

## Области применения

### Промышленное применение (60-70% от общего объема)

#### Электродвигатели

**Доля применения:** 60% всех шариковых радиальных подшипников

**Типичные условия:**
- Частота вращения: 1000-3000 об/мин
- Температура: до +80°C
- Нагрузка: легкая-средняя
- Требования: низкий шум, долгий срок службы

**Типичные размеры:** 6204, 6205, 6206, 6207, 6208

**Модификации:**
- 6205-2RS (герметичные, пожизненная смазка)
- 6205-2Z-C3 (с защитой и увеличенным зазором для компенсации температуры)
- 6205/P6 (повышенная точность для снижения вибрации)

#### Насосы и компрессоры

**Условия:**
- Высокие скорости (до 10 000 об/мин)
- Возможны осевые нагрузки
- Агрессивные среды (требуются уплотнения)

**Типичные размеры:** 6202, 6203, 6204, 6205

**Модификации:**
- Нержавеющие (AISI 440C) для химических сред
- Гибридные (керамические шарики) для высоких скоростей

#### Редукторы и коробки передач

**Условия:**
- Комбинированные нагрузки (радиальные + осевые от косозубых шестерен)
- Средние-высокие частоты вращения
- Повышенные температуры

**Типичные размеры:** 6305, 6306, 6307, 6308

**Особенности:**
- Группа зазора C3 (для компенсации температурного расширения)
- Посадки с натягом (k6, m6) для предотвращения проворачивания

### Транспортное применение (15-20%)

#### Автомобили

**Применение:**
- Генераторы: 6202, 6203
- Натяжители ремней: 6204, 6205
- Водяные помпы: 6203, 6204
- Коробки передач: 6305, 6306

**Требования:**
- Надежность (срок службы > 100 000 км)
- Низкий шум
- Работа в широком диапазоне температур (-40...+120°C)
- Герметичность (2RS)

### Бытовая техника и электроинструмент (10-15%)

**Применение:**
- Стиральные машины: 6204-2RS, 6205-2RS
- Пылесосы: 608-2RS, 609-2RS
- Кухонная техника: 6200-2RS, 6201-2RS
- Электродрели: 6201, 6202

**Требования:**
- Низкий шум (< 50 дБ)
- Долгий срок службы без обслуживания
- Низкая стоимость
- Герметичность

## Практические примеры

### Пример 1: Подбор подшипника для электродвигателя

**Условия эксплуатации:**
- Мощность двигателя: 1,5 кВт
- Частота вращения: 1500 об/мин
- Радиальная нагрузка: Fr = 500 Н
- Требуемый срок службы: 20 000 часов

**Решение:**

1. **Расчет требуемого ресурса в оборотах:**
   ```
   Lh = 20 000 часов × 60 мин × 1500 об/мин = 1 800 млн. оборотов
   ```

2. **Расчет требуемой динамической грузоподъемности:**
   ```
   C = P × ³√L₁₀ = 0,5 кН × ³√1800 = 0,5 × 12,2 = 6,1 кН
   ```

3. **Выбор подшипника:**
   - Требуется C > 6,1 кН
   - Подходящие варианты: 6203 (C=9,6 кН), 6204 (C=12,8 кН)
   - **Выбираем:** 6203-2RS (с уплотнениями для защиты от пыли)

4. **Проверка:**
   ```
   L₁₀ = (9,6 / 0,5)³ = 7 077 млн. об. = 78 600 часов > 20 000 часов ✓
   ```

### Пример 2: Расшифровка маркировки 6205-2RS-C3/P5

**Пошаговая расшифровка:**

1. **6205** — основное обозначение:
   - **62** — серия (легкая)
   - **05** — внутренний диаметр: 05 × 5 = 25 мм
   - **Тип:** шариковый радиальный однорядный

2. **2RS** — два резиновых контактных уплотнения:
   - Герметичный подшипник
   - Заполнен пожизненной смазкой
   - Не требует обслуживания

3. **C3** — увеличенный радиальный зазор:
   - Для компенсации термического расширения
   - Рекомендуется при посадке с натягом
   - Для повышенных температур эксплуатации

4. **/P5** — класс точности 5 по ISO:
   - Радиальное биение дорожки качения < 7 мкм
   - Для применений, требующих повышенной точности
   - Снижен уровень вибрации и шума

**Полная расшифровка:**  
Шариковый радиальный однорядный подшипник легкой серии с внутренним диаметром 25 мм, наружным диаметром 52 мм, шириной 15 мм, с двумя резиновыми уплотнениями, увеличенным радиальным зазором и классом точности P5.

**Применение:** Электродвигатели повышенной мощности, насосы, вентиляторы высокой точности.

### Пример 3: Выбор смазки и интервала замены

**Исходные данные:**
- Подшипник: 6208
- Частота вращения: 3000 об/мин
- Температура: +70°C
- Условия: чистые, без влаги

**Варианты смазки:**

| Тип смазки | Интервал замены | Макс. скорость | Преимущества |
|------------|-----------------|----------------|--------------|
| **Пластичная (NLGI 2)** | 5 000-10 000 часов | 9 000 об/мин | Простота, не требует уплотнений |
| **Масляная (ISO VG 32)** | Постоянная циркуляция | 13 000 об/мин | Лучшее охлаждение, меньше трение |
| **Гибрид (пластичная + масляный туман)** | 15 000-20 000 часов | 16 000 об/мин | Компромисс |

**Рекомендация:**  
При n = 3000 об/мин < 9000 об/мин → **пластичная смазка NLGI 2** (например, Литол-24, Mobil Polyrex EM)

**Интервал замены:**
```
t = k × (14 000 / n) часов
где k = 1 (нормальные условия), n = 3000 об/мин
t = 1 × (14 000 / 3000) = 4,67 ≈ 5 000 часов
```

## Ключевые выводы

1. **Шариковые радиальные однорядные подшипники** — это самый распространенный тип подшипников качения (70% мирового производства), обеспечивающий оптимальное сочетание низкого трения (0,0015-0,0020), высоких скоростей (до 50 000 об/мин) и доступной стоимости.

2. **Глубокие желобчатые дорожки качения** позволяют подшипникам воспринимать не только радиальную нагрузку, но и осевую (до 50% от статической радиальной грузоподъемности C₀), что делает их универсальными для большинства применений.

3. **Размерные серии** (60, 62, 63, 64) обеспечивают гибкость выбора: серия 62 (легкая) — самая распространенная (80% применений), серии 63-64 для повышенных нагрузок, серия 60 для компактных конструкций при ограничении радиального пространства.

4. **Модификации с уплотнениями** (2RS, 2Z) составляют 40-50% производства, обеспечивая пожизненную смазку и защиту от загрязнений, что критично для электродвигателей, бытовой техники и автомобильных применений.

5. **Класс точности** напрямую влияет на максимальную скорость, уровень вибрации и шума: переход с класса 0 на P5 снижает радиальное биение в 2 раза (с 15 до 7 мкм), но увеличивает стоимость в 2-3 раза, что оправдано только для прецизионных применений.

6. **Основная область применения** — электродвигатели (60% всех применений), где подшипники работают при частотах 1000-3000 об/мин, температурах до +80°C и требуют надежности 20 000+ часов, что обеспечивается правильным выбором размера, класса зазора (C3 для компенсации температуры) и типа уплотнения (2RS для защиты).
