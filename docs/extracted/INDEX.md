# Индекс извлечённых каталогов

Данная директория содержит Markdown-файлы, которые являются извлечениями из PDF, DOCX и других первичных источников производителей подшипников и сопутствующей продукции.

## Назначение

Эти файлы представляют собой текстовые копии каталогов производителей, извлечённые для:
- Полнотекстового поиска по содержимому каталогов
- Быстрого доступа к информации без необходимости открывать PDF
- Индексации и структурирования данных из первичных источников

## Первичные источники

Оригинальные PDF, DOCX, XLSX файлы хранятся в директории `/sources/`. См. `/sources/README.md` для подробной информации о первичных источниках.

## Использование

1. Для поиска конкретного производителя используйте список выше
2. Для полнотекстового поиска используйте команду `grep` или поиск в IDE
3. Для получения структурированных данных обращайтесь к CSV файлам в `/data/`

## Связь с другими директориями

- **`/sources/`** - первичные PDF/DOCX/XLSX источники и метаданные
- **`/data/`** - структурированные CSV данные, извлечённые из каталогов
- **`/docs/articles/`** - аналитические статьи на основе данных каталогов
- **`/docs/brands/`** - информация о производителях (планируется)

## Примечания

- Файлы в этой директории не редактируются вручную - они являются результатом автоматического извлечения
- При обновлении первичного источника в `/sources/` соответствующий файл здесь должен быть переизвлечён
- Некоторые каталоги могут быть неполными или содержать ошибки распознавания текста
- **Процесс извлечения:** Используется скрипт `/sources/pdf_text_extractor.py` для конвертации PDF в Markdown
- Для переизвлечения каталога см. документацию в `/sources/README.md`
