## Введение

Линейные направляющие (рельсовые направляющие, linear guides) — прецизионные механические устройства, обеспечивающие точное линейное перемещение подвижных узлов машин и оборудования. Они широко применяются в станках с ЧПУ, робототехнике, медицинском оборудовании и автоматизированных системах.

## Конструкция

### Основные элементы:
1. **Направляющая рельса** (rail) — закрепляется на основании
2. **Каретка** (carriage, block) — перемещается по рельсе
3. **Тела качения** — шарики или ролики
4. **Система рециркуляции** — возвращает тела качения

### Принцип работы:
Каретка перемещается по рельсе на телах качения (шариках или роликах), которые циркулируют внутри каретки. Это обеспечивает низкое трение и высокую точность.

## Типы линейных направляющих

### 1. По типу тел качения:

#### Шариковые (ball type)
- **Преимущества**: Плавный ход, высокая скорость (до 5 м/с), низкое трение
- **Недостатки**: Меньшая грузоподъемность, чувствительны к ударам
- **Применение**: Станки, принтеры 3D, медоборудование

#### Роликовые (roller type)
- **Преимущества**: Высокая грузоподъемность (в 2-3 раза выше), жесткость
- **Недостатки**: Ниже скорость, выше трение
- **Применение**: Тяжелые станки, прессы, металлообработка

### 2. По конструкции рельса:

#### Стандартные (normal flange)
- Каретка охватывает рельсу
- Универсальное применение

#### Миниатюрные (miniature)
- Размеры 3-15 мм
- Точное оборудование, медтехника

#### Широкие (wide)
- Увеличенная ширина для жесткости
- Станки, роботы

## Обозначение направляющих

### Система THK (пример):
**HGH 20 CA**

- **HG** — серия (Heavy Guide)
- **H** — тип каретки (High, высокая)
- **20** — размер рельсы (ширина 20 мм)
- **C** — класс точности (C, H, P, SP, UP)
- **A** — предварительный натяг (Z0, ZA, ZB)

### Система HIWIN:
**HGW 15 CC**

- **HG** — серия
- **W** — тип каретки (Wide, широкая)
- **15** — размер
- **CC** — точность и предварительный натяг

## Классы точности

| Класс | Точность параллельности, мкм | Применение |
|-------|----------------------------|------------|
| **C** (Normal) | 0-23 | Общепромышленное оборудование |

| **UP** (Ultra Precision) | 0-2 | Сверхточное оборудование |

## Предварительный натяг

### Зачем нужен:
- Устранение люфта
- Увеличение жесткости
- Повышение точности позиционирования

### Обозначения:
- **Z0** — без натяга (зазор)
- **ZA** — легкий натяг
- **ZB** — средний натяг
- **ZC** — тяжелый натяг

### Выбор натяга:
- **Z0**: Длинные перемещения, высокие скорости
- **ZA**: Общие применения, станки
- **ZB**: Высокая точность, жесткость
- **ZC**: Максимальная жесткость, низкие скорости

## Монтаж

### Требования к установочной поверхности:
- **Плоскостность**: ≤ 0.02 мм на 1000 мм
- **Шероховатость**: Ra ≤ 3.2 мкм
- **Твердость**: ≥ HRC 55 (для точных применений)

### Последовательность монтажа:
1. Очистить поверхности
2. Установить первую рельсу, затянуть
3. Установить вторую рельсу параллельно (проверка параллелометром)
4. Смазать направляющие
5. Установить каретки

### Допуски параллельности:
- **Стандарт**: ≤ 0.05 мм на 1000 мм
- **Точные станки**: ≤ 0.02 мм на 1000 мм

## Смазка

### Типы смазки:
1. **Пластичная** (консистентная) — литиевая, NLGI 2
2. **Масляная** — ISO VG 32-68
3. **Автоматическая** — централизованные системы

### Периодичность:
- **Нормальные условия**: 100-300 км хода
- **Тяжелые условия**: 50-100 км хода
- **Автосмазка**: Каждые 10-50 км

## Срок службы

### Расчет ресурса:
```
L = (C / P)³ × 50 км

где:
L — ресурс (км)
C — динамическая грузоподъемность (кН)
P — эквивалентная нагрузка (кН)
```

**Пример**: HG 20 CA
- C = 19.8 кН
- P = 3 кН
- L = (19.8 / 3)³ × 50 ≈ 14500 км

## Производители

### Японские (премиум):
- **THK** — мировой лидер, изобретатель LM-направляющих
- **HIWIN** (Тайвань) — соотношение цена/качество
- **NSK** — высокая точность
- **IKO** — миниатюрные направляющие

### Европейские:
- **Bosch Rexroth** (Германия)
- **Schneeberger** (Швейцария) — сверхточные

### Китайские:
- **STAF**, **PMI**, **TBI** — бюджетные решения

## Преимущества

✅ Высокая точность позиционирования (±1 мкм)  
✅ Низкое трение (коэффициент < 0.002)  
✅ Высокая жесткость  
✅ Долгий срок службы (10000+ часов)  
✅ Высокие скорости (до 5 м/с)  
✅ Низкое обслуживание  

## Применение

- **Станки с ЧПУ**: Фрезерные, токарные, шлифовальные
- **Роботы**: Манипуляторы, pick-and-place
- **3D-принтеры**: Оси X, Y, Z
- **Медицинское оборудование**: Томографы, сканеры
- **Измерительные машины**: КИМ (координатно-измерительные)
- **Полупроводниковое оборудование**: Чистые комнаты
